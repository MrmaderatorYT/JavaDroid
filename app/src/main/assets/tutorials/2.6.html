Цикли

Останнє оновлення: 16.09.2025

Ще одним видом керуючих конструкцій є цикли. Цикли дозволяють залежно від певних умов виконувати певну дію багато разів. У мові Java є такі види циклів:

  • for

  • while

  • do...while

Цикл for

Цикл for має таке формальне визначення:


for ([ініціалізація счетчика]; [условие]; [зміна счетчика])
{
    // действия
}

Розглянемо стандартний цикл for:


for (int i = 1; i < 9; i++){
    System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
}

Первая часть оголошення цикла - int i = 1 створює и ініціалізує счетчик i. Счетчик необязательно повинен представлять тип int. Це может быть и любой другой числовой тип, наприклад, float. Перед виконанням цикла значення счетчика буде равно 1. В данном случае це то же самое, что и оголошення змінної.

Друга частина - умова, за якої виконуватиметься цикл. У разі цикл буде виконуватися, доки i не досягне 9.

И третья часть - приращение счетчика на единицу. Опять же нам необязательно увеличивать на единицу. Можна уменьшать: i--.

У результаті блок циклу спрацює 8 разів, поки значення i не стане рівним 9. І щоразу це значення збільшуватиметься на 1.

Нам необов'язково вказувати всі умови під час оголошення циклу. Наприклад, ми можемоо написати так:


int i = 1;
for (; ;){
    System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
}

Визначенийие цикла осталось тем же, лише теперь блоки в визначенийии у нас пустые: for (; ;). Теперь нет инициализированной змінної-счетчика, нет условия, поцему цикл буде працювати вечно - бесконечный цикл.

Або можна опустити ряд блоків:


int i = 1;
for (; i<9;){
    System.out.printf("Квадрат числа %d равен %d \n", i, i * i);
	i++;
}

Цей приклад еквівалентний першому прикладу: ми також маємо лічильник, тільки створений він поза циклом. Ми маємо умову виконання циклу. І є збільшення лічильника вже в самому блоці for.

Цикл for може визначати відразу кілька змінних та керувати ними:


int n = 10;
for(int i=0, j = n - 1; i < j; i++, j--){
			
	System.out.println(i * j);
}

Вкладені цикли

Цикли можуть бути вкладені. Наприклад:


class Program{

    public static void main(String[] args) {  

        // внешний цикл
        for (int i = 1; i <= 9; i++) {
            // внутренний цикл
            for (int j = 1; j <= 9; j++) {
                System.out.print(i * j); 
                System.out.print('\t'); 
            }
            System.out.println(); // Перемещаемся на новую рядок
        }
    }
}

Здесь у нас есть внешний цикл (цикл for (int i = 1; i <= 9; i++) ), який управляет кількістьм рядків. Он начинается с i = 1 и продолжается, поки i не достигнет значення 9.

На каждой итерации внешнего цикла мы вивідим одну рядок. Всередині внешнего цикла у нас есть другой цикл (цикл for (int j = 1; j <= 9; j++)). Он начинается с j = 1 и продолжается, поки j не достигнет значення 9. На каждой итерации внутреннего цикла мы вивідим звездочку произведение i * j и табуляцию для отступа. Після завершення внутреннего цикла мы переходим к следующей рядківе (System.out.println()), чтобы начать новую рядок. В результаті вложенных циклов мы получим некое підобие таблиці умножения.

Цикл do

Цикл do спочатку виконує код циклу, а потім перевіряє умову в інструкції while. І поки ця умова істинна, цикл повторюється. Наприклад:


int j = 7;
do{
    System.out.println(j);
    j--;
}
while (j > 0);

У цьому випадку код циклу виконуватиметься 7 разів, поки j не дорівнюватиме нулю. Важливо відзначити, що цикл do гарантує, що дії будутьь виконані принаймні один раз. навіть якщо умова в операторі while не є істинною. Отже, ми можемоо написати:


int j = -1;
do{
    System.out.println(j);
    j--;
}
while (j > 0);

Незважаючи на те, що спочатку j менше 0, цикл все одно буде виконано один раз.

цикл while

Цикл while відразу проверяет истинность некоторого условия, и якщо условие истинно, то код цикла виконується:


int j = 6;
while (j > 0){

    System.out.println(j);
    j--;
}

Продовжити та розірвати оператори

Оператор перервати дозволяє выйти из цикла в любой его момент, даже якщо цикл не закончил свою работу:

Наприклад:


for (int i = 0; i < 10; i++){
    if (i == 5)
        break;
    System.out.println(i);
}

Коли лічильник досягне 5, спрацює оператор break і цикл завершиться.

Теперь сделаем так, чтобы якщо число равно 5, цикл не завершался, а просто переходил к следующей итерации. Для цього використовуємо оператор продовжити:


for (int i = 0; i < 10; i++){
    if (i == 5)
        continue;
    System.out.println(i);
}

У цьому випадку, коли цикл дійде до числа 5, програма просто пропустить це число і перейде до наступного.
