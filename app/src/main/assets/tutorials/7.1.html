Введення у рядки. Клас String

Останнє оновлення: 16.09.2025

Строка представляє собой последовательность символов. Для работы со строками в Java определен класс String, який надає ряд методів для манипуляции строками. Физически об'єкт String представляє собой ссылку на область в памяти, в которой размщены символы.

Для создания новой строки мы можемо використовувати один из конструкторов класса String, або напрямую присвоить строку в двойных кавычках:


public static void main(String[] args) {
        
    String str1 = "Java";
    String str2 = new String(); // пустая строка
    String str3 = new String(new char[] {'h', 'e', 'l', 'l', 'o'});
    String str4 = new String(new char[]{'w', 'e', 'l', 'c', 'o', 'm', 'e'}, 3, 4);//3 -начальный индекс, 4 -кол-во символов
        
    System.out.println(str1); // Java
    System.out.println(str2); //
    System.out.println(str3); // hello
    System.out.println(str4); // come
}

При работе со строками важно понимать, что об'єкт String є неизменяемым (immutable). То есть при любых операциях над строкой, которые изменяют эту строку, фактически буде создаваться новая строка.

Длина строк. Пустые строки и null

Поскольку строка рассматривается як набор символов, то мы можемо применить метод length() для нахождения длины строки або длины набора символов:


String str1 = "Java";
System.out.println(str1.length()); // 4

А за допомогою метода toCharArray() можна обратно преобразовать строку в массив символов:


String str1 = new String(new char[] {'h', 'e', 'l', 'l', 'o'});
char[] helloArray = str1.toCharArray();

Строка может быть пустой. Для цього ей можна присвоить пустые кавычки або удалить из стоки все символы:


String s = "";   // пустая строка
if(s.length() == 0) System.out.println("String is empty");

В цем случае длина строки, возвращаемая методом length(), равна 0.

Класс String имїїт специальный метод, який дозволяє проверить строку на пустоту - isEmpty(). Если строка пуста, он повертає true:


String s = "";   // пустая строка
if(s.isEmpty()) System.out.println("String is empty");

Переменная String может не указывать на якой-або об'єкт и иметь значение null:


String s = null;   // строка не указывает на об'єкт
if(s == null) System.out.println("String is null");

Значение null не эквивалентно пустой строке. Наприклад, в следующем случае мы столкнемся с ошибкой виконання:


String s = null;   // строка не указывает на об'єкт
if(s.length()==0) System.out.println("String is empty");	// ! Ошибка

Так як переменная не указывает ни на якой об'єкт String, то соответственно мы не можемо обращаться к методам об'єкта String. Щоб избежать подобных ошибок, можна предварительно проверять строку на null:


String s = null;   // строка не указывает на об'єкт
if(s==null || s.length()==0) System.out.println("String is empty");

Основные методы класса String

Основні операції з рядками раскрывается через методы класса String, среди которых можна выделить следующие:

  • concat(): об'єднує строки

  • valueOf(): преобразует об'єкт в строковый вид

  • join(): соединяет строки с учетом разделителя

  • сompareTo(): сравнивает две строки

  • charAt(): повертає символ строки по индексу

  • getChars(): повертає группу символов

  • equals(): сравнивает строки с учетом регистра

  • equalsIgnoreCase(): сравнивает строки без учета регистра

  • regionMatches(): сравнивает подстроки в строках

  • indexOf(): находит индекс первого вхождения подстроки в строку

  • lastIndexOf(): находит индекс последнего вхождения подстроки в строку

  • startsWith(): определяет, начинается ли строка с подстроки

  • endsWith(): определяет, заканчивается ли строка на определенную подстроку

  • replace(): заменяет в строке одну подстроку на другую

  • strip(): видаляє начальные и конечные пробелы

  • substring(): повертає подстроку, начиная с определенного индекса до конца або до определенного индекса

  • toLowerCase(): переводит все символы строки в нижний регистр

  • toUpperCase(): переводит все символы строки в верхний регистр
