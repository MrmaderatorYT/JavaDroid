Введення у рядки. Клас String

Останнє оновлення: 16.09.2025

Рядок представляє собой післядовательность символів. Для работы со рядокми в Java визначений клас String, який надає ряд методів для манипуляции рядокми. Физически об'єкт String представляє собой ссылку на область в памяти, в которой размщены символи.

Для створенняя новой рядки мы можемо використовувати один из конструкторов класа String, або напрямую присвоить рядок в двойных кавычках:


public static void main(String[] args) {
        
    String str1 = "Java";
    String str2 = new String(); // пустая рядок
    String str3 = new String(new char[] {'h', 'e', 'l', 'l', 'o'});
    String str4 = new String(new char[]{'w', 'e', 'l', 'c', 'o', 'm', 'e'}, 3, 4);//3 -начальный индекс, 4 -кол-во символів
        
    System.out.println(str1); // Java
    System.out.println(str2); //
    System.out.println(str3); // hello
    System.out.println(str4); // come
}

При работе со рядокми важливо понимать, что об'єкт String є незмінюємоым (immutable). То есть при любых операціях над рядківой, которые змінюють эту рядок, фактически буде создаваться новая рядок.

Длина рядків. Пустые рядки и null

Поскольку рядок рассматривается як набор символів, то мы можемо пріменіть метод length() для нахождения длины рядки або длины набора символів:


String str1 = "Java";
System.out.println(str1.length()); // 4

А за допомогою методу toCharArray() можна обратно преобразовать рядок в масив символів:


String str1 = new String(new char[] {'h', 'e', 'l', 'l', 'o'});
char[] helloArray = str1.toCharArray();

Рядок может быть порожній. Для цього ей можна присвоить пустые кавычки або удалить из стоки все символи:


String s = "";   // пустая рядок
if(s.length() == 0) System.out.println("String is empty");

В цем случае длина рядки, повертаємоая методом length(), равна 0.

Клас String имїїт специальный метод, який дозволяє проверить рядок на пустоту - isEmpty(). Если рядок пуста, он повертає true:


String s = "";   // пустая рядок
if(s.isEmpty()) System.out.println("String is empty");

Змінна String может не указывать на якой-або об'єкт и иметь значення null:


String s = null;   // рядок не вказує на об'єкт
if(s == null) System.out.println("String is null");

Значення null не эквивалентно порожній рядківе. Наприклад, в следующем случае мы столкнемся с ошибкой виконання:


String s = null;   // рядок не вказує на об'єкт
if(s.length()==0) System.out.println("String is empty");	// ! Ошибка

Так як змінна не вказує ни на якой об'єкт String, то соответственно мы не можемо обращаться к методум об'єкта String. Щоб избежать підобных ошибок, можна предварительно проверять рядок на null:


String s = null;   // рядок не вказує на об'єкт
if(s==null || s.length()==0) System.out.println("String is empty");

Основные методи класа String

Основні операції з рядками раскрывается через методи класа String, среди которых можна выделить следующие:

  • concat(): об'єднує рядки

  • valueOf(): преобразует об'єкт в рядківовый вид

  • join(): соединяет рядки с учетом розділителя

  • сompareTo(): сравнивает две рядки

  • charAt(): повертає символ рядки по индексу

  • getChars(): повертає группу символів

  • equals(): сравнивает рядки с учетом регистра

  • equalsIgnoreCase(): сравнивает рядки без учета регистра

  • regionMatches(): сравнивает підрядки в рядокх

  • indexOf(): находит индекс первого вхождения підрядки в рядок

  • lastIndexOf(): находит индекс післяднего вхождения підрядки в рядок

  • startsWith(): визначає, начинается ли рядок с підрядки

  • endsWith(): визначає, заканчивается ли рядок на визначенийную підрядок

  • replace(): заменяет в рядківе одну підрядок на другую

  • strip(): видаляє начальные и конечные пробелы

  • substring(): повертає підрядок, начиная с визначенийного индекса до конца або до визначенийного индекса

  • toLowerCase(): переводит все символи рядки в нижний регистр

  • toUpperCase(): переводит все символи рядки в верхний регистр
