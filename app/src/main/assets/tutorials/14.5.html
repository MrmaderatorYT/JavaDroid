Клас Constructor та створення об'єктів

Останнє оновлення: 29.09.2025

Класс Constructor из пакета java.lang.reflect представляє конструктор класса. Он надає функционал для получения информации о конструкторе, а також возможнасти для создания новых об'єктов. Отмечу основные его возможнасти:

  • String getName(): повертає имя конструктора

  • int getModifiers(): повертає целое число с различными включенными и выключенными битами, которые описують используемые модификаторы.

  • Object newInstance(Object... initargs): створює новый экземпляр класса, передавая конструктору аргументы через параметр initargs.

  • Class[] getParameterTypes(): повертає массив об'єктов Class, которые представляють типы параметров.

  • Class[] getExceptionTypes(): повертає массив об'єктов Class, представляющих типы исключений, генерируемых методом.

Получение конструктора

Для получения конструктора класса в виде об'єкта Constructor застосовується ряд методів об'єкта Class:

  • Constructor[] getConstructors(): повертає массив об'єктов Constructor для открытых конструкторов (с модификатором public) данного класса

  • Constructor[] getDeclaredConstructors(): повертає массив об'єктов Constructor для всех конструкторов данного класса. Метод повертають массив нулевой длины, якщо таких конструкторов нет або якщо об'єкт представляє собой примитивный тип або массив.

  • Constructor getConstructor(Class... parameterTypes): повертає об'єкт Constructor для публичного конструктора. Параметр parameterTypes — це массив об'єктов Class, которые идентифицируют формальные типы параметров конструктора в порядке их оголошення. Благодаря цим определениям параметров Java может выбрать нужную версию, якщо определено декілька конструкторов. Если подобный публичный конструктор отсутствует, то метод генерує исключение NoSuchConstructorException

  • Constructor getDeclaredConstructor(String name): повертає об'єкт Constructor для любого конструктора класса. Параметр parameterTypes — це массив об'єктов Class, которые идентифицируют формальные типы параметров конструктора в порядке их оголошення. Благодаря цим определениям параметров Java может выбрать нужную версию, якщо определено декілька конструкторов. Если подобный публичный конструктор отсутствует, то метод генерує исключение NoSuchConstructorException

Наприклад, возьмем следующий класс:


class Person {
     
    private String name;
    private int age;

    Person(){
        this("Undefined", 1);
    }

    Person(String name, int age){ 

        this.name = name;
        this.age = age;
    }
    
    void print(){
        System.out.printf("Person. Name: %s;  Age: %d\n", name, age);
    }
}

Получим конструкторы цього класса:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Person.class;
        Constructor[] ctors = cl.getDeclaredConstructors();
        for(Constructor ctor : ctors){

            System.out.println(ctor);
        }
    }
}

Консольный вывод:


Person()
Person(java.lang.String,int)

Теперь получим один конструктор, исходя из его параметров. Параметы передаются в виде набора об'єктов Class, який представляють типы соответствующих параметров. Наприклад, в случае с классом Person второй конструктор приймає два параметра с типуми String и int. Получим цей конструктор:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Person.class;
        try{
            Constructor ctor = cl.getDeclaredConstructor(String.class, int.class);
            
            System.out.println(ctor);   // Person(java.lang.String,int)
        }
        catch(Exception ex){
            System.out.println(ex);
        }
    }
}

Получение параметров

С помощью метода getParameterTypes() можна отримати массив параметров, где каждый параметр представляє тип Class:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Person.class;
        Constructor[] ctors = cl.getDeclaredConstructors();
        for(Constructor ctor : ctors){

            // получаем имя конструктора
            String name = ctor.getName();
            System.out.print(name + "(");

            // получаем типы параметров
            Class[] params = ctor.getParameterTypes();
            for(Class p : params){  
                System.out.printf("%s, ", p.getName());
            }

            System.out.println(")");
        }
    }
}

Консольный вывод:


Person()
Person(java.lang.String, int, )

Створення об'єкта

С помощью метода Object newInstance(Object... initargs) можна создать новый об'єкт класса:


public T newInstance(Object... initargs)
              throws InstantiationException,
                    IllegalAccessException,
                    IllegalArgumentException,
                    InvocationTargetException

Параметры метода представляють набор об'єктов, которые предоставляют значения параметрам конструктора. Наприклад, создадим об'єкт Person:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Person.class;
        try{
            Constructor ctor = cl.getDeclaredConstructor(String.class, int.class);
            Person person = (Person) ctor.newInstance("Sam", 23);   
            person.print();     // Person. Name: Sam;  Age: 23
        }
        catch(Exception ex){
            System.out.println(ex);
        }
    }
}

Здесь сначала получаем конструктор с двумя параметрами:

Constructor ctor = cl.getDeclaredConstructor(String.class, int.class);

А затем используем полученный конструктор, передавая в метод newInstance() параметры для него

Person person = (Person) ctor.newInstance("Sam", 23);
