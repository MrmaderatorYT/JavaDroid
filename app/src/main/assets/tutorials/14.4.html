Модифікатори доступу и клас Modifier

Останнє оновлення: 29.09.2025

Класы Field (поле класа), Method (метод класа) и Constructor (конструктор класа) имеют метод getModifiers(), який повертає целое число с различными включенными и выключенными битами, которые описують використовуємоые модификаторы, такие як public и static. Затем можна використовувати статические методи класа Modifier для анализа целого числа, повертаємоого getModifiers():

  • boolean isAbstract(int mod): повертає true, якщо параметр mod містить модификатор abstract

  • boolean isFinal(int mod): повертає true, якщо параметр mod містить модификатор final

  • boolean isInterface(int mod): повертає true, якщо параметр mod містить модификатор interface

  • boolean isNative(int mod): повертає true, якщо параметр mod містить модификатор native

  • boolean isPrivate(int mod): повертає true, якщо параметр mod містить модификатор private

  • boolean isProtected(int mod): повертає true, якщо параметр mod містить модификатор protected

  • boolean isPublic(int mod): повертає true, якщо параметр mod містить модификатор public

  • boolean isStatic(int mod): повертає true, якщо параметр mod містить модификатор static

  • boolean isSynchronized(int mod): повертає true, якщо параметр mod містить модификатор synchronized

  • boolean isTransient(int mod): повертає true, якщо параметр mod містить модификатор transient

  • boolean isVolatile(int mod): повертає true, якщо параметр mod містить модификатор volatile

Исследуем модификаторы полей:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Person.class;

        // отримуємо все поля типу Person
        Field[] fields = cl.getDeclaredFields();

        // вивідим информацию о полях
        for(Field f : fields){
            // отримуємо модификаторы
            int mod = f.getModifiers();
            if(Modifier.isPublic(mod)){
                System.out.print("public ");
            }
            else if(Modifier.isProtected(mod)){
                System.out.print("protected ");
            }
            else if(Modifier.isPrivate(mod)){
                System.out.print("private ");
            }
            if(Modifier.isStatic(mod)){
                System.out.print("static ");
            }

            // отримуємо тип 
            String type = f.getType().getName();
            System.out.print(type + " ");
            // отримуємо ім'я
            String name = f.getName();
            System.out.println(name + ";");
        }
    }
}
class Person {
     
    public String name;
    private int id;
    private static int count;

    Person(String name){  
        this.name = name; 
        this.id = ++count;
    }
    
    void print(){
        System.out.printf("Person. Id: %d  Name: %s  \n", id,  name);
    }
}

У даному випадку отримуємо все поля класа Person. Затем в цикле переьираем каждое поле, отримуємо его модификаторы и смотрим, представляютьь они public, protected. private або static:

Затем отримуємо тип и ім'я поля и вивідим на консоль. В итоге мы получим следующий консольный вивід:


public java.lang.String name;
private int id;
private static int count;

Для вивіда модификаторов можна використовувати метод Modifier.toString(). Наприклад:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Person.class;

        // отримуємо все поля типу Person
        Field[] fields = cl.getDeclaredFields();

        // вивідим информацию о полях
        for(Field f : fields){
            // отримуємо модификаторы
            String modifiers = Modifier.toString(f.getModifiers());
             // отримуємо тип 
            String type = f.getType().getName();
            // отримуємо ім'я
            String name = f.getName();

            System.out.printf("%s %s %s;\n", modifiers, type, name);
        }
    }
}
class Person {
     
    public String name;
    private int id;
    private static int count;

    Person(String name){  
        this.name = name; 
        this.id = ++count;
    }
    
    void print(){
        System.out.printf("Person. Id: %d  Name: %s  \n", id,  name);
    }
}

Для получения модификаторов в виде рядки передаємо целочисленный код модификаторов в метод Modifier.toString()
