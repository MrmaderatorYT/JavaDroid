Обмін між потіками. Клас Exchanger

Останнє оновлення: 28.04.2018

Клас Exchanger призначений для обмена данными між потіками. Он є типизированным и типизируется типом данных, которыми потіки повинні обмениваться.

Обмен данными производится за допомогою единственного методу цього класа exchange():


V exchange(V x) throws InterruptedException
V exchange(V x, long timeout, TimeUnit unit) throws InterruptedException, TimeoutException

Параметр x представляє буфер данных для обмена. Вторая форма методу також визначає параметр timeout - час ожидания и unit - тип временных единиц, застосовуємоых для параметра timeout.

Данный клас дуже просто використовувати:


import java.util.concurrent.Exchanger;
 
public class Program {
 
    public static void main(String[] args) {
         
        Exchanger<String> ex = new Exchanger<String>();
        new Thread(new PutThread(ex)).start();
        new Thread(new GetThread(ex)).start();
    }
}
 
class PutThread implements Runnable{
     
    Exchanger<String> exchanger;
    String message;
 
    PutThread(Exchanger<String> ex){
         
        this.exchanger=ex;
        message = "Hello Java!";
    }
    public void run(){
         
        try{
            message=exchanger.exchange(message);
            System.out.println("PutThread has received: " + message);
        }
        catch(InterruptedException ex){
            System.out.println(ex.getMessage());
        }
    }
} 
class GetThread implements Runnable{
     
    Exchanger<String> exchanger;
    String message;
 
    GetThread(Exchanger<String> ex){
         
        this.exchanger=ex;
        message = "Hello World!";
    }
    public void run(){
         
        try{
            message=exchanger.exchange(message);
            System.out.println("GetThread has received: " + message);
        }
        catch(InterruptedException ex){
            System.out.println(ex.getMessage());
        }
    }
}

В класе PutThread надсилає в буфер сообщение "Hello Java!":

message=exchanger.exchange(message);

Причем в ответ метод exchange повертає дані, которые отправил в буфер другой потік. То есть происходит обмен данными. Хотя нам необязательно получать дані, мы можемо просто их отправить:

exchanger.exchange(message);

Логика класа GetThread аналогична - також надсилаєся сообщение.

В итоге консоль выведет следующий результат:

PutThread has received: Hello World!
GetThread has received: Hello Java!
