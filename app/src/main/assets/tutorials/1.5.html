Перша програма у IntelliJ IDEA

Останнє оновлення: 20.03.2024

В прошлой теме мы рассмотрели, як создавать первую программу с понаступним її запуском в командной строке. Для упрощения разработки мы також можемо використовувати такие инструменты як IDE або интегрированные среды разработки, которые упрощают и ускоряют написание кода и створення приложений. На данный момент наиболїї популярной средой разработки для Java є IntelliJ IDEA от компании JetBrains. Рассмотрим, як використовувати данную среду.

Прежде всего загрузим установочный дистрибутив с официального сайта https://www.jetbrains.com/idea/download. По цему адресу можна найти пакеты для Windows, MacOS, Linux. Крім того, сама среда доступна в двух версиях - Ultimate (платная с триальным бесплатным периодом) и Community (бесплатная). В данном случае выберем бесплатную версию Community.

Зверніть увагу, що на сайті спочатку йде платна Ultimate-версія і лише потім безкоштовна Community-версія. Це середовище доступне як для платформи Intel x64, так і для платформи ARM64. Для Windows та MacOS доступні інсталяційні пакети, для Linux доступні лише архіви. Однак ряд ОС на Linux, наприклад, на Ubuntu, для встановлення можна використовувати пакетний менеджер snap та встановити середовище за допомогою команди:

sudo snap install intellij-idea-community --classic

Варто відзначити, Community-версія не має багатьох можливостей, які доступні в Ultimate-версії (зокрема, в Community недоступні опції для веб-застосунків на Java). Але Community-версія теж досить функціональна і теж дозволяє робити досить багато, у тому числі програми JavaFX і Android.

После установки запустим IntelliJ IDEA и создадим первый проект. Для цього на стартовом экране выберем New Project:

Далїї откроется окно создания проекта. В левой части в качестве типу проекта выберем New Project.

В поле Name укажем имя проекта. В моем случае проект буде называться HelloJava.

В поле Location можна указать путь к проекту, якщо не устраивает путь по умолчанию.

Поскольку мы будем работать с языком Java, в поле Language выберем пункт Java

Крім того, в поле JDK можна указать путь к Java SDK, який буде використовуватися в проекте. Как правило, це поле по умолчанию вже містить путь к JDK, який установлен на локальном компьютере. Если це поле пусто, то его треба установить.

Після цього натисніть кнопку Create. Після цього середа створить та відкриє проект.

В левой части мы можемо увидеть структуру проекта. Все файлы с исходным кодом помещаются в папку src. По умолчанию эта папка пуста, нияких файлов кода у нас в проекте поки нет. Тому добавим файл с исходным кодом. Для цього нажмем на папку src правой кнопкой мыши и в контекстном меню выберем пункт New -> Java Class:

После цього нам откроется небольшое окошко, в которое треба ввести имя класса. Пусть класс буде называться Program:

Після натискання клавіші Enter до папки src буде додано новий файл із класом java (у разі вище клас Program). А в центральній частині відкриється його вміст – власне вихідний код:

Змінимо код класу в такий спосіб:


public class Program {

    public static void main(String[] args) {

        System.out.println("Hello !");
    }
}

С помощью зеленой стрелки на панели инструментов або через меню Run -> Run... запустим проект.

І внизу IntelliJ IDEA з'явиться вікно виведення, де ми можемоо побачити результат роботи нашої програми.
