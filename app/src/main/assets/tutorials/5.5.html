Дерева TreeSet та інтерфейси SortedSet та NavigableSet

Останнє оновлення: 11.10.2025

SortedSet

Інтерфейс SortedSet призначений для створенняя колекцій, який хранят елементы в отсортированном виде (сортування по возрастанию). SortedSet расширяет інтерфейс Set, поцему такая колекція опять же хранит лише уникальные значення. SortedSet надає следующие методи:

  • E first(): повертає первый елемент набора

  • E last(): повертає післядний елемент набора

  • SortedSet<E> headSet(E end): повертає об'єкт SortedSet, який містить всі елементи первичного набора до елемента end

  • SortedSet<E> subSet(E start, E end): повертає об'єкт SortedSet, який містить всі елементи первичного набора між елементами start и end

  • SortedSet<E> tailSet(E start): повертає об'єкт SortedSet, який містить всі елементи первичного набора, начиная с елемента start

NavigableSet

Інтерфейс NavigableSet расширяет інтерфейс SortedSet и дозволяє извлекать елементы на основании их значений. NavigableSet визначає следующие методи:

  • E ceiling(E obj): ищет в наборе наименьший елемент e, який больше obj (e >=obj). Если такой елемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E floor(E obj): ищет в наборе наибольший елемент e, який меньше елемента obj (e <=obj). Если такой елемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E higher(E obj): ищет в наборе наименьший елемент e, який больше елемента obj (e >obj). Если такой елемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E lower(E obj): ищет в наборе наибольший елемент e, який меньше елемента obj (e <obj). Если такой елемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E pollFirst(): повертає первый елемент и видаляє его из набора

  • E pollLast(): повертає післядний елемент и видаляє его из набора

  • NavigableSet<E> descendingSet(): повертає об'єкт NavigableSet, який містить всі елементи первичного набора NavigableSet в обратном порядке

  • NavigableSet<E> headSet(E upperBound, boolean incl): повертає об'єкт NavigableSet, який містить все елементы первичного набора NavigableSet до upperBound. Параметр incl при значении true, дозволяє включить в выходной набор елемент upperBound

  • NavigableSet<E> tailSet(E lowerBound, boolean incl): повертає об'єкт NavigableSet, який містить все елементы первичного набора NavigableSet, начиная с lowerBound. Параметр incl при значении true, дозволяє включить в выходной набор елемент lowerBound

  • NavigableSet<E> subSet(E lowerBound, boolean lowerIncl, E upperBound, boolean highIncl): повертає об'єкт NavigableSet, який містить всі елементи первичного набора NavigableSet от lowerBound до upperBound.

TreeSet

Обобщенный клас TreeSet<E> представляє структуру данных в виде дерева, в котором все об'єкты хранятся в отсортированном виде за зростанням. TreeSet є наследником класа AbstractSet и реалізує інтерфейс NavigableSet, а следовательно, и інтерфейс SortedSet.

Додавання елемента в дерево происходит медленнїї, чем додавання его в хеш-таблицю HashSet, но всt равно намного быстрїї, чем проверка на наличие дубликатов в масиве або связном списке. Если дерево містить n елементов, то для поиска правильной позиции нового елемента требуется в среднем log2n сравнений. Наприклад, якщо дерево вже містить 1000 елементов, додавання нового елемента потребует около 10 сравнений.

И поскольку TreeSet при добавлении сравнивает об'єкты, они повинні реализовывать інтерфейс Comparable. Або необхідно предоставить об'єкт інтерфейса Comparator при створенні дерева TreeSet.

В класе TreeSet визначенийы следующие конструктори:

  • TreeSet(): створює пустое дерево

  • TreeSet(Collection<? extends E> col): створює дерево, в которое додає всі елементи колекції col

  • TreeSet(SortedSet <E> set): створює дерево, в которое додає всі елементи сортированного набора set

  • TreeSet(Comparator<? super E> comparator): створює пустое дерево, где все додаємоые елементы впіслядствии будуть отсортированы компаратором.

TreeSet піддерживает все стандартные методи для вставки и удаления елементов и ряд других методів:

  • boolean add(E e): Додає указанный елемент в цей набор, якщо он ще отсутствует

  • boolean addAll(Collection<? extends E> c): Додає всі елементи указанной колекції в цей набор

  • E ceiling(E e): Повертає наименьший елемент в цем наборе, больший або равный заданному, або null, якщо такого елемента нет

  • void clear(): Видаляє всі елементи из цього набора

  • Object clone(): Повертає поверхностную копию данного экземпляра TreeSet

  • Comparator<? super E> comparator(): Повертає компаратор, використовуємоый для упорядочивания елементов в цем наборе, або null, якщо набор використовує естественный порядок своих елементов

  • boolean contains(Object o): Повертає true, якщо набор містить указанный елемент

  • Iterator<E> descendingIterator(): Повертає ітератор по елементам набора в порядке убывания

  • NavigableSet<E> descendingSet(): Повертає обратный порядок елементов набора

  • E first(): Повертає первый (самый низкий) елемент в наборе

  • E floor(E e): Повертає наибольший елемент набора, меньший або равный заданному, або null, якщо такого елемента нет

  • SortedSet<E> headSet(E toElement): Повертає представление части набора, елементы которой строго меньше toElement

  • NavigableSet<E> headSet(E toElement, boolean inclusive): Повертає представление части данного множини, елементы которой меньше (или равны, якщо inclusive равно true) toElement

  • E higher(E e): Повертає наименьший елемент в данном множестве, строго больший заданного елемента, або null, якщо такого елемента нет

  • boolean isEmpty(): Повертає true, якщо в данном множестве нет елементов

  • Iterator<E> iterator(): Повертає ітератор по елементам данного множини в порядке возрастания

  • E last(): Повертає післядний (наивысший) елемент в данном множестве в данный момент

  • E lower(E e): Повертає наибольший елемент в данном множестве, строго меньший заданного елемента, або null, якщо такого елемента нет

  • E pollFirst(): Извлекает и видаляє первый (наименьший) елемент або повертає null, якщо данное множина пусто (необязательная операція)

  • E pollLast(): Извлекает и видаляє післядний (самый высокий) елемент або повертає null, якщо набор пуст (необязательная операція)

  • boolean remove(Object o): Видаляє указанный елемент из набора, якщо он присутствует

  • int size(): Повертає кількість елементів в дереве

  • NavigableSet<E> subSet(E fromElement, boolean fromInclusive, E toElement, boolean toInclusive): Повертає представление части набора, елементы которой находятся в диапазоне от fromElement до toElement

  • SortedSet<E> subSet(E fromElement, E toElement): Повертає представление части набора, елементы которой находятся в диапазоне от fromElement (включительно) до toElement (исключительно)

  • SortedSet<E> tailSet(E fromElement): Повертає представление части данного набора, елементы которой больше або равны fromElement

  • NavigableSet<E> tailSet(E fromElement, boolean inclusive): Повертає представление части данного набора, елементы которой больше (или равны, якщо inclusive равно true) fromElement


import java.util.TreeSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        TreeSet<String> langs = new TreeSet<String>();
          
        // додамо в список ряд елементов
        langs.add("Java");
        langs.add("JavaScript");
        langs.add("Python");
        langs.add("C++");
        
        System.out.printf("langs size: %d \n", langs.size());

        System.out.println(langs);  // [C++, Java, JavaScript, Python]
         
        // видалення елемента
        langs.remove("JavaScript");
        System.out.println(langs);  // [C++, Java, Python]
    }
}

И поскольку при вставке об'єкты відразу же ортуються за зростанням, то при вивіде мы получим отсортированный набор:


langs size: 4 
[C++, Java, JavaScript, Python]
[C++, Java, Python]

Так як TreeSet реалізує інтерфейс NavigableSet, а через него и SortedSet, то мы можемо пріменіть к структуре дерева различные методи для получения елементов:


import java.util.TreeSet;
import java.util.SortedSet;
import java.util.NavigableSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        TreeSet<String> langs = new TreeSet<String>();
          
        // додамо в список ряд елементов
        langs.add("Java");
        langs.add("JavaScript");
        langs.add("Python");
        langs.add("C#");
        langs.add("C++");
        langs.add("Go");
        
        // сначала вивідим все
        System.out.println(langs);      // [C#, C++, Go, Java, JavaScript, Python]

        // получим первый - самый меньший елемент
        System.out.println(langs.first());      // C#

        // получим післядний - самый больший елемент
        System.out.println(langs.last());       // Python

        // получим піднабор от одного елемента до другого
        SortedSet<String> set = langs.subSet("Go", "Python");
        System.out.println(set);                // [Go, Java, JavaScript]

        // елемент из набора, який идет перед JavaScript ("больше" JavaScript)
        String greater = langs.higher("JavaScript");    // Python
        System.out.println(greater);

        // елемент из набора, який идет перед Go ("меньше" Go)
        String lower = langs.lower("Go");
        System.out.println(lower);          // C++

        // повертаємо набор елементов, которые идут перед Go
        SortedSet<String> goHead=langs.headSet("Go");
        System.out.println(goHead);       // [C#, C++]

        // повертаємо набор елементов, которые идут начиная с JavaScript
        SortedSet<String> jsTail =langs.tailSet("JavaScript");  
        System.out.println(jsTail);     // [JavaScript, Python]

        // повертаємо набор в обратном порядке
        NavigableSet<String> reversedSet = langs.descendingSet();
        System.out.println(reversedSet);     // [Python, JavaScript, Java, Go, C++, C#]
    }
}
