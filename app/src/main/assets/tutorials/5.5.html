Дерева TreeSet та інтерфейси SortedSet та NavigableSet

Останнє оновлення: 11.10.2025

SortedSet

Интерфейс SortedSet призначений для создания коллекций, який хранят элементы в отсортированном виде (сортировка по возрастанию). SortedSet расширяет інтерфейс Set, поцему такая коллекция опять же хранит лише уникальные значения. SortedSet надає следующие методы:

  • E first(): повертає первый элемент набора

  • E last(): повертає последний элемент набора

  • SortedSet<E> headSet(E end): повертає об'єкт SortedSet, який містить все элементы первичного набора до элемента end

  • SortedSet<E> subSet(E start, E end): повертає об'єкт SortedSet, який містить все элементы первичного набора между элементами start и end

  • SortedSet<E> tailSet(E start): повертає об'єкт SortedSet, який містить все элементы первичного набора, начиная с элемента start

NavigableSet

Интерфейс NavigableSet расширяет інтерфейс SortedSet и дозволяє извлекать элементы на основании их значений. NavigableSet определяет следующие методы:

  • E ceiling(E obj): ищет в наборе наименьший элемент e, який больше obj (e >=obj). Если такой элемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E floor(E obj): ищет в наборе наибольший элемент e, який меньше элемента obj (e <=obj). Если такой элемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E higher(E obj): ищет в наборе наименьший элемент e, який больше элемента obj (e >obj). Если такой элемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E lower(E obj): ищет в наборе наибольший элемент e, який меньше элемента obj (e <obj). Если такой элемент найден, то он повертаєся в качестве результата. Иначе повертаєся null.

  • E pollFirst(): повертає первый элемент и видаляє его из набора

  • E pollLast(): повертає последний элемент и видаляє его из набора

  • NavigableSet<E> descendingSet(): повертає об'єкт NavigableSet, який містить все элементы первичного набора NavigableSet в обратном порядке

  • NavigableSet<E> headSet(E upperBound, boolean incl): повертає об'єкт NavigableSet, який містить все элементы первичного набора NavigableSet до upperBound. Параметр incl при значении true, дозволяє включить в выходной набор элемент upperBound

  • NavigableSet<E> tailSet(E lowerBound, boolean incl): повертає об'єкт NavigableSet, який містить все элементы первичного набора NavigableSet, начиная с lowerBound. Параметр incl при значении true, дозволяє включить в выходной набор элемент lowerBound

  • NavigableSet<E> subSet(E lowerBound, boolean lowerIncl, E upperBound, boolean highIncl): повертає об'єкт NavigableSet, який містить все элементы первичного набора NavigableSet от lowerBound до upperBound.

TreeSet

Обобщенный класс TreeSet<E> представляє структуру данных в виде дерева, в котором все об'єкты хранятся в отсортированном виде по возрастанию. TreeSet є наследником класса AbstractSet и реализует інтерфейс NavigableSet, а следовательно, и інтерфейс SortedSet.

Добавление элемента в дерево происходит медленнїї, чем добавление его в хеш-таблицу HashSet, но всt равно намного быстрїї, чем проверка на наличие дубликатов в массиве або связном списке. Если дерево містить n элементов, то для поиска правильной позиции нового элемента требуется в среднем log2n сравнений. Наприклад, якщо дерево вже містить 1000 элементов, добавление нового элемента потребует около 10 сравнений.

И поскольку TreeSet при добавлении сравнивает об'єкты, они повинні реализовывать інтерфейс Comparable. Або необхідно предоставить об'єкт інтерфейса Comparator при создании дерева TreeSet.

В классе TreeSet определены следующие конструкторы:

  • TreeSet(): створює пустое дерево

  • TreeSet(Collection<? extends E> col): створює дерево, в которое додає все элементы коллекции col

  • TreeSet(SortedSet <E> set): створює дерево, в которое додає все элементы сортированного набора set

  • TreeSet(Comparator<? super E> comparator): створює пустое дерево, где все добавляемые элементы впоследствии будуть отсортированы компаратором.

TreeSet поддерживает все стандартные методы для вставки и удаления элементов и ряд других методів:

  • boolean add(E e): Додає указанный элемент в цей набор, якщо он ще отсутствует

  • boolean addAll(Collection<? extends E> c): Додає все элементы указанной коллекции в цей набор

  • E ceiling(E e): Повертає наименьший элемент в цем наборе, больший або равный заданному, або null, якщо такого элемента нет

  • void clear(): Видаляє все элементы из цього набора

  • Object clone(): Повертає поверхностную копию данного экземпляра TreeSet

  • Comparator<? super E> comparator(): Повертає компаратор, используемый для упорядочивания элементов в цем наборе, або null, якщо набор использует естественный порядок своих элементов

  • boolean contains(Object o): Повертає true, якщо набор містить указанный элемент

  • Iterator<E> descendingIterator(): Повертає итератор по элементам набора в порядке убывания

  • NavigableSet<E> descendingSet(): Повертає обратный порядок элементов набора

  • E first(): Повертає первый (самый низкий) элемент в наборе

  • E floor(E e): Повертає наибольший элемент набора, меньший або равный заданному, або null, якщо такого элемента нет

  • SortedSet<E> headSet(E toElement): Повертає представление части набора, элементы которой строго меньше toElement

  • NavigableSet<E> headSet(E toElement, boolean inclusive): Повертає представление части данного множества, элементы которой меньше (или равны, якщо inclusive равно true) toElement

  • E higher(E e): Повертає наименьший элемент в данном множестве, строго больший заданного элемента, або null, якщо такого элемента нет

  • boolean isEmpty(): Повертає true, якщо в данном множестве нет элементов

  • Iterator<E> iterator(): Повертає итератор по элементам данного множества в порядке возрастания

  • E last(): Повертає последний (наивысший) элемент в данном множестве в данный момент

  • E lower(E e): Повертає наибольший элемент в данном множестве, строго меньший заданного элемента, або null, якщо такого элемента нет

  • E pollFirst(): Извлекает и видаляє первый (наименьший) элемент або повертає null, якщо данное множество пусто (необязательная операция)

  • E pollLast(): Извлекает и видаляє последний (самый высокий) элемент або повертає null, якщо набор пуст (необязательная операция)

  • boolean remove(Object o): Видаляє указанный элемент из набора, якщо он присутствует

  • int size(): Повертає количество элементов в дереве

  • NavigableSet<E> subSet(E fromElement, boolean fromInclusive, E toElement, boolean toInclusive): Повертає представление части набора, элементы которой находятся в диапазоне от fromElement до toElement

  • SortedSet<E> subSet(E fromElement, E toElement): Повертає представление части набора, элементы которой находятся в диапазоне от fromElement (включительно) до toElement (исключительно)

  • SortedSet<E> tailSet(E fromElement): Повертає представление части данного набора, элементы которой больше або равны fromElement

  • NavigableSet<E> tailSet(E fromElement, boolean inclusive): Повертає представление части данного набора, элементы которой больше (или равны, якщо inclusive равно true) fromElement


import java.util.TreeSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        TreeSet<String> langs = new TreeSet<String>();
          
        // добавим в список ряд элементов
        langs.add("Java");
        langs.add("JavaScript");
        langs.add("Python");
        langs.add("C++");
        
        System.out.printf("langs size: %d \n", langs.size());

        System.out.println(langs);  // [C++, Java, JavaScript, Python]
         
        // удаление элемента
        langs.remove("JavaScript");
        System.out.println(langs);  // [C++, Java, Python]
    }
}

И поскольку при вставке об'єкты відразу же сортируются по возрастанию, то при выводе мы получим отсортированный набор:


langs size: 4 
[C++, Java, JavaScript, Python]
[C++, Java, Python]

Так як TreeSet реализует інтерфейс NavigableSet, а через него и SortedSet, то мы можемо применить к структуре дерева различные методы для получения элементов:


import java.util.TreeSet;
import java.util.SortedSet;
import java.util.NavigableSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        TreeSet<String> langs = new TreeSet<String>();
          
        // добавим в список ряд элементов
        langs.add("Java");
        langs.add("JavaScript");
        langs.add("Python");
        langs.add("C#");
        langs.add("C++");
        langs.add("Go");
        
        // сначала выводим все
        System.out.println(langs);      // [C#, C++, Go, Java, JavaScript, Python]

        // получим первый - самый меньший элемент
        System.out.println(langs.first());      // C#

        // получим последний - самый больший элемент
        System.out.println(langs.last());       // Python

        // получим поднабор от одного элемента до другого
        SortedSet<String> set = langs.subSet("Go", "Python");
        System.out.println(set);                // [Go, Java, JavaScript]

        // элемент из набора, який идет перед JavaScript ("больше" JavaScript)
        String greater = langs.higher("JavaScript");    // Python
        System.out.println(greater);

        // элемент из набора, який идет перед Go ("меньше" Go)
        String lower = langs.lower("Go");
        System.out.println(lower);          // C++

        // возвращаем набор элементов, которые идут перед Go
        SortedSet<String> goHead=langs.headSet("Go");
        System.out.println(goHead);       // [C#, C++]

        // возвращаем набор элементов, которые идут начиная с JavaScript
        SortedSet<String> jsTail =langs.tailSet("JavaScript");  
        System.out.println(jsTail);     // [JavaScript, Python]

        // возвращаем набор в обратном порядке
        NavigableSet<String> reversedSet = langs.descendingSet();
        System.out.println(reversedSet);     // [Python, JavaScript, Java, Go, C++, C#]
    }
}
