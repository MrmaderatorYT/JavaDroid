Встановлення JDK

Останнє оновлення: 16.09.2025

Для розробки програм на Java нам зтребабиться спеціальний комплект для розробки JDK (Java Development Kit). JDK включає ряд програм та утиліт, які дозволяють компілювати, запускати програми на Java, а також виконувати низку інших функцій.

JDK от Oracle для Windows (x64), Linux (x64, ARM64) и MacOS (x64, ARM64) доступны по адресу https://www.oracle.com/java/technologies/downloads/

OpenJDK для Windows (x64), Linux (x64, ARM64) и MacOS (x64, ARM64) доступны по адресу https://jdk.java.net/25/

OpenJDK надає лише архіви, Oracle також надає інсталяційні пакети. Розглянемо інсталяцію JDK для різних систем.

Установка Windows

Установка за допомогою установника

Наиболїї распространенный вариант установки JDK на Windows - це применение установщика с официального сайта Oracle. В частности, перейдем на страницу: https://www.oracle.com/java/technologies/downloads/#jdk25-windows

Натисніть кнопку Next. На наступному екрані необхідно вказати, в яку папку проводитиметься установка:

Залишимо вибір папки за замовчуванням і натисніть Next для виконання установки.

Після завершення установки JDK ми побачимо таке вікно:

При установке JDK все файлы по умолчанию помещаются в каталог C:\Program Files\Java\jdk-[номер_версії] (якщо при установке не буво изменено расположение). Так, якщо мы откроем в нем подкаталог bin, то мы сможемо увидеть в нем ряд утилит. Нас прежде всего интересует утилита компилятора javac и утилита java.exe. javac компилирует код на мові Java в бинарный файл, а java.exe дозволяє запускать скомпилированный файл:

Установка через пакетний менеджер

Через пакетний менеджер winget можна також встановити OpenJDK від Microsoft. Для цього у командному рядку/PowerShell треба виконати команду:

winget install Microsoft.OpenJDK.25

У цьому випадку встановлюється 25-версія JDK.

Встановлення на Linux

Если мы используем JDK от Oracle, то в данном случае нам доступен ряд вариантов установки. В частности, перейдем на страницу: https://www.oracle.com/java/technologies/downloads/#jdk25-linux

  • ARM64 Compressed Archive: архив для систем на архитектуре ARM64

  • ARM64 RPM Package: RPM-пакет для систем на архитектуре ARM64, которые для управления пакетами применяют RPM (наприклад, Fedora, Suse и т.д.)

  • x64 Compressed Archive: архив для систем на архитектуре Intel x86-64

  • x64 Debian Package: Debian-пакет для систем на архитектуре Intel x86-64 (наприклад, Ubuntu, Debian и т.д.)

  • x64 RPM Package: RPM-пакет для систем на архитектуре Intel x86-64, которые для управления пакетами применяют RPM (наприклад, Fedora, Suse и т.д.)

Если наша операционная система Ubuntu (на платформе Intel x86-64), то выберем пункт x64 Debian Package, и нам загрузится debian-пакет (в моем случае файл "jdk-25_linux-x64_bin.deb"). Для его установки выполним команду

sudo dpkg -i jdk-25_linux-x64_bin.deb

Если у нас операционная система, которая для управления пакетами применяет RPM, наприклад, Fedora, Suse и др, на платформе Intel x86-64, то необхідно выбрать пункт x64 RPM Package. И затем установить загрвженный пакет командой

sudo rpm -ivh jdk-25_linux-x64_bin.rpm

Если у нас ОС Fedora, Suse и др, на платформе ARM64, то необхідно выбрать пункт ARM64 RPM Package. И затем установить загрвженный пакет командой

sudo rpm -ivh jdk-25_linux-aarch64_bin.rpm

Також для різних операційних систем репозиторії можуть бути доступні пакети JDK для установки. Наприклад, встановлення за допомогою пакетного менеджера snap:

sudo snap install openjdk

Установка за допомогою пакетного менеджера apt:

sudo apt install default-jdk

Встановлення певної версії JDK за допомогою пакетного менеджера apt:

sudo apt install openjdk-25-jdk-headless

У випадку встановлення архівів все ідентично для всіх основних систем на базі Linux - завантажується архів, розпаковується в певне місце. Потім змінні середовища додається шлях до папки bin розпакованого архіву.

Встановлення Java на MacOS

Для установки Java на MacOS перейдем на страницу загрузок на сайте Oracle по адресу https://www.oracle.com/java/technologies/downloads/#jdk25-mac:

Тут нам доступно 4 варіанти пакетів:

  • ARM64 Compressed Archive: архивный пакет для MacOS на ARM

  • ARM64 DMG Installer: установщик для MacOS на архитектуре ARM

  • x64 Compressed Archive: архивный пакет для MacOS на архитектуре Intel x86-64

  • x64 DMG Installer: установщик для MacOS на архитектуре Intel x86-64

Завантажимо та після завантаження запустимо пакет установника. Спочатку нам відобразиться вітальне вікно.

Далі на всіх наступних кроках залишимо всі налаштування за замовчуванням. І після встановлення нам відобразиться вікно про її успішне закінчення:

Перевірка установки

Після встановлення Java перейдемо до командного рядка/терміналу та для перевірки версії введемо команду

java -version

Консоль нам має у відповідь ввести номер щойно встановленої версії JDK (у моєму випадку 25 версії):


java version "25" 2025-09-16 LTS
Java(TM) SE Runtime Environment (build 25+37-LTS-3491)
Java HotSpot(TM) 64-Bit Server VM (build 25+37-LTS-3491, mixed mode, sharing)

Документація

Документация є важным вспомогательным элементом при разработке на любом языке программирования, и Java - не исключения. Всю документацию по Java можна найти на оф. сайте Oracle по адресу https://docs.oracle.com/en/java/javase/25/docs/ (для 25-й версии JDK, для других версий треба заменить номер на соответствующий).

Також документация доступна офлайн - її також можна загрузить с оф. сайта в виде архива со страницы https://www.oracle.com/java/technologies/javase-jdk25-doc-downloads.html (для 25-й версии JDK).
