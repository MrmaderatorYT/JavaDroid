Клас Console

Останнє оновлення: 28.12.2015

Специально для работы с консолью в Java определен класс Console, який хранится в пакеті java.io. Он не отримує консольный ввод-вывод сам по себе, а использует вже имеющиеся потоки System.in и System.out. Но в то же час Console значительно упрощает ряд операций, связанных с консолью.

Для получения об'єкта консоли треба вызвать статический метод System.console():


Console console = System.console();

Основные методы класса Console:

  • flush(): выводит на консоль все дані из буфера

  • format(): выводит на консоль строку с использованием форматирования

  • printf(): выводит на консоль строку с использованием форматирования (фактически то же самое, что и предыдущий метод)

  • String readLine(): считывает с консоли введенную пользователем строку

  • char[] readPassword(): считывает с консоли введенную пользователем строку, при цем символы строки не отображаются на консоли

Используем класс Console:


import java.io.Console;

public class Program {

    public static void main(String[] args) {
        
        // получаем консоль
        Console console = System.console();
		if(console!=null){
			// считываем дані с консоли
			String login = console.readLine("Введите логин:");
			char[] password = console.readPassword("Введите пароль:");
			
			console.printf("Введенный логин: %s \n", login);
			console.printf("Введенный пароль: %s \n", new String(password));
		}
    }
}

Важно, что доступ к консоли мы можемо отримати лише из самой консоли. При запуске, наприклад, в Netbeans вызов System.console() буде возвращать значение null.
