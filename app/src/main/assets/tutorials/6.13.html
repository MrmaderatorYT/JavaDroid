Клас Console

Останнє оновлення: 28.12.2015

Специально для работы с консолью в Java визначений клас Console, який хранится в пакеті java.io. Он не отримує консольный ввод-вивід сам по себе, а використовує вже имеющиеся потіки System.in и System.out. Но в то же час Console значительно упрощает ряд операцій, связанных с консолью.

Для получения об'єкта консоли треба вызвать статический метод System.console():


Console console = System.console();

Основные методи класа Console:

  • flush(): виводить на консоль все дані из буфера

  • format(): виводить на консоль рядок с використанням форматирования

  • printf(): виводить на консоль рядок с використанням форматирования (фактически то же самое, что и предыдущий метод)

  • String readLine(): считывает с консоли введенную пользователем рядок

  • char[] readPassword(): считывает с консоли введенную пользователем рядок, при цем символи рядки не отображаются на консоли

Використовуємо клас Console:


import java.io.Console;

public class Program {

    public static void main(String[] args) {
        
        // отримуємо консоль
        Console console = System.console();
		if(console!=null){
			// считываем дані с консоли
			String login = console.readLine("Введите логин:");
			char[] password = console.readPassword("Введите пароль:");
			
			console.printf("Введенный логин: %s \n", login);
			console.printf("Введенный пароль: %s \n", new String(password));
		}
    }
}

Важливо, что доступ к консоли мы можемо отримати лише из самой консоли. При запуску, наприклад, в Netbeans вызов System.console() буде возвращать значення null.
