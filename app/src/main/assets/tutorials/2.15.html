Операції присвоєння та пріоритет операцій

Останнє оновлення: 16.04.2018

На завершення розглянемо операції присвоєння, які переважливо представляютььь комбінацію простого присвоєння коїться з іншими операціями:

  • =

    просто приравнивает одно значення іншому: c=b;

  • +=

    c+=b; (змінної c присвоюється результат сложения c и b)

  • -=

    c-=b; (змінної c присвоюється результат вычитания b из c)

  • *=

    c*=b; (змінної c присвоюється результат произведения c и b)

  • /=

    c/=b; (змінної c присвоюється результат деления c на b)

  • %=

    c%=b; (змінної c присвоюється остаток от деления c на b)

  • &=

    c&=b; (змінної c присвоюється значення c&b)

  • |=

    c|=b; (змінної c присвоюється значення c|b)

  • ^=

    c^=b; (змінної c присвоюється значення c^b)

  • <<=

    c<<=b; (змінної c присвоюється значення c<<b)

  • >>=

    c>>=b; (змінної c присвоюється значення c>>b)

  • >>>=

    c>>>=b; (змінної c присвоюється значення c>>>b)

Приклади операцій:


int a = 5;
a += 10;        // 15
a -= 3;         // 12
a *= 2;         // 24
a /= 6;         // 4
a <<= 4;      // 64
a >>= 2;      // 16
System.out.println(a);	// 16

Пріоритет операцій

При роботі з операціями важливо розуміти їхній пріоритет, який можна описати наступною таблицею:

expr++ expr--

++expr --expr +expr -expr~!

*/%

+ -

<< >> >>>

< > <= >= instanceof

== !=

&

^

&&

||

? : (Тернарний оператор)

= += -= *= /= %= &= ^= |= <<= >>= >>>= (оператори присвоєння)

Що оператор у цій таблиці, то більше вписувалося його пріоритет. При цьому дужки підвищують пріоритет операції, що використовується у виразі.
