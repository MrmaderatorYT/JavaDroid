Інтерфейс Set та хеш-таблиці HashSet

Останнє оновлення: 11.10.2025

Интерфейс Set расширяет інтерфейс Collection и представляє набор уникальных элементов. Set почти не додає новых методів, лише вносит изменения в унаследованные. В частности, метод add() додає элемент в коллекцию и повертає true, якщо в коллекции ще нет такого элемента.

Добавленных методів в Set йде відзначити следующие два статических метода:

  • static <E> Set<E>of(E... elements): повертає неизменяемый набор из элементов elements

  • static <E> Set<E>copyOf(Collection<? extends E> col): повертає неизменяемый набор, в яку додає все элементы коллекции col

Наприклад, используем метод of() для создания набора:


import java.util.Set;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> people = Set.<String>of("Tom", "Bob", "Sam");

        System.out.println(people); // [Tom, Sam, Bob]
    }
}

Метод Set.of() створює неизменяемый набор типу class java.util.ImmutableCollections$SetN.

Отличительной особенностью наборов є то, что все элементы в нем повинні быть уникальными. Для сравнения на равенстро застосовується метод equals(), який имеют все классы. Так, якщо мы попробуем добавить в набор два одинаковых значения, то при выполнении буде сгенерирована ошибка:


import java.util.Set;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> people = Set.<String>of("Tom", "Bob", "Sam", "Tom");  // Ошибка! duplicate element: Tom

        System.out.println(people);
    }
}

HashSet

Расспространенной реализацией інтерфейса Set є обобщенный класс HashSet, який представляє хеш-таблицу. Если быть точнїї, он найде свой функционал от класса AbstractSet, який, в свою очередь, реализует інтерфейс Set.

Хеш-таблица представляє такую структуру данных, в которой все об'єкты имеют уникальный ключ або хеш-код. Данный ключ дозволяє уникально идентифицировать об'єкт в таблице. Хеш-код представляє целое число, которое хеш-таблица вычисляет для каждого добавляемого об'єкта за допомогою его метода hashCode(). Хеш-код обычно виводиться из полей экземпляра об'єкта, предпочтительно таким чином, чтобы об'єкты с разными данными давали разные коды. И також важно, чтобы хэш-коды вычислялись быстро, и чтобы вычисление зависело лише от состояния об'єкта, який необхідно хэшировать.

Для создания об'єкта HashSet можна воспользоваться одним из следующих конструкторов:

  • HashSet(): створює порожній список

  • HashSet(Collection<? extends E> col): створює хеш-таблицу, в яку додає все элементы коллекции col

  • HashSet(int capacity): параметр capacity указывает начальную емкость таблицы, которая по умолчанию равна 16

  • HashSet(int capacity, float koef): параметр koef або коэффициент заполнения, значение которого повинно быть в пределах от 0.0 до 1.0, указывает, насколько повинна быть заполнена емкость об'єктами прежде чем произойдет її расширение. Наприклад, коэффициент 0.75 указывает, что при заполнении емкости на 3/4 произойдет її расширение.

Створення хеш-таблицы:


import java.util.HashSet;

public class Program{
     
	public static void main(String[] args) {
         
        HashSet<String> langs = new HashSet<String>();
    }
}

Створення хеш-таблицы по имеющейся коллекции (из неизменяемого набора):


import java.util.Set;
import java.util.HashSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> data = Set.<String>of("Java", "JavaScript", "C#");

        HashSet<String> langs = new HashSet<String>(data);

        System.out.println(langs); // [C#, Java, JavaScript]
    }
}

Класс HashSet в основном реализует те методы, которые оголошенийы в родительских классах и применяемых інтерфейсах. Приведу основные методы HashSet:

  • boolean add(E e): Додає указанный элемент в цей набор, якщо он ще отсутствует

  • void clear(): Видаляє все элементы из цього набора

  • Object clone(): Повертає поверхностную копию данного экземпляра HashSet: сами элементы не клонируются

  • boolean contains(Object o): Повертає true, якщо цей набор містить указанный элемент

  • boolean isEmpty(): Повертає true, якщо цей набор не містить элементов

  • Iterator<E> iterator(): Повертає итератор по элементам цього набора

  • static <T> HashSet<T> newHashSet(int numElements): Створює новый пустой HashSet, подходящий для ожидаемого количества элементов

  • boolean remove(Object o): Видаляє указанный элемент из цього набора, якщо он присутствует

  • int size(): Повертає количество элементов в цем наборе

  • Object[] toArray(): Повертає массив, содержащий все элементы цей коллекции

С помощью метода add() можна добавить новый об'єкт в хеш-таблицу. Если такого элемента ще нет, то об'єкт додаєся, а метод повертає true. Если элемент вже есть, ничего не происходит, а метод повертає false:


import java.util.HashSet;
 
public class Program{
      
    public static void main(String[] args) {

        HashSet<String> langs = new HashSet<String>();
        // добавим ряд об'єктов
        langs.add("Java");
        langs.add("C++");
        langs.add("C#");
        // пытаемся добавить элемент, який вже есть в коллекции
        boolean isAdded = langs.add("Java");
        System.out.println(isAdded);    // false
        
        System.out.printf("langs size: %d \n", langs.size());    // langs size: 3
          
        System.out.println(langs); // [C#, Java, C++]
    }
}

Метод remove() видаляє об'єкт из хеш-таблицы. ПРи успешном удалении метод повертає true. Если об'єкт отсутствует, ничего не происходит, а метод повертає false. Для полного же удаления всех элементов застосовується метод clear(). Пример удаления:


import java.util.Set;
import java.util.HashSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> data = Set.<String>of("Java", "JavaScript", "C#", "Python");
        HashSet<String> langs = new HashSet<String>(data);
        // удаляем язык JavaScript
        boolean isRemoved = langs.remove("JavaScript");
        System.out.println("JavaScript removed: " + isRemoved);    // true

        // удаляем язык OCalm
        isRemoved = langs.remove("OCalm");
        System.out.println("OCalm removed: " + isRemoved);    // false
          
        System.out.println(langs); // [C#, Java, Python]

        // удаляем все
        langs.clear();
        System.out.println(langs); // []
    }
}
