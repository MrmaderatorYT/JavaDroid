Інтерфейс Set та хеш-таблиці HashSet

Останнє оновлення: 11.10.2025

Інтерфейс Set расширяет інтерфейс Collection и представляє набор уникальных елементов. Set почти не додає новых методів, лише вносит изменения в унаследованные. Зокрема, метод add() додає елемент в колекцію и повертає true, якщо в колекції ще нет такого елемента.

Добавленных методів в Set йде відзначити следующие два статических методу:

  • static <E> Set<E>of(E... elements): повертає незмінюємоый набор из елементов elements

  • static <E> Set<E>copyOf(Collection<? extends E> col): повертає незмінюємоый набор, в яку додає всі елементи колекції col

Наприклад, використовуємо метод of() для створенняя набора:


import java.util.Set;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> people = Set.<String>of("Tom", "Bob", "Sam");

        System.out.println(people); // [Tom, Sam, Bob]
    }
}

Метод Set.of() створює незмінюємоый набор типу class java.util.ImmutableCollections$SetN.

Отличительной особенностью наборов є то, что всі елементи в нем повинні быть уникальными. Для сравнения на равенстро застосовується метод equals(), який имеют все класы. Так, якщо мы попробуем добавить в набор два одинаковых значення, то при выполнении буде сгенерирована ошибка:


import java.util.Set;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> people = Set.<String>of("Tom", "Bob", "Sam", "Tom");  // Ошибка! duplicate element: Tom

        System.out.println(people);
    }
}

HashSet

Расспространенной реализацией інтерфейса Set є обобщенный клас HashSet, який представляє хеш-таблицю. Если быть точнїї, он найде свой функционал от класа AbstractSet, який, в свою очередь, реалізує інтерфейс Set.

Хеш-таблиця представляє такую структуру данных, в которой все об'єкты имеют уникальный ключ або хеш-код. Данный ключ дозволяє уникально идентифицировать об'єкт в таблице. Хеш-код представляє целое число, которое хеш-таблиця вычисляет для каждого додаємоого об'єкта за допомогою его методу hashCode(). Хеш-код обычно виводиться из полей экземпляра об'єкта, предпочтительно таким чином, чтобы об'єкты с разными данными давали разные коды. И також важливо, чтобы хэш-коды вычислялись быстро, и чтобы вычисление зависело лише от состояния об'єкта, який необхідно хэшировать.

Для створенняя об'єкта HashSet можна воспользоваться одним из следующих конструкторов:

  • HashSet(): створює порожній список

  • HashSet(Collection<? extends E> col): створює хеш-таблицю, в яку додає всі елементи колекції col

  • HashSet(int capacity): параметр capacity вказує начальную емкость таблиці, которая по умолчанию равна 16

  • HashSet(int capacity, float koef): параметр koef або коэффициент заполнения, значення которого повинно быть в пределах от 0.0 до 1.0, вказує, насколько повинна быть заполнена емкость об'єктами прежде чем произойдет її расширение. Наприклад, коэффициент 0.75 вказує, что при заполнении емкости на 3/4 произойдет її расширение.

Створення хеш-таблиці:


import java.util.HashSet;

public class Program{
     
	public static void main(String[] args) {
         
        HashSet<String> langs = new HashSet<String>();
    }
}

Створення хеш-таблиці по имеющейся колекції (из незмінюємоого набора):


import java.util.Set;
import java.util.HashSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> data = Set.<String>of("Java", "JavaScript", "C#");

        HashSet<String> langs = new HashSet<String>(data);

        System.out.println(langs); // [C#, Java, JavaScript]
    }
}

Клас HashSet в основном реалізує те методи, которые оголошенийы в родительских класах и застосовуємоых інтерфейсах. Приведу основные методи HashSet:

  • boolean add(E e): Додає указанный елемент в цей набор, якщо он ще отсутствует

  • void clear(): Видаляє всі елементи из цього набора

  • Object clone(): Повертає поверхностную копию данного экземпляра HashSet: сами елементы не клонируются

  • boolean contains(Object o): Повертає true, якщо цей набор містить указанный елемент

  • boolean isEmpty(): Повертає true, якщо цей набор не містить елементов

  • Iterator<E> iterator(): Повертає ітератор по елементам цього набора

  • static <T> HashSet<T> newHashSet(int numElements): Створює новый порожній HashSet, підходящий для ожидаемого количества елементов

  • boolean remove(Object o): Видаляє указанный елемент из цього набора, якщо он присутствует

  • int size(): Повертає кількість елементів в цем наборе

  • Object[] toArray(): Повертає масив, содержащий всі елементи цей колекції

За допомогою методу add() можна добавить новый об'єкт в хеш-таблицю. Если такого елемента ще нет, то об'єкт додаєся, а метод повертає true. Если елемент вже есть, ничего не происходит, а метод повертає false:


import java.util.HashSet;
 
public class Program{
      
    public static void main(String[] args) {

        HashSet<String> langs = new HashSet<String>();
        // додамо ряд об'єктов
        langs.add("Java");
        langs.add("C++");
        langs.add("C#");
        // пытаемся добавить елемент, який вже есть в колекції
        boolean isAdded = langs.add("Java");
        System.out.println(isAdded);    // false
        
        System.out.printf("langs size: %d \n", langs.size());    // langs size: 3
          
        System.out.println(langs); // [C#, Java, C++]
    }
}

Метод remove() видаляє об'єкт из хеш-таблиці. ПРи успешном удалении метод повертає true. Если об'єкт отсутствует, ничего не происходит, а метод повертає false. Для полного же удаления всех елементов застосовується метод clear(). Пример удаления:


import java.util.Set;
import java.util.HashSet;
 
public class Program{
      
    public static void main(String[] args) {
          
        Set<String> data = Set.<String>of("Java", "JavaScript", "C#", "Python");
        HashSet<String> langs = new HashSet<String>(data);
        // видаляємо язык JavaScript
        boolean isRemoved = langs.remove("JavaScript");
        System.out.println("JavaScript removed: " + isRemoved);    // true

        // видаляємо язык OCalm
        isRemoved = langs.remove("OCalm");
        System.out.println("OCalm removed: " + isRemoved);    // false
          
        System.out.println(langs); // [C#, Java, Python]

        // видаляємо все
        langs.clear();
        System.out.println(langs); // []
    }
}
