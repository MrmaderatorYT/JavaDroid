Методи и класс Method

Останнє оновлення: 29.09.2025

Класс java.lang.reflect.Method надає информацию об одном методе класса або інтерфейса и доступ к нему. Для получения информации о методах и управления ими класс Method определеяет ряд методів. Отмечу основные из них:

  • String getName(): повертає имя метода

  • Class[] getExceptionTypes(): повертає генерируемые методом типы исключений в виде массива об'єктов Class

  • int getModifiers(): повертає целое число с различными включенными и выключенными битами, которые описують используемые модификаторы.

  • int getParameterCount(): повертає количество параметров

  • Class[] getParameterTypes(): повертає типы параметров метода

  • Class getReturnType(): повертає тип возвращаемого значения метода

  • Object invoke(Object obj, Object... args): вызывает метод, представленный цим об'єктом метода, для указанного об'єкта с указанными параметрами.

Получение методів

Для получения методів типу застосовується ряд методів класса Class:

  • Method[] getMethods(): повертає массив об'єктов Method для открытых методів (с модификатором public) данного класса або его суперклассов

  • Method[] getDeclaredКласс java.lang.reflect.Methods(): повертає массив об'єктов Method для всех методів данного класса. Метод повертають массив нулевой длины, якщо таких методів нет або якщо об'єкт представляє собой примитивный тип або массив.

  • Method getMethod(String name, Class... parameterTypes): повертає об'єкт Method для публичного метода, имя которого передаєся в качестве параметра. Второй параметр указывает на типы параметров метода. Поиск також идет среди методів суперклассов. Если подобное публичный метод отсутствует, то метод генерує исключение NoSuchMethodException

  • Method getDeclaredMethod(String name, Class... parameterTypes): повертає об'єкт Method для метода, имя которого передаєся в качестве параметра. Второй параметр указывает на типы параметров метода. Если подобный метод отсутствует, то метод генерує исключение NoSuchMethodException

Наприклад, получим все методы некоторого класса:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Operation.class;
        Method[] methods = cl.getDeclaredMethods();
        for(Method m : methods){
            System.out.println(m);
        }
    }
}

class Operation{

    static int sum(int val1, int val2){
        return val1 + val2;
    }

    static double sum(double val1, double val2){
        return val1 + val2;
    }

    static int subtract(int val1, int val2){
        return val1 - val2;
    }
}

В данном случае получаем методы класса Operation. И консоль повинна нам вывести следующие строки:


static int Operation.sum(int,int)
static double Operation.sum(double, double)
static int Operation.subtract(int,int)

Исследование метода

Исследуем метод sum из выше определенного класса Operation:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Operation.class;
        try{
            // получаем метод sum с двумя параметрами типу int
            Method m = cl.getDeclaredMethod("sum", int.class, int.class);
            // получаем имя метода
            String methodName = m.getName();
            // получаем модификаторы
            String modifiers = Modifier.toString(m.getModifiers());

            // получаем количество параметров
            int paramsCount = m.getParameterCount();
            // получаем возвращаемый тип
            Class retType = m.getReturnType();
            // получаем типы параметров
            Class[] paramTypes = m.getParameterTypes();

            // выводим информацию о методе
            System.out.print(modifiers + " " + retType.getName() + " " + methodName + "(");
            for (int j = 0; j < paramsCount; j++)
            {
                if (j > 0) System.out.print(", ");
                System.out.print(paramTypes[j].getName());
            }
            System.out.println(");");
        }
        catch(Exception ex){

            System.out.println(ex);
        }
    }
}

class Operation{

    static int sum(int val1, int val2){
        return val1 + val2;
    }

    static double sum(double val1, double val2){
        return val1 + val2;
    }

    static int subtract(int val1, int val2){
        return val1 - val2;
    }
}

В данном случае получаем метод sum, який приймає два параметра типу int:

Method m = cl.getDeclaredMethod("sum", int.class, int.class);

Далїї последовательно получаем имя, возвращаемый тип, модификаторы и типы параметров метода и выводим их на консоль. В итоге консоль повинна нам вывести следующую строку:

static int sum(int, int);

Вызов метода

Для вызова метода в классе Method определен метод invoke():


public Object invoke(Object obj, Object... args)
              throws IllegalAccessException, InvocationTargetException

Первый параметр (obj) представляє об'єкт, для которого буде вызываться метод. Если вызываемый метод статический, то параметр obj игнорируется.

Второй параметр (args) представляє аргументы, передаваемые вызываемому методу. Если вызываемый метод не приймає нияких параметров, то параметр args опускается.

Підзвращаемое значение - це результат вызываемого метода.

Вызов статического метода

Вызовем статический метод sum класса Operation:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Class cl = Operation.class;
        try{
            // получаем метод sum с двумя параметрами типу int
            Method m = cl.getDeclaredMethod("sum", int.class, int.class);
            int val1 = 23;
            int val2 = 4;
            // вызываем метод и получаем результат
            int result = (int) m.invoke(null, val1, val2);

            System.out.printf("sum(%d, %d) = %d\n", val1, val2, result);        // sum(23, 4) = 27
        }       
        catch(Exception ex){

            System.out.println(ex);
        }
    }
}

class Operation{

    static int sum(int val1, int val2){
        return val1 + val2;
    }

    static int subtract(int val1, int val2){
        return val1 - val2;
    }
}

Поскольку вызываемый метод sum - статический, то в качестве первого параметра передаєся null:

int result = (int) m.invoke(null, val1, val2);

Результаи преобразуется к типу int, оскільки мы значем, что метод повинен возвратить значение цього типу. Консольный вывод программы:

sum(23, 4) = 27

Вызов нестатического метода

Рассмотрим вызов нестатического метода:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        // об'єкт, для которого буде вызываться метод
        Person tom = new Person("Tom");
        Class cl = tom.getClass();
        try{
            // получаем метод setName с параметром типу String
            Method m = cl.getDeclaredMethod("setName", String.class);
            // новое имя
            String name = "Eugene";
            // вызываем метод
            m.invoke(tom, name);

            tom.print();        // Person Eugene
        }       
        catch(Exception ex){

            System.out.println(ex);
        }
    }
}
class Person {
     
    private String name;

    void setName(String name){ this.name = name; }

    Person(String name){  this.name = name; }
    
    void print(){
        System.out.println("Person " + name);
    }
}

В данном случае для об'єкта Person вызываем метод setName(), який змінює значение поля name.

Приватные методы

Как и в случае с полями и конструкторами, механизм рефлексии Java запрещает обращение к приватным методам (определенным с модификатором private). В цем случае для доступа к методу необхідно сделать его доступным за допомогою метода setAccessible(), передав в него true:


import java.lang.reflect.*;

public class Program{
      
    public static void main(String[] args) {
        
        Person tom = new Person("Tom");
        Class cl = tom.getClass();

        try{
            Method m = cl.getDeclaredMethod("setName", String.class);
            m.setAccessible(true);      // открываем доступ к методу

            String name = "Eugene";

            m.invoke(tom, name);

            tom.print();        // Person Eugene
        }       
        catch(Exception ex){

            System.out.println(ex);
        }
    }
}
class Person {
     
    private String name;
    // приватный метод
    private void setName(String name){ this.name = name; }

    Person(String name){  this.name = name; }
    
    void print(){
        System.out.println("Person " + name);
    }
}
