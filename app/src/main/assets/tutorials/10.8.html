Сортування

Останнє оновлення: 30.04.2018

Колекції, на основе которых нередко создаются потіки, вже имеют специальные методи для сортировки содержимого. Однак клас Stream також включает возможнасть сортировки. Такую сортировку мы можемо задействовать, когда у нас идет набор промежуточных операцій с потіком, которые создают новые наборы данных, и нам треба эти наборы отсортировать.

Для простой сортировки за зростанням застосовується метод sorted():


import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class Program {
 
    public static void main(String[] args) {
         
        List<String> phones = new ArrayList<String>();
		Collections.addAll(phones, "iPhone X", "Nokia 9", "Huawei Nexus 6P",
                "Samsung Galaxy S8", "LG G6", "Xiaomi MI6",
                "ASUS Zenfone 3", "Sony Xperia Z5", "Meizu Pro 6",
                "Pixel 2");
         
        phones.stream()
                .filter(p->p.length()<12)
                .sorted() // сортування за зростанням
                .forEach(s->System.out.println(s));
    } 
}

Консольный вивід після сортировки об'єктов:

LG G6
Meizu Pro 6
Nokia 9
Pixel 2
Xiaomi MI6
iPhone X

Однак данный метод не всегда підходит. Вже по консольному вивіду мы видим, что метод сортує об'єкты за зростанням, но при цем заглавные и строчные буквы рассматриваются отдельно.

Крім того, данный метод підходит лише для сортировки тех об'єктов, которые реалізують інтерфейс Comparable.

Если же у нас класы об'єктов не реалізують цей інтерфейс або мы хочемо створити якую-то свою логику сортировки, то мы можемо використовувати другую версию методу sorted(), которая в качестве параметра приймає компаратор.

Наприклад, пусть у нас есть следующий клас Phone:


class Phone{
    
    private String name;
    private String company;
    private int price;
    
    public Phone(String name, String comp, int price){
        this.name=name;
        this.company=comp;
        this.price = price;
    }
    
    public String getName() { return name; }
    public int getPrice() { return price; }
    public String getCompany() { return company; }
}

Отсортуємо потік обектов Phone:


import java.util.Comparator;
import java.util.stream.Stream;

public class Program {

    public static void main(String[] args) {

        Stream<Phone> phoneStream = Stream.of(new Phone("iPhone X", "Apple", 600), 
            new Phone("Pixel 2", "Google", 500),
            new Phone("iPhone 8", "Apple",450),
            new Phone("Nokia 9", "HMD Global",150),
            new Phone("Galaxy S9", "Samsung", 300));
        
        phoneStream.sorted(new PhoneComparator())
                .forEach(p->System.out.printf("%s (%s) - %d \n", 
                        p.getName(), p.getCompany(), p.getPrice()));
        
    } 
}
class PhoneComparator implements Comparator<Phone>{
 
    public int compare(Phone a, Phone b){
     
        return a.getName().toUpperCase().compareTo(b.getName().toUpperCase());
    }
}

Здесь визначений клас компаратора PhoneComparator, який сортує об'єкты по полю name. В итоге мы получим следующий вивід:

Galaxy S9 (Samsung) - 300 
iPhone 8 (Apple) - 450 
iPhone X (Apple) - 600
Nokia 9 (HMD Global) - 150 
Pixel 2 (Google) - 500
