Перша програма Java

Останнє оновлення: 20.03.2024

Отже, після встановлення JDK створимо першу програму на мові Java. Що потрібно для створення програми на Java? Насамперед нам треба написати код програми, і для цього потрібен текстовий редактор. Можна використовувати будь-який текстовий редактор, наприклад Notepad++.

І щоб перетворити код програми на додаток, що виконується, необхідний компілятор. Компілятор встановлюється разом із JDK.

Итак, створимо на жестком диске якой-нибудь каталог, в котором будуть располагаться файли с исходным кодом на мові Java. Допустим, це буде каталог C:/Java. Затем створимо в цем каталогу текстовый файл, який переименуем в Program.java. Откроем цей файл в любом текстовом редакторе и наберем в нем следующую програму:


public class Program{ 
     
    public static void main (String args[]){
         
        System.out.println("Hello !");
    }
}

Java є об'єктно-орієнтованою мовою, тому вся програма представляється як набір взаємодіючих класів. У разі визначено один клас Program.

При визначенийии класа вначале идет модификатор доступа public, який вказує, что данный клас буде доступен всем, то есть мы сможемо его запустить из командного рядка. Далїї идет ключевое слово class, а затем название класа. То есть клас називається Program. Після названия в фигурных скобках расположено вміст класа.

Клас может містятьь различные змінні и методи. У даному випадку у нас оголошений один метод main. Це главный метод в любой програме на Java, он є входной точкой програми и с него начинается все управление. Он обязательно повинен присутствовать в програме.

Метод main також имїїт модификатор public. Слово static вказує, что метод main - статический, а слово void - что он не повертає ниякого значення. Позже ми детально розберемо, що все це означає.

Далїї в скобках у нас идут параметри методу - String args[] - це масив args, який хранит значення типу String, то є рядки. У даному випадку нам поки не потрібні, але в реальній програмі це ті рядкові параметри, які передаються при запуску програми з командного рядка.

Після списку параметрів у фігурних дужках йде тіло методу - це власне ті інструкції, які і буде виконувати метод. У даному випадку фактически визначенийи визначенийа лише одна инструкция - вивід на консоль некоторой рядки. Для вивіда на консоль використовується встроенный метод System.out.println(). В цей метод передаєся вивідимая рядок. Каждая инструкция завершуєся точкой с комою.

Тепер скомпілюємо написану програму. Відкриємо командний рядок (у Windows) або термінал в Linux/MacOS і введемо там відповідні команди. Першим ділом перейдемо в каталог, де лежить наш файл з програмою за допомогою команди cd:

cd C:\Java

У цьому випадку файл знаходиться в каталозі C: ​​Java.

Потім компілюємо програму за допомогою команди

javac Program.java

Компілятор мови Java представляє утиліту javac. Через пробел название нашего файлу, який містить клас програми.

Після цього програма компилируется в байт-код, и в каталогу C:\Java можна буде найти новый файл Program.class. Це и буде файл с байт-кодом програми. Теперь нам треба его запустить за допомогою утилиты java:

java Program

Тут вже розширення файлу не треба використовувати.

Для ОС Windows весь процес виглядатиме так:

У прикладі вище весь процес розбивався на два етапи:

  1. Компилятор javac створює из програми на мові Java файл с расширением *.class, який містить байт-код для виртуальной машины Java або JVM

  2. Утилита java запускує скомпилированный байт-код из файлу с расширением *.class. То есть происходит виконання програми

Але в останніх версіях Java також можна було відразу виконати код програми мовою Java:

C:\Java>java Program.java
Hello !

C:\Java>

В цем случае утилите java передаєся файл с исходным кодом. При цем файлу с расширением *.class не створюєся.
