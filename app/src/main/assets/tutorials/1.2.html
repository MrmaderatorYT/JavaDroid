Перша програма Java

Останнє оновлення: 20.03.2024

Отже, після встановлення JDK створимо першу програму на мові Java. Що потрібно для створення програми на Java? Насамперед нам треба написати код програми, і для цього потрібен текстовий редактор. Можна використовувати будь-який текстовий редактор, наприклад Notepad++.

І щоб перетворити код програми на додаток, що виконується, необхідний компілятор. Компілятор встановлюється разом із JDK.

Итак, создадим на жестком диске якой-нибудь каталог, в котором будуть располагаться файлы с исходным кодом на мові Java. Допустим, це буде каталог C:/Java. Затем создадим в цем каталоге текстовый файл, який переименуем в Program.java. Откроем цей файл в любом текстовом редакторе и наберем в нем следующую программу:


public class Program{ 
     
    public static void main (String args[]){
         
        System.out.println("Hello !");
    }
}

Java є об'єктно-орієнтованою мовою, тому вся програма представляється як набору взаємодіючих класів. У разі визначено один клас Program.

При определении класса вначале идет модификатор доступа public, який указывает, что данный класс буде доступен всем, то есть мы сможемо его запустить из командного рядка. Далїї идет ключевое слово class, а затем название класса. То есть класс називається Program. После названия в фигурных скобках расположено содержимое класса.

Класс может містятьь различные переменные и методы. В данном случае у нас оголошений один метод main. Це главный метод в любой программе на Java, он є входной точкой программы и с него начинается все управление. Он обязательно повинен присутствовать в программе.

Метод main також имїїт модификатор public. Слово static указывает, что метод main - статический, а слово void - что он не повертає ниякого значения. Позже мы подробнїї разберем, что все це значит.

Далїї в скобках у нас идут параметры метода - String args[] - це массив args, який хранит значения типу String, то есть строки. В данном случае ни нам поки не нужны, но в реальной программе це те строковые параметры, которые передаются при запуске программы из командного рядка.

После списка параметров в фигурных скобках идет тело метода - це собственно те инструкции, которые и буде выполнять метод. В данном случае фактически определени определена лише одна инструкция - вывод на консоль некоторой строки. Для вывода на консоль використовується встроенный метод System.out.println(). В цей метод передаєся выводимая строка. Каждая инструкция завершуєся точкой с комою.

Теперь скомпилируем написанную программу. Откроем командную строку (в Windows) або терминал в Linux/MacOS и введем там соответствующие команды. Первым делом перейдем в каталог, где лежит наш файл с программой за допомогою команды cd:

cd C:\Java

У цьому випадку файл знаходиться в каталозі C: ​​Java.

Потім компілюємо програму за допомогою команди

javac Program.java

Компилятор мови Java представляє утилиту javac. Через пробел название нашего файла, який містить класс программы.

После цього программа компилируется в байт-код, и в каталоге C:\Java можна буде найти новый файл Program.class. Це и буде файл с байт-кодом программы. Теперь нам треба его запустить за допомогою утилиты java:

java Program

Тут вже розширення файлу не треба використовувати.

Для ОС Windows весь процес виглядатиме так:

У прикладі вище весь процес розбивався на два етапи:

  1. Компилятор javac створює из программы на мові Java файл с расширением *.class, який містить байт-код для виртуальной машины Java або JVM

  2. Утилита java запускає скомпилированный байт-код из файла с расширением *.class. То есть происходит выполнение программы

Але в останніх версіях Java також можна було відразу виконати код програми мовою Java:

C:\Java>java Program.java
Hello !

C:\Java>

В цем случае утилите java передаєся файл с исходным кодом. При цем файла с расширением *.class не створюєся.
