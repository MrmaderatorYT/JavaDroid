Форматируемый ввод и вывод. PrintStream и PrintWriter

Останнє оновлення: 26.04.2018

Класс PrintStream

Класс PrintStream - це именно тот класс, який використовується для вывода на консоль. Когда мы выводим на консоль деяку информацию за допомогою вызова System.out.println(), то тем самым мы задействует PrintStream, оскільки переменная out в классе System як раз и представляє об'єкт класса PrintStream, а метод println() - це метод класса PrintStream.

Но PrintStream полезен не лише для вывода на консоль. Мы можемо використовувати данный класс для записи информации в поток вывода. Для цього PrintStream определяет ряд конструкторов:


PrintStream(OutputStream outputStream)
PrintStream(OutputStream outputStream, boolean autoFlushingOn)
PrintStream(OutputStream outputStream, boolean autoFlushingOn, String charSet) throws UnsupportedEncodingException
PrintStream(File outputFile) throws FileNotFoundException
PrintStream(File outputFile, String charSet) throws FileNotFoundException, UnsupportedEncodingException
PrintStream(String outputFileName) throws FileNotFoundException
PrintStream(String outputFileName, String charSet) throws FileNotFoundException, UnsupportedEncodingException

Параметр outputStream - це об'єкт OutputStream, в який производится запись. Параметр autoFlushingOn при значении true дозволяє автоматически записывать дані в поток вывода. По умолчанию цей параметр равен false. Параметр charSet дозволяє указать кодировку символов.

В качестве источника для записи данных вместо OutputStream можна використовувати об'єкт File або строковый путь, по которому буде создаваться файл.

Для вывода информации в выходной поток PrintStream использует следующие методы:

  • println(): вывод строковой информации с переводом строки

  • print(): вывод строковой информации без перевода строки

  • printf(): форматированный вывод

Наприклад, запишем информацию в файл:


import java.io.*;

public class Program {

    public static void main(String[] args) {
        
        String text = "Привет мир!"; // строка для записи
        try(FileOutputStream fos=new FileOutputStream("C://SomeDir//notes3.txt");
                PrintStream printStream = new PrintStream(fos))
        {
            printStream.println(text);
            System.out.println("Запись в файл произведена");
        }
        catch(IOException ex){
            
            System.out.println(ex.getMessage());
        }  
    } 
}

В данном случае застосовується форма конструктора PrintStream, которая в качестве параметра приймає поток вывода: PrintStream (OutputStream out). Крім того, мы могли бы використовувати ряд других форм конструктора, наприклад, указывая названия файла для записи: PrintStream (string filename)

В качестве потока вывода використовується об'єкт FileOutputStream. С помощью метода println() производится запись информации в выходной поток - то есть в об'єкт FileOutputStream. (В случае с выводом на консоль за допомогою System.out.println() в качестве потока вывода выступает консоль)

Крім того, як и любой поток вывода и наследник класса OutputStream он имїїт метод write:


import java.io.*;

public class Program {
 
    public static void main(String[] args) {
         
        try(PrintStream printStream = new PrintStream("notes3.txt"))
        {
            printStream.print("Hello World!");
            printStream.println("Welcome to Java!");
			
            printStream.printf("Name: %s Age: %d \n", "Tom", 34);
			
			String message = "PrintStream";
            byte[] message_toBytes = message.getBytes();
            printStream.write(message_toBytes);
			
            System.out.println("The file has been written");
        }
        catch(IOException ex){
             
            System.out.println(ex.getMessage());
        }  
    } 
}

После виконання цей программы получится файл со наступним вмістм:

Hello World!Welcome to Java!
Name: Tom Age: 34 
PrintStream

PrintWriter

На PrintStream похож другой класс PrintWriter. Его можна використовувати як для вывода информации на консоль, так и в файл або любой другой поток вывода. Данный класс имїїт ряд конструкторов:

  • PrintWriter(File file): автоматически додає информацию в указанный файл

  • PrintWriter(File file, String csn): автоматически додає информацию в указанный файл с учетом кодировки csn

  • PrintWriter(OutputStream out): для вывода информации використовується существующий об'єкт OutputStream, автоматически сбрасывая в него дані

  • PrintWriter(OutputStream out, boolean autoFlush): для вывода информации використовується существующий об'єкт OutputStream, второй параметр указывает, треба ли автоматически добавлять в OutputStream дані

  • PrintWriter(String fileName): автоматически додає информацию в файл по указанному имени

  • PrintWriter(String fileName, String csn): автоматически додає информацию в файл по указанному имени, используя кодировку csn

  • PrintWriter(Writer out): для вывода информации використовується существующий об'єкт Writer, в який автоматически идет запись данных

  • PrintWriter(Writer out, boolean autoFlush): для вывода информации використовується существующий об'єкт Writer, второй параметр указывает, треба ли автоматически добавлять в Writer дані

PrintWriter реализует інтерфейсы Appendable, Closable и Flushable, и поцему после использования представляемый им поток треба закрывать.

Для записи данных в поток он також використовується методы printf() и println().

Наприклад, применим данный класс для вывода на консоль:


try(PrintWriter pw = new PrintWriter(System.out))
{
    pw.println("Hello world!");
}

В качестве потока вывода здесь застосовується System.out, а на консоль буде выведена строка "Hello world!"
