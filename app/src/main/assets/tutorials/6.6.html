Форматируемый ввод и вивід. PrintStream и PrintWriter

Останнє оновлення: 26.04.2018

Клас PrintStream

Клас PrintStream - це именно тот клас, який використовується для вивіда на консоль. Когда мы вивідим на консоль деяку информацию за допомогою вызова System.out.println(), то тем самым мы задействует PrintStream, оскільки змінна out в класе System як раз и представляє об'єкт класа PrintStream, а метод println() - це метод класа PrintStream.

Но PrintStream полезен не лише для вивіда на консоль. Мы можемо використовувати данный клас для записи информации в потік вивіда. Для цього PrintStream визначає ряд конструкторов:


PrintStream(OutputStream outputStream)
PrintStream(OutputStream outputStream, boolean autoFlushingOn)
PrintStream(OutputStream outputStream, boolean autoFlushingOn, String charSet) throws UnsupportedEncodingException
PrintStream(File outputFile) throws FileNotFoundException
PrintStream(File outputFile, String charSet) throws FileNotFoundException, UnsupportedEncodingException
PrintStream(String outputFileName) throws FileNotFoundException
PrintStream(String outputFileName, String charSet) throws FileNotFoundException, UnsupportedEncodingException

Параметр outputStream - це об'єкт OutputStream, в який производится запис. Параметр autoFlushingOn при значении true дозволяє автоматически записывать дані в потік вивіда. По умолчанию цей параметр равен false. Параметр charSet дозволяє указать кодировку символів.

В качестве источника для записи данных вместо OutputStream можна використовувати об'єкт File або рядківовый путь, по которому буде создаваться файл.

Для вивіда информации в выходной потік PrintStream використовує следующие методи:

  • println(): вивід рядківовой информации с переводом рядки

  • print(): вивід рядківовой информации без перевода рядки

  • printf(): форматированный вивід

Наприклад, запишем информацию в файл:


import java.io.*;

public class Program {

    public static void main(String[] args) {
        
        String text = "Привет мир!"; // рядок для записи
        try(FileOutputStream fos=new FileOutputStream("C://SomeDir//notes3.txt");
                PrintStream printStream = new PrintStream(fos))
        {
            printStream.println(text);
            System.out.println("Запис в файл произведена");
        }
        catch(IOException ex){
            
            System.out.println(ex.getMessage());
        }  
    } 
}

У даному випадку застосовується форма конструктора PrintStream, которая в качестве параметра приймає потік вивіда: PrintStream (OutputStream out). Крім того, мы могли бы використовувати ряд других форм конструктора, наприклад, указывая названия файлу для записи: PrintStream (string filename)

В качестве потіка вивіда використовується об'єкт FileOutputStream. За допомогою методу println() производится запис информации в выходной потік - то есть в об'єкт FileOutputStream. (В случае с вивідом на консоль за допомогою System.out.println() в качестве потіка вивіда выступает консоль)

Крім того, як и любой потік вивіда и наследник класа OutputStream он имїїт метод write:


import java.io.*;

public class Program {
 
    public static void main(String[] args) {
         
        try(PrintStream printStream = new PrintStream("notes3.txt"))
        {
            printStream.print("Hello World!");
            printStream.println("Welcome to Java!");
			
            printStream.printf("Name: %s Age: %d \n", "Tom", 34);
			
			String message = "PrintStream";
            byte[] message_toBytes = message.getBytes();
            printStream.write(message_toBytes);
			
            System.out.println("The file has been written");
        }
        catch(IOException ex){
             
            System.out.println(ex.getMessage());
        }  
    } 
}

Після виконання цей програми получится файл со наступним вмістм:

Hello World!Welcome to Java!
Name: Tom Age: 34 
PrintStream

PrintWriter

На PrintStream похож другой клас PrintWriter. Его можна використовувати як для вивіда информации на консоль, так и в файл або любой другой потік вивіда. Данный клас имїїт ряд конструкторов:

  • PrintWriter(File file): автоматически додає информацию в указанный файл

  • PrintWriter(File file, String csn): автоматически додає информацию в указанный файл с учетом кодировки csn

  • PrintWriter(OutputStream out): для вивіда информации використовується существующий об'єкт OutputStream, автоматически сбрасывая в него дані

  • PrintWriter(OutputStream out, boolean autoFlush): для вивіда информации використовується существующий об'єкт OutputStream, второй параметр вказує, треба ли автоматически добавлять в OutputStream дані

  • PrintWriter(String fileName): автоматически додає информацию в файл по указанному імені

  • PrintWriter(String fileName, String csn): автоматически додає информацию в файл по указанному імені, використовуючи кодировку csn

  • PrintWriter(Writer out): для вивіда информации використовується существующий об'єкт Writer, в який автоматически идет запис данных

  • PrintWriter(Writer out, boolean autoFlush): для вивіда информации використовується существующий об'єкт Writer, второй параметр вказує, треба ли автоматически добавлять в Writer дані

PrintWriter реалізує інтерфейсы Appendable, Closable и Flushable, и поцему після использования представляемый им потік треба закрывать.

Для записи данных в потік он також використовується методи printf() и println().

Наприклад, застосуємо данный клас для вивіда на консоль:


try(PrintWriter pw = new PrintWriter(System.out))
{
    pw.println("Hello world!");
}

В качестве потіка вивіда здесь застосовується System.out, а на консоль буде выведена рядок "Hello world!"
