Пов'язаний список LinkedList

Останнє оновлення: 11.10.2025

Обобщенный клас LinkedList<E> представляє структуру данных в виде связанного списку. Он найдеся от класа AbstractSequentialList и реалізує інтерфейсы List, Dequeue и Queue. То есть он соединяет функциональность работы со списком и фукциональность очереди.

Клас LinkedList имїїт следующие конструктори:

  • LinkedList(): створює порожній список

  • LinkedList(Collection<? extends E> col): створює список, в який додає всі елементи колекції col

LinkedList містить все те методи, которые визначенийы в інтерфейсах List, Queue, Deque. Некоторые из них:

  • addFirst() / offerFirst(): додає елемент в начало списку

  • addLast() / offerLast(): додає елемент в конец списку

  • removeFirst() / pollFirst(): видаляє первый елемент из начала списку

  • removeLast() / pollLast(): видаляє післядний елемент из конца списку

  • getFirst() / peekFirst(): отримує первый елемент

  • getLast() / peekLast(): отримує післядний елемент

Рассмотрим застосування некоторых методів связанного списку. Додавання в список:


import java.util.LinkedList;
 
public class Program{
      
    public static void main(String[] args) {
          
        var people = new LinkedList<String>();
          
        // додамо в список ряд елементов
        people.add("Tom");              // Tom
        people.add("Bob");              // Tom - Bob

        // додаємо на післяднїї место
        people.addLast("Sam");          // Tom - Bob - Sam
        // додаємо на первое место
        people.addFirst("Alice");       // Alice - Tom - Bob - Sam
        // додаємо елемент по индексу 1
        people.add(1, "Kate");                 // Alice - Kate - Tom - Bob - Sam

        System.out.println(people);         // [Alice, Kate, Tom, Bob, Sam]

        // отримуємо размер списку
        System.out.printf("List has %d elements \n", people.size());
    }
}

Отримання, зміна, перебор и проверка на наличие в списке:


import java.util.LinkedList;
import java.util.Collections;
 
public class Program{
      
    public static void main(String[] args) {
          
        var people = new LinkedList<String>();
        Collections.addAll(people, "Alice", "Kate", "Tom", "Bob", "Sam");

        // отримуємо третий елемент
        String second = people.get(2);
        System.out.println(second); // Tom

        // отримуємо первый елемент
        System.out.println(people.getFirst()); // Alice
        // отримуємо післядний елемент
        System.out.println(people.getLast()); // Sam

        // Змінюємо третий елемент
        people.set(2, "Tomas");

        System.out.println(people); // [Alice, Kate, Tomas, Bob, Sam]

        // перебор 
        for(var p : people){
          
            System.out.print(p + ", ");
        }
        System.out.println();

        // проверка на наличие елемента в списке
        if(people.contains("Tom")){
          
            System.out.println("List contains `Tom`");
        }
        else{
            System.out.println("List doesn`t contain `Tom`");
        }
    }
}

Видалення из списку:


import java.util.LinkedList;
import java.util.Collections;
 
public class Program{
      
    public static void main(String[] args) {
          
        var people = new LinkedList<String>();
        Collections.addAll(people, "Alice", "Kate", "Tom", "Bob", "Sam");

        System.out.println(people); // [Alice, Kate, Tomas, Bob, Sam]
        // видаляємо конкретный елемент
        people.remove("Bob");    // [Alice, Kate, Tom, Sam]
        people.remove("Boby");    // якщо елемента нет, ничего не происходит
         // видалення первого елемента
        people.removeFirst();    // [Kate, Tom, Sam]
         // видалення післяднего елемента
        people.removeLast();    // [Kate, Tom]
        System.out.println(people); // [Kate, Tom]

        // видалення всех елементов
        people.clear();
        System.out.println(people); // []
    }
}
