Зіставлення шаблону. шаблон запису

Останнє оновлення: 27.09.2025

Начиная с версии Java 21 мы можемо використовувати так называемый шаблон запису - паттерн, який дозволяє проверить на соответствие типу класу record и використовувати его поля. Для применения паттерна record можна застосовувати як оператор instanceof, так и конструкции/выражения switch.

примірник

Пример record patter с застосуванням оператора instanceof:


public class Program{
      
    public static void main(String[] args) {
             
        Person tom = new Person("Tom", 22);

        if(tom instanceof Person(var pName, var pAge)){
            System.out.printf("Person. Name: %s;  Age: %d\n", pName, pAge);
        }
    }
}
record Person(String name, int age){}

У даному випадку за допомогою выражения tom instanceof Person(var pName, var pAge) мы не лише проверяем, что змінна tom представляє тип Person, но и оголошуємо змінні name и age, которые отримують значення одноименных полей класа Person. Затем всередині блока if мы можемо використовувати эти змінні.

Крім того, ми можемоо явно вказати тип змінних


if(tom instanceof Person(String pName, int pAge)){
    System.out.printf("Person. Name: %s;  Age: %d\n", pName, pAge);
}

Якщо нам не потрібне поле, ми можемоо опустити його та використовувати тире замість змінної. Наприклад, нам потрібно лише вікове поле:


if(tom instanceof Person(_, int pAge)){
    System.out.printf("Person Age: %d\n", pAge);
}

перемикач

Конструкции/выражения switch також дозволяють використовувати данный паттерн:


Person tom = new Person("Tom", 22);

switch(tom){

    case Person(var pName, var pAge) ->
        System.out.printf("Person. Name: %s;  Age: %d\n", pName, pAge);

    case null -> System.out.printf("Undefined");
}

Константи та захисні умови охорони

Варто враховувати, що ми не можемоо використовувати константи в шаблоні запису. Наприклад, нам потрібно окремо відстежити ситуацію, коли ім’я користувача «Том». Але ми не можемоо писати так:


if(tom instanceof Person("Tom", _)){  // Ошибка: "Tom" - константа
    System.out.printf("Person Tom");
}

Однак ми можемоо використовувати захисні умови, коли змінні порівнюються з певними константами:


public class Program{
      
    public static void main(String[] args) {
             
        Person tom = new Person("Tom", 22);
        Person bob = new Person("Bob", 46);

        printPerson(tom);       // Hello Tom
        printPerson(bob);       // Hello
    }

    static void printPerson(Person person){

        if(person instanceof Person(var pName, _) && pName == "Tom"){
             System.out.println("Hello Tom");
        }
        else if(person instanceof Person(_, _) ){
             System.out.println("Hello");
        }
    }
}

Таким чином, якщо ім'я "Tom", то срабатывает блок if:


if(person instanceof Person(var pName, _) && pName == "Tom"){
    System.out.println("Hello Tom");
}

Если же у пользователя другое ім'я, то срабатывает блок else if


else if(person instanceof Person(_, _) ){
    System.out.println("Hello");
}

При использовании switch для створенняя защитных условий застосовується оператор when, після которого идет условие:


public class Program{
      
    public static void main(String[] args) {
             
        Person tom = new Person("Tom", 22);
        Person bob = new Person("Bob", 46);

        printPerson(tom);       // Hello Tom
        printPerson(bob);       // Hello
    }

    static void printPerson(Person person){

        switch(person){

            case Person(var pName, _)  when pName == "Tom" -> // лише для тех Person, где name = "Tom"
                System.out.println("Hello Tom");

            case Person(_, _) ->                // для всех остальных Person
                System.out.println("Hello");

            case null -> {}   // якщо null, ничего не вивідим
        }
    }
}

Вкладені шаблони

Шаблони можуть бути вкладеними:


public class Program{
      
    public static void main(String[] args) {
             
        Person tom = new Person("Tom", 41);
        Company mycorp = new Company("MyCorp", tom);

        if(mycorp instanceof Company(_, Person(var name, _))){
            System.out.println("CEO: " + name);
        }
    }
}
record Person(String name, int age){}
record Company(String name, Person ceo){}

У даному випадку второе поле класа Company представляє тип Person. А за допомогою выражения mycorp instanceof Company(_, Person(var name, _)) отримуємо поле name у класа Person в одноименную змінну name.

Аналогичный пример с switch:


public class Program{
      
    public static void main(String[] args) {
             
        Person tom = new Person("Tom", 41);
        Company mycorp = new Company("MyCorp", tom);

        switch(mycorp){

            case Company(_, Person(var name, _)) ->
                System.out.println("CEO: " + name);

            case null -> {}
            default -> {}
        }
    }
}
record Person(String name, int age){}
record Company(String name, Person ceo){}

Зверніть увагу на останні два відповідні блоки:


case null -> {}
default -> {}

При визначенні дизайну слід враховувати, що варіанти в ньому повинні бути «вичерпними» (exhaustive), тобто охоплювати всі можливі варіанти. У нашому випадку наша компанія може бути нульовою:

Company mycorp = null;

Цей варіант відстежується блоком


case null -> {}

Однак у нас може бути варіант, що Person має значення null:


Company mycorp = new Company("MyCorp", null);

Цет вариант отслеживается блоком default, який отслеживает все остальные варианты:


default -> {}
