Інтерфейси SortedMap и NavigableMap. Класс TreeMap

Останнє оновлення: 13.10.2025

Интерфейс Map, який застосовується в мові Java для создания словарей, не предусматривает нияких возможнастей для сортировки. Но Java також надає ряд дополнительных інтерфейсов: SortedMap и NavigableMap

SortedMap

Интерфейс SortedMap расширяет Map и дозволяє обеспечить полное упорядочение своих ключей. По умолчанию элементы упорядочиваются в соответствии с естественным порядком ключей (по возрастанию) або за допомогою компаратора. Цет порядок, наприклад, отражается при переборе коллекции.

SortedMap додає ряд методів:

  • K firstKey(): повертає ключ первого элемента коллекции

  • K lastKey(): повертає ключ последнего элемента коллекции

  • SortedMap<K, V> headMap(K end): повертає об'єкт SortedMap, який містить все элементы оригинального SortedMap вплоть до элемента с ключом end

  • SortedMap<K, V> tailMap(K start): повертає об'єкт SortedMap, який містить все элементы оригинального SortedMap, начиная с элемента с ключом start

  • SortedMap<K, V> subMap(K start, K end): повертає об'єкт SortedMap, которые містить все элементы оригинального SortedMap вплоть от элемента с ключом start до элемента с ключом end

NavigableMap

Интерфейс NavigableMap расширяет інтерфейс SortedMap и забезпечує возможнасть получения элементов коллекции относительно других элементов (условно говоря, навигацию). Его основные методы:

  • Map.Entry<K, V> ceilingEntry(K key): повертає элемент с наименьшим ключом k, який больше або равен ключу key (k >=key). Если такого ключа нет, то повертаєся null.

  • Map.Entry<K, V> floorEntry(K key): повертає элемент с наибольшим ключом k, який меньше або равен ключу key (k <=key). Если такого ключа нет, то повертаєся null.

  • Map.Entry<K, V> higherEntry(K key): повертає элемент с наименьшим ключом k, який больше ключа key (k >key). Если такого ключа нет, то повертаєся null.

  • Map.Entry<K, V> lowerEntry(K key): повертає элемент с наибольшим ключом k, який меньше ключа key (k <key). Если такого ключа нет, то повертаєся null.

  • Map.Entry<K, V> firstEntry(): повертає первый элемент коллекции

  • Map.Entry<K, V> lastEntry(): повертає последний элемент коллекции

  • Map.Entry<K, V> pollFirstEntry(): повертає и одновременно видаляє первый элемент из коллекции

  • Map.Entry<K, V> pollLastEntry(): повертає и одновременно видаляє последний элемент из коллекции

  • K ceilingKey(K key): повертає наименьший ключ k, який больше або равен ключу key (k >=key). Если такого ключа нет, то повертаєся null.

  • K floorKey(K key): повертає наибольший ключ k, який меньше або равен ключу key (k <=key). Если такого ключа нет, то повертаєся null.

  • K lowerKey(K key): повертає наибольший ключ k, який меньше ключа key (k <key). Если такого ключа нет, то повертаєся null.

  • K higherKey(K key): повертає наименьший ключ k, який больше ключа key (k >key). Если такого ключа нет, то повертаєся null.

  • NavigableSet<K> descendingKeySet(): повертає об'єкт NavigableSet, який містить все ключи коллекции в обратном порядке

  • NavigableMap<K, V> descendingMap(): повертає коллекцию NavigableMap, которое містить все элементы в обратном порядке

  • NavigableSet<K> navigableKeySet(): повертає об'єкт NavigableSet, який містить все ключи коллекции

  • NavigableMap<K, V> headMap(K upperBound, boolean incl): повертає коллекцию NavigableMap, которое містить все элементы оригинальной коллекции NavigableMap вплоть от элемента с ключом upperBound. Параметр incl при значении true указывает, что элемент с ключом upperBound також включается в выходной набор.

  • NavigableMap<K, V> tailMap(K lowerBound, boolean incl): повертає коллекцию NavigableMap, которое містить все элементы оригинальной коллекции NavigableMap, начиная с элемента с ключом lowerBound. Параметр incl при значении true указывает, что элемент с ключом lowerBound також включается в выходной набор.

  • NavigableMap<K, V> subMap(K lowerBound, boolean lowIncl, K upperBound, boolean highIncl): повертає коллекцию NavigableMap, которое містить все элементы оригинальной коллекции NavigableMap от элемента с ключом lowerBound до элемента с ключом upperBound. Параметры lowIncl и highIncl при значении true включают в выходной набор элементы с ключами lowerBound и upperBound соответственно.

TreeMap

Встроенной реализацией выше упомянутых інтерфейсов є класс TreeMap<K, V>. Он представляє коллекцию в виде дерева, где все элементы отсортированы. Цет класс найдеся от класса AbstractMap и реализует інтерфейс NavigableMap, а следовательно, також и інтерфейсы SortedMap и Map. Тому для управления элементами в дереве можна використовувати методы інтерфейса Map. Но в отличие от коллекции HashMap в TreeMap все об'єкты автоматически сортируются по возрастанию их ключей.

Класс TreeMap имїїт следующие конструкторы:

  • TreeMap(): створює пустое коллекцию в виде дерева

  • TreeMap(Map<? extends K, ? extends V> map): створює дерево, в которое додає все элементы из коллекции map

  • TreeMap(SortedMap<K, ? extends V> smap): створює дерево, в которое додає все элементы из коллекции smap

  • TreeMap(Comparator<? super K> comparator): створює пустое дерево, где все добавляемые элементы впоследствии будуть отсортированы компаратором.

Элементы дерева повинні реализовать інтерфейс Comparable() для сортировки, або необхідно предоставлять компаратор через последнюю версию конструктора. При цем порядок, поддерживаемый в TreeMap, (независимо от того, предоставлен ли явный компаратор), повинен быть согласован с equals(), чтобы эта сортированная коллекция правильно реализовала інтерфейс Map. Примеры создания дерева:


import java.util.Map;
import java.util.TreeMap;
 
public class Program{
      
    public static void main(String[] args) {
          
        // пустое дерево
        var treeMap = new TreeMap<String,String>();
        System.out.println(treeMap);     // {}

        // дерево на основе другого словаря
        var dict = new TreeMap<String,String>(
                                Map.ofEntries( 
                                    Map.entry("+79876543210", "Tom"),
                                    Map.entry("+79666543210", "Bob"),
                                    Map.entry("+79176543210", "Sam"),
                                    Map.entry("+79276543210", "Alice")
                                ));
        System.out.println(dict);  // {blue=синий, green=зеленый, red=красный, yellow=желтый}
    }
}

Обратите внимание на вывод второго дерева-словаря - dict:

{blue=синий, green=зеленый, red=красный, yellow=желтый}

Вне зависимости в яком порядке об'єкты буви добавлены в дерево, они упорядочиваются по возрастанию. Так, в данном случае у нас переменная dict представляє условный словарь, а TreeMap дозволяє упорядочить дані словаря по возрастанию.

Добавление элементов:


import java.util.Map;
import java.util.TreeMap;
 
public class Program{
      
    public static void main(String[] args) {
          
        // пустое дерево
        var dict = new TreeMap<String,String>();
 
        // добавляем в него элементы
        dict.put("red", "красный");
        dict.put("green", "зеленый");
        dict.put("blue", "синий");
        System.out.println(dict);   // {blue=синий, green=зеленый, red=красный}
 
        // добавляем другой словарь с данными
        var map = Map.ofEntries( 
                Map.entry("yellow", "желтый"),
                Map.entry("black", "черный") 
        );
        dict.putAll(map);

        System.out.println(dict);   // {black=черный, blue=синий, green=зеленый, red=красный, yellow=желтый}
    }
}

Получение элементов:


import java.util.Map;
import java.util.Set;
import java.util.Collection;
import java.util.TreeMap;
 
public class Program{
      
    public static void main(String[] args) {
          
        // пустой словарь
        var dict = new TreeMap<String,String>(
            Map.ofEntries( 
                Map.entry("red", "красный"),
                Map.entry("green", "зеленый"),
                Map.entry("yellow", "желтый"),
                Map.entry("blue", "синий"),
                Map.entry("black", "черный") 
            )
        );

        // перебор элементов
       for(var word : dict.entrySet()){
         
           System.out.printf("%s: %s \n", word.getKey(), word.getValue());
       }

        // получим об'єкт по ключу "red"
        String red = dict.get("red");
        System.out.println("`red` переводится як `" + red + "`");  // `red` переводится як `красный`

        // получим весь набор ключей
        Set<String> keys = dict.keySet();
        System.out.println(keys);       // [black, blue, green, red, yellow]

        // отримати набор всех значений
        Collection<String> values = dict.values();
        System.out.println(values);     // [черный, синий, зеленый, красный, желтый]
            
        // получаем все об'єкты, которые стоят начиная с об'єкта с ключом "green"
        Map<String, String> afterGreen = dict.tailMap("green");
        System.out.println(afterGreen);     // {green=зеленый, red=красный, yellow=желтый}
            
        // получаем все об'єкты, которые стоят до об'єкта с ключом "green"
        Map<String, String> beforeGreen = dict.headMap("green");
        System.out.println(beforeGreen);    // {black=черный, blue=синий}
         
        // получим последний элемент дерева
        Map.Entry<String, String> lastItem = dict.lastEntry();
        System.out.println(lastItem);   // yellow=желтый
    }
}

Изменение и удаление элементов:


import java.util.Map;
import java.util.TreeMap;
 
public class Program{
      
    public static void main(String[] args) {
          
        // пустой словарь
        var dict = new TreeMap<String,String>(
            Map.ofEntries( 
                Map.entry("red", "красный"),
                Map.entry("green", "зеленый"),
                Map.entry("yellow", "желтый"),
                Map.entry("blue", "синий"),
                Map.entry("black", "черный") 
            )
        );

        // меняем по ключу "red" значение на "червонный"
        dict.replace("red", "червонный");
        System.out.println(dict);   // {black=черный, blue=синий, green=зеленый, red=червонный, yellow=желтый}
 
        // меняем по ключу "red" значение на "алый", лише якщо значение "красный"
        dict.replace("red", "алый", "красный");
 
        // удаляем yellow
        dict.remove("yellow");
        System.out.println(dict);   // {black=черный, blue=синий, green=зеленый, red=червонный}

        // получаем с удалением первый элемент
        Map.Entry<String, String> firstWord = dict.pollFirstEntry();
        System.out.println(firstWord);      // black=черный

        // получаем с удалением последний элемент
        Map.Entry<String, String> lastWord = dict.pollLastEntry();
        System.out.println(lastWord);       // red=червонный
 
        // удаляем все элементы
        dict.clear();
        System.out.println(dict);   // {}
    }
}
