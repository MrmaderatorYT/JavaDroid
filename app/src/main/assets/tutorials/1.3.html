Перша програма у NetBeans

Останнє оновлення: 28.09.2020

Ще одной популярной средой програмирования під Java є Netbeans. Проєкт NetBeans во многом піддерживается и спонсируется компанией Oracle, при цем его разработка ведется независимым сообществом разработчиков (Apache NetBeans Community). NetBeans є кроссплатформенным програмным обеспечением, поцему есть версии среды для различных платформ и ОС. Болїї підробную информацию можна найти на официальном сайте даной среды https://netbeans.apache.org/.

На данный момент післядней версией є 12-я версия. Итак, установим післяднюю версию NetBeans, загрузив со страницы https://netbeans.apache.org/download/nb121/nb121.html.

На странице загрузки мы можемо найти два варианта: скомпилированные бинарные файли в виде архива (пункт Binaries) и установщики для кожної конкретної операційної системи (пункт Installers). Завантажимо середовище у скомпільованому, обравши пункт Binaries.

Після вибору цієї опції нас переадресує на сторінку із прямим посиланням на дзеркало для завантаження:

Після завантаження розпакуємо архів у папку, де ми хочемо розмістити середовище Netbeans. Варто відзначити, что після распаковки архива мы можемо вже відразу запустить среду, не треба ничего устанавливать. В архиві вже готові для використання файли. Для запуску в розпакованому архиві у папці bin мы можемо найти собственно запускуємоые файли. В случае с ОС Windows це netbeans.exe и netbeans64.exe

Запустимоо netbeans.exe чи netbeans64.exe:

NetBeans досить інтелектуально зрозуміла, и с ней дуже легко працювати. Створимо новый проєкт. Для цього оберемо в меню пункт File->New Project.... Після цього перед нами откроется окно створенняя нового проєкта:

В окне створенняя нового проєкта оберемо в левой колонке первый пункт - Java with Ant, а в правой в качестве типу проєкта - Java Application. И нажмем кнопку Next >

При першому створенні проєкту Java нам буде запропоновано активувати відповідну функцію:

И на данном этапе нажмем на кноку Activate.

Потім з'явиться вікно налаштувань проєкту:

В поле Project Name дадим проєкту якое-нибудь название (в моем случае це HelloApp). Для всех остальных полей можна оставить значення по умолчанию. Післяднїї поле "Create Main Class" вказує, что автоматически в проєкте буде создан клас програми HelloApp, який буде находиться в одноименном пакеті helloapp. И в завершении створенняя проєкта нажмем на кнопку "Finish". И перед нами откроется новый проєкт в Netbeans:

Вузьке вікно зліва відображає всі відкриті проєкти та їхню структуру. У даному випадку у нас відкритий поки що лише один проєкт. А велике вікно справа представляє редактор коду з деякими додатковими функціями, як підцвітання коду, інтелектуальна підказка і т.д. Редактор коду вже має певний зміст за замовчуванням:


/*
 * To change this license header, choose License Headers in Project Properties.
 * To change this template file, choose Tools | Templates
 * and open the template in the editor.
 */
package helloapp;

/**
 *
 * @author Eugene
 */
public class HelloApp {

    /**
     * @param args the command line arguments
     */
    public static void main(String[] args) {
        // TODO code application logic here
    }
    
}

Тут, по-перше, багато коментарів, які нам не потрібні і які ми можемо видалити. По-друге, на початку файлу вказується, що клас буде знаходитися в пакеті helloapp за допомогою директивы package: package helloapp;. Затем, як вже рассматривалось в прошлой теме, у нас есть клас програми - клас HelloApp, який имїїт метод main.

Змінимо код за замовчуванням на наступний, щоб програма виводила найпростіший рядок на консоль:


package helloapp;

public class HelloApp {

    public static void main(String[] args) {
        
        System.out.println("Hello Java 15 from Netbeans!");
    }
    
}

Тепер запустимоо проєкт виконання. Для цього на панелі інструментів NetBeans натиснемо на зелену стрілку (або можна натиснути на клавішу F6, або вибрати меню пункт Run -> Run Project). І внизу NetBeans відкриється вікно виведення, в якому ми зможемоо спостерігати результати нашої програми:
