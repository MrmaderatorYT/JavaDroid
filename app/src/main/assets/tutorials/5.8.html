Словники. Інтерфейс Map та клас HashMap

Останнє оновлення: 12.10.2025

Словники (також называемые картами и отображениями) дозволяють сопоставить с визначенийным ключом некоторое значення. В мові Java для визначенийия базового функционала словарей имїїтся інтерфейс Map<K, V>. Он представляє структуру данных, где каждый елемент представляє пару "ключ-значення". При цем в рамках об'єкта Map все ключи уникальные. Такие колекції облегчают поиск елемента, якщо нам известен ключ - уникальный идентификатор об'єкта.

Следует відзначити, что в отличие от других інтерфейсов, которые представляютьь колекції, інтерфейс Map НЕ расширяет інтерфейс Collection.

Среди методів інтерфейса Map можна выделить следующие:

  • void clear(): очищает колекцію

  • boolean containsKey(Object k): повертає true, якщо колекція містить ключ k

  • boolean containsValue(Object v): повертає true, якщо колекція містить значення v

  • Set<Map.Entry<K, V>> entrySet(): повертає набор елементов колекції. Все елементы представляютьь об'єкт Map.Entry

  • boolean equals(Object obj): повертає true, якщо колекція идентична колекції, передаваемой через параметр obj

  • boolean isEmpty(): повертає true, якщо колекція пуста

  • V get(Object k): повертає значення об'єкта, ключ которого равен k. Если такого елемента не окажется, то повертаєся значення null

  • V getOrDefault(Object k, V defaultValue): повертає значення об'єкта, ключ которого равен k. Если такого елемента не окажется, то повертаєся значення defaultVlue

  • V put(K k, V v): помещает в колекцію новый об'єкт с ключом k и значенням v. Если в колекції вже есть об'єкт с підобным ключом, то он перезаписуєся. Після добавления повертає предыдущїї значення для ключа k, якщо он вже був в колекції. Если же ключа ще не буво в колекції, то повертаєся значення null

  • V putIfAbsent(K k, V v): помещает в колекцію новый об'єкт с ключом k и значенням v, якщо в колекції ще нет елемента с підобным ключом.

  • Set<K> keySet(): повертає набор всех ключей словника

  • Collection<V> values(): повертає набор всех значений словника

  • void putAll(Map<? extends K, ? extends V> map): додає в колекцію все об'єкты из словника map

  • V remove(Object k): видаляє об'єкт с ключом k

  • boolean remove(Object k, Object v): видаляє об'єкт с ключом k, якщо лише його значення равно v

  • V replace(K k, V v): заменяет значення для указанного ключа k на v

  • boolean replace(K key, V oldValue, V newValue): заменяет значення для указанного ключа key на newValue, якщо лише старое значення равно oldValue

  • int size(): повертає кількість елементів колекції

  • static <K,V> Map<K,V> ofEntries(Map.Entry<? extends K, ? extends V>... entries): повертає незмінюємоые словник, який містить елементы из entries

Щоб положить об'єкт в колекцію, використовується метод put, а чтобы отримати по ключу - метод get. Реалізація інтерфейса Map також дозволяє отримати наборы як ключей, так и значений. А метод entrySet() повертає набор всех елементов в виде об'єктов Map.Entry<K, V>.

Обобщенный інтерфейс Map.Entry<K, V> представляє об'єкт с ключом типу K и значенням типу V и визначає следующие методи:

  • boolean equals(Object obj): повертає true, якщо об'єкт obj, представляющий інтерфейс Map.Entry, идентичен текущему

  • K getKey(): повертає ключ об'єкта словника

  • V getValue(): повертає значення об'єкта словника

  • V setValue(V v): встановлює для текущего об'єкта значення v

  • int hashCode(): повертає хеш-код данного об'єкта

При переборе об'єктов словника мы будем оперировать цими методуми для работы с ключами и значеннями об'єктов.

Наприклад, створимо незмінюємоый словник:


import java.util.Map;
 
public class Program{
      
    public static void main(String[] args) {
          
        Map<Integer,String> map = Map.ofEntries( 
                                        Map.entry(1, "Tom"),
                                        Map.entry(2, "Bob"),
                                        Map.entry(3, "Sam")
                                    );
        System.out.println(map);      // {3=Sam, 2=Bob, 1=Tom}
    }
}

У даному випадку створюєся словник, где ключи представляютьь тип Integer, а значення - тип String. Для створенняя словника застосовується мтеод Map.ofEntries(), в який передаються об'єкты Map.Entry, создаваемые за допомогою методу Map.entry(). Первый параметр цього методу - ключ, а второй - сопоставляемое с ним значення.

Наприклад, выражение

Map.entry(1, "Tom")

Створює об'єкт Map.Entry, где ключ - число 1, а значення - рядок "Tom".

Поскольку результатом методу є незмінюємоый словник типу java.util.ImmutableCollections$MapN, то мы не можемо изменять его значення. Но посмотрим, якие другие операції мы можемо произвести со словарем.


import java.util.Map;
 
public class Program{
      
    public static void main(String[] args) {
          
        Map<Integer,String> map = Map.ofEntries( 
                                        Map.entry(1, "Tom"),
                                        Map.entry(2, "Bob"),
                                        Map.entry(3, "Sam")
                                    );

        // отримуємо размер словникая
        System.out.println(map.size());   // 3

        // отримуємо все ключи
        System.out.println(map.keySet());   // [3, 2, 1]
        // отримуємо все значення
        System.out.println(map.values());   // [Sam, Bob, Tom]

        // отримуємо значення по ключу 2 из словника
        String item2 = map.get(2);
        System.out.println("Item 2: " + item2);   // Item 2: Bob

        // перебор словника
        for(Map.Entry<Integer,String> item : map.entrySet()){

            // отримуємо ключ
            Integer key = item.getKey();
            // отримуємо значення
            String value = item.getValue();
            System.out.printf("{ %d : %s}\n", key, value);
        }
    }
}

HashMap

Базовым класом для всех словарей є абстрактный клас AbstractMap, який реалізує большую часть методів інтерфейса Map. Наиболїї распространенным класом словарей є HashMap, який найдеся от класа AbstractMap и представляє реализацию інтерфейса Map на основе хеш-таблиці.

Для створенняя словника HashMap застосовується ряд конструкторов:

  • HashMap(): створює порожнійHashMapс начальной емкостью по умолчанию (16) и коэффициентом загрузки по умолчанию (0,75)

  • HashMap(int initialCapacity): створює порожнійHashMapс указанной начальной ємністью и коэффициентом загрузки по умолчанию (0,75)

  • HashMap(int initialCapacity, float loadFactor): створює порожнійHashMapс указанной начальной емкостью и коэффициентом загрузки

  • HashMap(Map<? extends K, ? extends V> m): створює новыйHashMapс теми же сопоставлениями, что и у заданного Map

Створення HashMap и додавання в него елементов:


import java.util.Map;
import java.util.HashMap;
 
public class Program{
      
    public static void main(String[] args) {
          
        // порожній словник
        var dict = new HashMap<String,String>();

        // додаємо в него елементы
        dict.put("red", "красный");
        dict.put("green", "зеленый");
        dict.put("blue", "синий");
        System.out.println(dict);   // {red=красный, green=зеленый, blue=синий}

        Map<String,String> map = Map.ofEntries( 
                                    Map.entry("yellow", "желтый"),
                                    Map.entry("brown", "коричневый")
                                );
        // додаємо другой словник с данными
        dict.putAll(map);
        System.out.println(dict);   // {red=красный, green=зеленый, blue=синий, yellow=желтый, brown=коричневый}
    }
}

У даному випадку створюємо словник, где и ключи, и значення представляютьь рядки. Затем за допомогою методів put()/putAll() додаємо ряд об'єктов.

ПРимер получения данных из словника:


import java.util.Map;
import java.util.HashMap;
 
public class Program{
      
    public static void main(String[] args) {

        Map<String,String> map = Map.ofEntries( 
                                    Map.entry("red", "красный"),
                                    Map.entry("green", "зеленый"),
                                    Map.entry("blue", "синий"),
                                    Map.entry("yellow", "желтый")
                                );
        var dict = new HashMap<String,String>(map);
        System.out.println(dict);   // {red=красный, yellow=желтый, green=зеленый, blue=синий}

        // отримуємо размер словникая
        System.out.println(dict.size());   // 4

        // отримуємо все ключи
        System.out.println(dict.keySet());   // [red, yellow, green, blue]
        // отримуємо все значення
        System.out.println(dict.values());   // [красный, желтый, зеленый, синий]

        // отримуємо значення по ключу "red" из словника
        String red = dict.get("red");
        System.out.println("red: " + red);   // red: красный

        // проверяем наличие слова в словаре
        var hasBlack = dict.containsKey("black");
        System.out.println("black: " + (hasBlack ? dict.get("black"):" отсутствует"));   // black:  отсутствует

        // перебор словника
        for(var word : dict.entrySet()){

            // отримуємо ключ 
            var source = word.getKey();
            // отримуємо значення
            var translation = word.getValue();
            System.out.printf("%s : %s\n", source, translation);
        }
    }
}

Пример изменения и удаления елементов словника:


import java.util.Map;
import java.util.HashMap;
 
public class Program{
      
    public static void main(String[] args) {

        Map<String,String> map = Map.ofEntries( 
                                    Map.entry("red", "красный"),
                                    Map.entry("green", "зеленый"),
                                    Map.entry("blue", "синий"),
                                    Map.entry("yellow", "желтый")
                                );
        var dict = new HashMap<String,String>(map);
        System.out.println(dict);   // {red=красный, green=зеленый, blue=синий, yellow=желтый}

        // меняем по ключу "red" значення на "червонный"
        dict.replace("red", "червонный");
        System.out.println(dict);   // {red=червонный, green=зеленый, blue=синий, yellow=желтый}

        // меняем по ключу "red" значення на "алый", лише якщо значення "красный"
        dict.replace("red", "алый", "красный");

        // видаляємо yellow
        dict.remove("yellow");
        System.out.println(dict);   // {red=червонный, green=зеленый, blue=синий}

        // видаляємо всі елементи
        dict.clear();
        System.out.println(dict);   // {}
    }
}

Щоб добавить або заменить елемент, використовується метод put, або replace, а чтобы отримати його значення по ключу - метод get. За допомогою других методів інтерфейса Map також производятся другие манипуляции над елементами: перебор, отримання ключей, значений, видалення.
