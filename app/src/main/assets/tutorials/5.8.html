Словники. Інтерфейс Map та клас HashMap

Останнє оновлення: 12.10.2025

Словари (також называемые картами и отображениями) дозволяють сопоставить с определенным ключом некоторое значение. В мові Java для определения базового функционала словарей имїїтся інтерфейс Map<K, V>. Он представляє структуру данных, где каждый элемент представляє пару "ключ-значение". При цем в рамках об'єкта Map все ключи уникальные. Такие коллекции облегчают поиск элемента, якщо нам известен ключ - уникальный идентификатор об'єкта.

Следует відзначити, что в отличие от других інтерфейсов, которые представляють коллекции, інтерфейс Map НЕ расширяет інтерфейс Collection.

Среди методів інтерфейса Map можна выделить следующие:

  • void clear(): очищает коллекцию

  • boolean containsKey(Object k): повертає true, якщо коллекция містить ключ k

  • boolean containsValue(Object v): повертає true, якщо коллекция містить значение v

  • Set<Map.Entry<K, V>> entrySet(): повертає набор элементов коллекции. Все элементы представляють об'єкт Map.Entry

  • boolean equals(Object obj): повертає true, якщо коллекция идентична коллекции, передаваемой через параметр obj

  • boolean isEmpty(): повертає true, якщо коллекция пуста

  • V get(Object k): повертає значение об'єкта, ключ которого равен k. Если такого элемента не окажется, то повертаєся значение null

  • V getOrDefault(Object k, V defaultValue): повертає значение об'єкта, ключ которого равен k. Если такого элемента не окажется, то повертаєся значение defaultVlue

  • V put(K k, V v): помещает в коллекцию новый об'єкт с ключом k и значением v. Если в коллекции вже есть об'єкт с подобным ключом, то он перезаписуєся. После добавления повертає предыдущїї значення для ключа k, якщо он вже був в коллекции. Если же ключа ще не буво в коллекции, то повертаєся значение null

  • V putIfAbsent(K k, V v): помещает в коллекцию новый об'єкт с ключом k и значением v, якщо в коллекции ще нет элемента с подобным ключом.

  • Set<K> keySet(): повертає набор всех ключей словаря

  • Collection<V> values(): повертає набор всех значений словаря

  • void putAll(Map<? extends K, ? extends V> map): додає в коллекцию все об'єкты из словаря map

  • V remove(Object k): видаляє об'єкт с ключом k

  • boolean remove(Object k, Object v): видаляє об'єкт с ключом k, якщо лише його значення равно v

  • V replace(K k, V v): заменяет значение для указанного ключа k на v

  • boolean replace(K key, V oldValue, V newValue): заменяет значение для указанного ключа key на newValue, якщо лише старое значение равно oldValue

  • int size(): повертає количество элементов коллекции

  • static <K,V> Map<K,V> ofEntries(Map.Entry<? extends K, ? extends V>... entries): повертає неизменяемые словарь, який містить элементы из entries

Щоб положить об'єкт в коллекцию, використовується метод put, а чтобы отримати по ключу - метод get. Реализация інтерфейса Map також дозволяє отримати наборы як ключей, так и значений. А метод entrySet() повертає набор всех элементов в виде об'єктов Map.Entry<K, V>.

Обобщенный інтерфейс Map.Entry<K, V> представляє об'єкт с ключом типу K и значением типу V и определяет следующие методы:

  • boolean equals(Object obj): повертає true, якщо об'єкт obj, представляющий інтерфейс Map.Entry, идентичен текущему

  • K getKey(): повертає ключ об'єкта словаря

  • V getValue(): повертає значение об'єкта словаря

  • V setValue(V v): устанавливает для текущего об'єкта значение v

  • int hashCode(): повертає хеш-код данного об'єкта

При переборе об'єктов словаря мы будем оперировать цими методами для работы с ключами и значениями об'єктов.

Наприклад, создадим неизменяемый словарь:


import java.util.Map;
 
public class Program{
      
    public static void main(String[] args) {
          
        Map<Integer,String> map = Map.ofEntries( 
                                        Map.entry(1, "Tom"),
                                        Map.entry(2, "Bob"),
                                        Map.entry(3, "Sam")
                                    );
        System.out.println(map);      // {3=Sam, 2=Bob, 1=Tom}
    }
}

В данном случае створюєся словарь, где ключи представляють тип Integer, а значения - тип String. Для создания словаря застосовується мтеод Map.ofEntries(), в який передаются об'єкты Map.Entry, создаваемые за допомогою метода Map.entry(). Первый параметр цього метода - ключ, а второй - сопоставляемое с ним значение.

Наприклад, выражение

Map.entry(1, "Tom")

Створює об'єкт Map.Entry, где ключ - число 1, а значение - строка "Tom".

Поскольку результатом метода є неизменяемый словарь типу java.util.ImmutableCollections$MapN, то мы не можемо изменять его значения. Но посмотрим, якие другие операции мы можемо произвести со словарем.


import java.util.Map;
 
public class Program{
      
    public static void main(String[] args) {
          
        Map<Integer,String> map = Map.ofEntries( 
                                        Map.entry(1, "Tom"),
                                        Map.entry(2, "Bob"),
                                        Map.entry(3, "Sam")
                                    );

        // получаем размер словаряя
        System.out.println(map.size());   // 3

        // получаем все ключи
        System.out.println(map.keySet());   // [3, 2, 1]
        // получаем все значения
        System.out.println(map.values());   // [Sam, Bob, Tom]

        // получаем значение по ключу 2 из словаря
        String item2 = map.get(2);
        System.out.println("Item 2: " + item2);   // Item 2: Bob

        // перебор словаря
        for(Map.Entry<Integer,String> item : map.entrySet()){

            // получаем ключ
            Integer key = item.getKey();
            // получаем значение
            String value = item.getValue();
            System.out.printf("{ %d : %s}\n", key, value);
        }
    }
}

HashMap

Базовым классом для всех словарей є абстрактный класс AbstractMap, який реализует большую часть методів інтерфейса Map. Наиболїї распространенным классом словарей є HashMap, який найдеся от класса AbstractMap и представляє реализацию інтерфейса Map на основе хеш-таблицы.

Для создания словаря HashMap застосовується ряд конструкторов:

  • HashMap(): створює пустойHashMapс начальной емкостью по умолчанию (16) и коэффициентом загрузки по умолчанию (0,75)

  • HashMap(int initialCapacity): створює пустойHashMapс указанной начальной ємністью и коэффициентом загрузки по умолчанию (0,75)

  • HashMap(int initialCapacity, float loadFactor): створює пустойHashMapс указанной начальной емкостью и коэффициентом загрузки

  • HashMap(Map<? extends K, ? extends V> m): створює новыйHashMapс теми же сопоставлениями, что и у заданного Map

Створення HashMap и добавление в него элементов:


import java.util.Map;
import java.util.HashMap;
 
public class Program{
      
    public static void main(String[] args) {
          
        // пустой словарь
        var dict = new HashMap<String,String>();

        // добавляем в него элементы
        dict.put("red", "красный");
        dict.put("green", "зеленый");
        dict.put("blue", "синий");
        System.out.println(dict);   // {red=красный, green=зеленый, blue=синий}

        Map<String,String> map = Map.ofEntries( 
                                    Map.entry("yellow", "желтый"),
                                    Map.entry("brown", "коричневый")
                                );
        // добавляем другой словарь с данными
        dict.putAll(map);
        System.out.println(dict);   // {red=красный, green=зеленый, blue=синий, yellow=желтый, brown=коричневый}
    }
}

В данном случае создаем словарь, где и ключи, и значения представляють строки. Затем за допомогою методів put()/putAll() добавляем ряд об'єктов.

ПРимер получения данных из словаря:


import java.util.Map;
import java.util.HashMap;
 
public class Program{
      
    public static void main(String[] args) {

        Map<String,String> map = Map.ofEntries( 
                                    Map.entry("red", "красный"),
                                    Map.entry("green", "зеленый"),
                                    Map.entry("blue", "синий"),
                                    Map.entry("yellow", "желтый")
                                );
        var dict = new HashMap<String,String>(map);
        System.out.println(dict);   // {red=красный, yellow=желтый, green=зеленый, blue=синий}

        // получаем размер словаряя
        System.out.println(dict.size());   // 4

        // получаем все ключи
        System.out.println(dict.keySet());   // [red, yellow, green, blue]
        // получаем все значения
        System.out.println(dict.values());   // [красный, желтый, зеленый, синий]

        // получаем значение по ключу "red" из словаря
        String red = dict.get("red");
        System.out.println("red: " + red);   // red: красный

        // проверяем наличие слова в словаре
        var hasBlack = dict.containsKey("black");
        System.out.println("black: " + (hasBlack ? dict.get("black"):" отсутствует"));   // black:  отсутствует

        // перебор словаря
        for(var word : dict.entrySet()){

            // получаем ключ 
            var source = word.getKey();
            // получаем значение
            var translation = word.getValue();
            System.out.printf("%s : %s\n", source, translation);
        }
    }
}

Пример изменения и удаления элементов словаря:


import java.util.Map;
import java.util.HashMap;
 
public class Program{
      
    public static void main(String[] args) {

        Map<String,String> map = Map.ofEntries( 
                                    Map.entry("red", "красный"),
                                    Map.entry("green", "зеленый"),
                                    Map.entry("blue", "синий"),
                                    Map.entry("yellow", "желтый")
                                );
        var dict = new HashMap<String,String>(map);
        System.out.println(dict);   // {red=красный, green=зеленый, blue=синий, yellow=желтый}

        // меняем по ключу "red" значение на "червонный"
        dict.replace("red", "червонный");
        System.out.println(dict);   // {red=червонный, green=зеленый, blue=синий, yellow=желтый}

        // меняем по ключу "red" значение на "алый", лише якщо значение "красный"
        dict.replace("red", "алый", "красный");

        // удаляем yellow
        dict.remove("yellow");
        System.out.println(dict);   // {red=червонный, green=зеленый, blue=синий}

        // удаляем все элементы
        dict.clear();
        System.out.println(dict);   // {}
    }
}

Щоб добавить або заменить элемент, використовується метод put, або replace, а чтобы отримати його значення по ключу - метод get. С помощью других методів інтерфейса Map також производятся другие манипуляции над элементами: перебор, получение ключей, значений, удаление.
