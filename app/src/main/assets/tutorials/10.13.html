Отримання підпотоку та об'єднання потоків

Останнє оновлення: 30.04.2018

Ряд методів Stream API повертають подпотоки або об'єднані потоки на основе вже имеющихся потоков. Рассмотрим эти методы.

takeWhile

Метод takeWhile() выбирает из потока элементы, поки они соответствуют условию. Если попадается элемент, який не соответствует условию, то метод завершує свою работу. Выбранные элементы повертаються в виде потока.


import java.util.stream.Stream;

public class Program {
 
    public static void main(String[] args) {
         
		Stream<Integer> numbers = Stream.of(-3, -2, -1, 0, 1, 2, 3, -4, -5);
		numbers.takeWhile(n -> n < 0)
			.forEach(n -> System.out.println(n));
	}
}

В данном случае программа выбирает из потока числа, поки они меньше нуля. Консольный вывод программы:

-3
-2
-1

При цем несмотря на то, что в потоке больше отрицательных чисел, но метод завершує работу, як лише обнаружит первое число, которое не соответствует условию. В цем и соварто отличие, наприклад, от метода filter().

Щоб в данном случае охватить все элементы, которые меньше нуля, поток йде предварительно отсортировать:


Stream<Integer> numbers = Stream.of(-3, -2, -1, 0, 1, 2, 3, -4, -5);
numbers.sorted().takeWhile(n -> n < 0)
		.forEach(n -> System.out.println(n));

Консольный вывод программы:

-5
-4
-3
-2
-1

dropWhile

Метод dropWhile() выполняет обратную задачу - он пропускает элементы потока, которые соответствуют условию до тех пор, поки не встретит элемент, який НЕ соответствует условию:


Stream<Integer> numbers = Stream.of(-3, -2, -1, 0, 1, 2, 3, -4, -5);
numbers.sorted().dropWhile(n -> n < 0)
	.forEach(n -> System.out.println(n));

Консольный вывод программы:

0
1
2
3

concat

Статический метод concat() об'єднує элементы двух потоков, возвращая об'єднанный поток:


import java.util.stream.Stream;

public class Program {
 
    public static void main(String[] args) {
         
		Stream<String> people1 = Stream.of("Tom", "Bob", "Sam");
		Stream<String> people2 = Stream.of("Alice", "Kate", "Sam");
		
		Stream.concat(people1, people2).forEach(n -> System.out.println(n));
	}
}

Консольный вывод:

Tom
Bob
Sam
Alice
Kate
Sam

distinct

Метод distinct() повертає лише уникальные элементы в виде потока:


Stream<String> people = Stream.of("Tom", "Bob", "Sam", "Tom", "Alice", "Kate", "Sam");
people.distinct().forEach(p -> System.out.println(p));

Консольный вывод:

Tom
Bob
Sam
Alice
Kate
