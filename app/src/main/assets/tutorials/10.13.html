Отримання підпотіку та об'єднання потіків

Останнє оновлення: 30.04.2018

Ряд методів Stream API повертають підпотіки або об'єднані потіки на основе вже имеющихся потіков. Рассмотрим эти методи.

takeWhile

Метод takeWhile() выбирает из потіка елементы, поки они відповідають условию. Если попадается елемент, який не відповідає условию, то метод завершує свою работу. Выбранные елементы повертаються в виде потіка.


import java.util.stream.Stream;

public class Program {
 
    public static void main(String[] args) {
         
		Stream<Integer> numbers = Stream.of(-3, -2, -1, 0, 1, 2, 3, -4, -5);
		numbers.takeWhile(n -> n < 0)
			.forEach(n -> System.out.println(n));
	}
}

У даному випадку програма выбирает из потіка числа, поки они меньше нуля. Консольный вивід програми:

-3
-2
-1

При цем несмотря на то, что у потоці больше отрицательных чисел, но метод завершує работу, як лише обнаружит первое число, которое не відповідає условию. В цем и соварто отличие, наприклад, от методу filter().

Щоб у даному випадку охватить всі елементи, которые меньше нуля, потік йде предварительно отсортировать:


Stream<Integer> numbers = Stream.of(-3, -2, -1, 0, 1, 2, 3, -4, -5);
numbers.sorted().takeWhile(n -> n < 0)
		.forEach(n -> System.out.println(n));

Консольный вивід програми:

-5
-4
-3
-2
-1

dropWhile

Метод dropWhile() виконує обратную завданьу - он пропускает елементы потіка, которые відповідають условию до тех пор, поки не встретит елемент, який НЕ відповідає условию:


Stream<Integer> numbers = Stream.of(-3, -2, -1, 0, 1, 2, 3, -4, -5);
numbers.sorted().dropWhile(n -> n < 0)
	.forEach(n -> System.out.println(n));

Консольный вивід програми:

0
1
2
3

concat

Статический метод concat() об'єднує елементы двух потіков, возвращая об'єднанный потік:


import java.util.stream.Stream;

public class Program {
 
    public static void main(String[] args) {
         
		Stream<String> people1 = Stream.of("Tom", "Bob", "Sam");
		Stream<String> people2 = Stream.of("Alice", "Kate", "Sam");
		
		Stream.concat(people1, people2).forEach(n -> System.out.println(n));
	}
}

Консольный вивід:

Tom
Bob
Sam
Alice
Kate
Sam

distinct

Метод distinct() повертає лише уникальные елементы в виде потіка:


Stream<String> people = Stream.of("Tom", "Bob", "Sam", "Tom", "Alice", "Kate", "Sam");
people.distinct().forEach(p -> System.out.println(p));

Консольный вивід:

Tom
Bob
Sam
Alice
Kate
