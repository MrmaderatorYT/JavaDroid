Перша програма у Eclipse

Останнє оновлення: 11.03.2022

Ще одной очень популярной средой для разработки приложений под Java є Eclipse. IDE Eclipse первоначально бува створена и развивалась компанией IBM, но впоследствии разработка бува выделена в отдельный проект Eclipse Foundation. Eclipse є бесплатной средой, которая к тому же обладает большими возможнастями, позволяя создавать на Java самые разные приложения.

Для начала установим последнюю версию Eclipse, яку можна найти по адресу https://www.eclipse.org/downloads/. На странице загрузок найдем рядом с названием текущей версии Eclipse (на момент написания статьи це версия Oxygen) кнопку "Download" и нажмем на нїї.

Після натискання на кнопку нас перенаправить власне на сторінку завантаження, де необхідно буде натиснути кнопку "Download" для завантаження установчого пакета:

Після завантаження інсталяційного пакета запустимо його:

За фактом Eclipse – це цілий набір середовищ для різних ситуацій та мов програмування. І програма установки пропонує нам вибрати одну із опцій для інсталяції. Оскільки ми збираємося працювати з Java, то нас цікавитимуть у разі перш за все дві опції:

  • Eclipse IDE для Java Developers

  • Eclipse IDE для Enterprise Java та Web Developers

Первая опция надає базовые возможнасти для работы с Java, а вторая дозволяє також создавать веб-приложения и сервисы. По сути вторая опция включает и первую опцию. Тому выберем Eclipse IDE для Enterprise Java та Web Developers.

Далі потрібно буде вказати папку для встановлення (або залишити вибрану папку за замовчуванням), а також вказати шлях до JDK і натиснути кнопку Install:

Далі треба буде ухвалити ліцензійну угоду:

І потім власне розпочнеться встановлення. Після встановлення запустимо Eclipse. Спочатку буде запропоновано встановити каталог проектів. Залишимо значення за промовчанням і натиснемо на кнорку Launch:

Далїї перейдем к пункту меню File -> New -> Project:

Далїї нам откроется окно выбора шаблона проекта. Выберем Java Project и нажмем на кнопку Next:

Затем буде предложено установить настройки проекта. В частности, в поле Project Name необхідно задать имя проекта. Допустим, оно буде helloapp:

Також обращаю внимание на поле JRE, которое задает используемую версию Java. Здесь можна використовувати версию по умолчанию або изменить її в зависимости от того, якую версию JDK/JRE мы хочемо використовувати.

Крім того, в самом низу окна есть поле Create module-info.java file. Поки эта опция не принципиальна, поцему чтобы не создавать лишние файлы в проекте, можна снять отметку с цього поля.

Всі інші налаштування можна залишити за замовчуванням, і наприкінці натиснемо на кнопку Finish для створення проекту.

Наступне діалогове вікно спитає, чи треба відкривати перспективу проекту:

В данном случае эта возможнасть не имїїт значения, но нажмем на кнопку Open Perspective.

Після створення проект буде відкрито в Eclipse. Якщо середовище не відкриває за замовчуванням проект, необхідно натиснути на кнопку Restore:

За замовчуванням проект буде порожнім.

Все файлы с исходным кодом на Java в проекте помещаются в папку src. Нажмем на нїї правой кнопкой мыши и в контекстном меню выберем New -> Class:

Потім з'явиться вікно налаштування класу. Тут у полі Name необхідно встановити назву класу. Припустимо, він називатиметься Program. Решта налаштування можна залишити за замовчуванням:

Після цього до проекту буде додано файл Program.java, а в центральній частині Eclipse ми зможемоо побачити його код. Змінимо цей код на наступний:


package helloapp;

public class Program {

	public static void main(String[] args) {
		System.out.println("Hello Eclipse!");
	}
}

Для запуска проекта нажмем на панели инструментов на зеленую стрелочку, або на пункт меню Run -> Run. И внизу среды в поле Console мы увидим результат работы программы (то есть фактически консольный вывод).
