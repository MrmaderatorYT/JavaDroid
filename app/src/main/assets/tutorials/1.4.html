Перша програма у Eclipse

Останнє оновлення: 11.03.2022

Ще одной дуже популярной средой для разработки приложений під Java є Eclipse. IDE Eclipse первоначально бува створена и развивалась компанией IBM, но впіслядствии разработка бува выделена в отдельный проєкт Eclipse Foundation. Eclipse є безкоштовним середовищем, яке до того ж володіє великими можливостями, дозволяючи створювати на Java найрізноманітніші додатки.

Для начала установим післяднюю версию Eclipse, яку можна найти по адресу https://www.eclipse.org/downloads/. На сторінці завантажень знайдемо поруч з назвою текущей версии Eclipse (на момент написания статьи це версия Oxygen) кнопку "Download" и нажмем на нїї.

Після натискання на кнопку нас перенаправить власне на сторінку завантаження, де необхідно буде натиснути кнопку "Download" для завантаження установчого пакету:

Після завантаження інсталяційного пакету запустимоо його:

За фактом Eclipse – це цілий набір середовищ для різних ситуацій та мов програмування. І програма установки пропонує нам вибрати одну із опцій для інсталяції. Оскільки ми збираємося працювати з Java, то нас цікавитимуть у разі перш за все дві опції:

  • Eclipse IDE для Java Developers

  • Eclipse IDE для Enterprise Java та Web Developers

Перша опція надає базові можливості для роботи з Java, а друга дозволяє також створювати веб-додатки та сервіси. По сути вторая опция включает и первую опцию. Тому оберемо Eclipse IDE для Enterprise Java та Web Developers.

Далі потрібно буде вказати папку для встановлення (або залишити вибрану папку за замовчуванням), а також вказати шлях до JDK і натиснути кнопку Install:

Далі треба буде ухвалити ліцензійну угоду:

І потім власне розпочнеться встановлення. Після встановлення запустимоо Eclipse. Спочатку буде запропоновано встановити каталог проєктів. Залишимо значення за промовчанням і натиснемо на кнорку Launch:

Далїї перейдем к пункту меню File -> New -> Project:

Далїї нам откроется окно выбора шаблона проєкта. Выберем Java Project и нажмем на кнопку Next:

Затем буде предложено установить настройки проєкта. Зокрема, в поле Project Name необхідно задать ім'я проєкта. Допустим, оно буде helloapp:

Також обращаю внимание на поле JRE, которое задає використовуємоую версию Java. Здесь можна використовувати версию по умолчанию або змінити її в зависимости от того, якую версию JDK/JRE мы хочемо використовувати.

Крім того, у самому низу вікна є поле Create module-info.java file. Поки ця опція не принципова, щоб не створювати лишние файли в проєкте, можна снять отметку с цього поля.

Всі інші налаштування можна залишити за замовчуванням, і наприкінці натиснемо на кнопку Finish для створення проєкту.

Наступне діалогове вікно спитає, чи треба відкривати перспективу проєкту:

У даному випадку эта возможнасть не имїїт значення, но нажмем на кнопку Open Perspective.

Після створення проєкт буде відкрито в Eclipse. Якщо середовище не відкриває за замовчуванням проєкт, необхідно натиснути на кнопку Restore:

За замовчуванням проєкт буде порожнім.

Все файли с исходным кодом на Java в проєкте помещаются в папку src. Нажмем на нїї правой кнопкой мыши и в контекстном меню оберемо New -> Class:

Потім з'явиться вікно налаштування класу. Тут у полі Name необхідно встановити назву класу. Припустимо, він називатиметься Program. Решта налаштування можна залишити за замовчуванням:

Після цього до проєкту буде додано файл Program.java, а в центральній частині Eclipse ми зможемоо побачити його код. Змінимо цей код на наступний:


package helloapp;

public class Program {

	public static void main(String[] args) {
		System.out.println("Hello Eclipse!");
	}
}

Для запуску проєкта нажмем на панели инструментов на зеленую стрелочку, або на пункт меню Run -> Run. И внизу среды в поле Console мы увидим результат работы програми (то есть фактически консольный вивід).
