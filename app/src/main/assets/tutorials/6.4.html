Класы ByteArrayInputStream и ByteArrayOutputStream

Останнє оновлення: 26.04.2018

Для работы с масивами байтов - их читання и записи використовуються класы ByteArrayInputStream и ByteArrayOutputStream.

Читання масива байтов и клас ByteArrayInputStream

Клас ByteArrayInputStream представляє входной потік, использующий в качестве источника данных масив байтов. Он имїїт следующие конструктори:


ByteArrayInputStream(byte[] buf) 
ByteArrayInputStream(byte[] buf, int offset, int length)

В качестве параметрів конструктори використовують масив байтов buf, из которого производится считывание, смщение относительно начала масива offset и кількість считываемых символів length.

Считаємо масив байтов и выведем его на экран:


import java.io.*;

public class Program {

    public static void main(String[] args) {
        
        byte[] array1 = new byte[]{1, 3, 5, 7};
        ByteArrayInputStream byteStream1 = new ByteArrayInputStream(array1);
        int b;
        while((b=byteStream1.read())!=-1){
        
            System.out.println(b);
        }
        
        String text = "Hello world!";
        byte[] array2 = text.getBytes();
		// считываем 5 символів
        ByteArrayInputStream byteStream2 = new ByteArrayInputStream(array2, 0, 5);
        int c;
        while((c=byteStream2.read())!=-1){
        
            System.out.println((char)c);
		}
    } 
}

В отличие от других класов потіков для закрытия об'єкта ByteArrayInputStream не требуется вызывать метод close.

Запис масива байт и клас ByteArrayOutputStream

Клас ByteArrayOutputStream представляє потік вивіда, использующий масив байтов в качестве места вивіда.

Щоб створити об'єкт данного класа, мы можемо використовувати один из его конструкторов:


ByteArrayOutputStream() 
ByteArrayOutputStream(int size)

Первая версия створює масив для хранения байтов длиной в 32 байта, а вторая версия створює масив длиной size.

Рассмотрим застосування класа:


import java.io.*;

public class Program {

    public static void main(String[] args) {
        
        ByteArrayOutputStream baos = new ByteArrayOutputStream();
        String text = "Hello Wolrd!";
        byte[] buffer = text.getBytes();
        try{
            baos.write(buffer);
        }
        catch(Exception ex){
        
            System.out.println(ex.getMessage());
        }
        // превращаем масив байтов в рядок
        System.out.println(baos.toString());
        
        // отримуємо масив байтов и вивідим по символьно
        byte[] array = baos.toByteArray();
        for(byte b: array){
        
            System.out.print((char)b);
        }
        System.out.println();
    } 
}

Как и в других потіках вивіда в класе ByteArrayOutputStream визначений метод write, який записує в потік некоторые дані. У даному випадку мы записуємо в потік масив байтов. Цет масив байтов записуєся в об'єкте ByteArrayOutputStream в защищенное поле buf, которое представляє також масив байтов (protected byte[] buf).

Так як метод write может сгенерировать виняток, то вызов цього методу помещается в блок try..catch.

Використовуючи методи toString() и toByteArray(), можна отримати масив байтов buf в виде текста або непосредственно в виде масива байт.

За допомогою методу writeTo мы можемо вывести масив байт в другой потік. Данный метод в качестве параметра приймає об'єкт OutputStream, в який производится запис масива байт:


ByteArrayOutputStream baos = new ByteArrayOutputStream();
String text = "Hello Wolrd!";
byte[] buffer = text.getBytes();
try{
    baos.write(buffer);
}
catch(Exception ex){
        
    System.out.println(ex.getMessage());
}
try(FileOutputStream fos = new FileOutputStream("hello.txt")){
    
	baos.writeTo(fos);
}
catch(IOException e){
            
    System.out.println(e.getMessage());
}

Після виконання цей програми в папке с програмой появится файл hello.txt, який буде містятьь рядок "Hello Wolrd!".

И в заключении також треба сказать, что як и для об'єктов ByteArrayInputStream, для ByteArrayOutputStream не треба явным чином закрывать потік за допомогою методу close.
