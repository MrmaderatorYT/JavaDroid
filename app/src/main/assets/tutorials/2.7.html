Методи

Останнє оновлення: 16.09.2025

Если змінні класа хранят состояние об'єкта, то методи представляютьь его поведение. Методи містять собой набор операторов, которые виконують визначенийные действия. И якщо нам треба визначити в класе некоторые действия, нам треба визначити методи.

Визначення методів

Загальне визначення методів виглядає так:


[модификаторы] тип_повертаємоого_значення название_методу ([параметри]){
	// тело методу
}

Модифікатори та параметри необов'язкові.

За замовчуванням головний клас будь-якої програми Java містить метод main, який служить точкою входу в програму:


class Program{

    public static void main(String[] args) {  

        System.out.println("Hello !");
    }
}

Ключові слова public и static є модификаторами. Далїї идет тип повертаємоого значення. Ключевое слово void вказує на то, что метод ничего не повертає.

Затем идут название методу - main и в скобках параметри методу - String[] args. И в фигурные скобки заключено тело методу - все действия, которые он виконує.

Наприклад створимо клас і визначимо у ньому метод:

      
class Person{ 
    String name;
    int age;

    void print(){
        System.out.printf("Name: %s	\tAge: %d\n", name, age);
    }
}

class Program{

    public static void main(String[] args) {  

    }
}

Здесь визначений клас Person с двумя полями - name и age и одним методом print. Иетод print в качестве повертаємоого типу имїїт тип void (то есть по сути ничего не повертає) и не приймає нияких параметрів. Все что робить цей метод - це виводить на консоль значення змінних name и age.

Но чтобы пріменіть метод, его треба вызвать.

Виклик методу

Виклик методу здійснюється у формі:


ім'я_методу(аргументы);

Після імені методу вказуються дужки, у яких перераховуються аргументи - значення параметрів методу.

Обращаю внимание, что в класе Program визначений метод main, и нам не треба буво его нияк вызывать. Дело в том, что цей метод автоматически вызывается средой виконання Java, и нам не треба его явным чином вызывать. Остальные методи нам треба явным чином вызвать. И поскольку при запуску програми запускує метод main(), то все остальные треба вручную явным чином вызывать в методі main (или в других методух, которые буви також вызваны ранїї). Наприклад, вызовем выше визначенийный метод print:


class Program{

    public static void main(String[] args) {  

  		// створюємо об'єкт Person
        Person tom = new Person();

  		// устанавливаем значення полей класа
        tom.name = "Tom";
        tom.age = 41;

  		// вызываем метод print
        tom.print();
    }
}

class Person{ 
    String name;
    int age;

    void print(){
        System.out.printf("Name: %s	\tAge: %d\n", name, age);
    }
}

Для вызова методу print в методі main використовується выражение tom.print(). То есть як и в случае с полями, після імені змінної класа идет крапка и затем ім'я методу. Поскольку метод print не имеtт нияких параметрів, то після его названия при вызове ставятся пустые скобки. В итоге при выполнении програми мы увидим на консоли информацию о полях об'єкта:


Name: Tom		Age: 41

Одна з переваг методів полягає в тому, що ми можемоо винести деякі спільні дії окремий метод і потім викликати багаторазово їх у різних місцях програми. Наприклад:


class Program{

    public static void main(String[] args) {  

        Person tom = new Person();

        tom.name = "Tom";
        tom.age = 41;
        tom.print();

        tom.name = "Tomas";
        tom.print();

        tom.name = "Tommy";
        tom.print();
    }
}

class Person{ 
    String name;
    int age;

    void print(){
        System.out.printf("Name: %s	\tAge: %d\n", name, age);
    }
}

Тут ми три рази встановлюємо ім'я людини та за допомогою методу print перевіряємо його встановлення. У результаті отримаємо наступний консольний висновок:


Name: Tom		Age: 41
Name: Tomas		Age: 41
Name: Tommy		Age: 41

Виклик методів у інших методух класу

Одні методи класу можуть викликати інші методи:


class Program{

    public static void main(String[] args) {  

        Person tom = new Person();  // створюємо об'єкт Person

        tom.name = "Tom";
        tom.age = 41;
        tom.print();
    }
}

// клас Person
class Person{ 
    String name;
    int age;

    void printName(){
        System.out.printf("Name: %s\n", name);
    }
    void printAge(){
        System.out.printf("Age: %d\n", age);
    }
    void print(){
        printName();
        printAge();
    }
}

Тут методі print викликаються два інших методу класу - printName і printAge. Консольне виведення програми:


Name: Tom
Age: 41
