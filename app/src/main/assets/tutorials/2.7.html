Методи

Останнє оновлення: 16.09.2025

Если переменные класса хранят состояние об'єкта, то методы представляють его поведение. Методи містять собой набор операторов, которые выполняют определенные действия. И якщо нам треба определить в классе некоторые действия, нам треба определить методы.

Визначення методів

Загальне визначення методів виглядає так:


[модификаторы] тип_возвращаемого_значения название_метода ([параметры]){
	// тело метода
}

Модифікатори та параметри необов'язкові.

За замовчуванням головний клас будь-якої програми Java містить метод main, який служить точкою входу в програму:


class Program{

    public static void main(String[] args) {  

        System.out.println("Hello !");
    }
}

Ключові слова public и static є модификаторами. Далїї идет тип возвращаемого значения. Ключевое слово void указывает на то, что метод ничего не повертає.

Затем идут название метода - main и в скобках параметры метода - String[] args. И в фигурные скобки заключено тело метода - все действия, которые он выполняет.

Наприклад створимо клас і визначимо у ньому метод:

      
class Person{ 
    String name;
    int age;

    void print(){
        System.out.printf("Name: %s	\tAge: %d\n", name, age);
    }
}

class Program{

    public static void main(String[] args) {  

    }
}

Здесь определен класс Person с двумя полями - name и age и одним методом print. Иетод print в качестве возвращаемого типу имїїт тип void (то есть по сути ничего не повертає) и не приймає нияких параметров. Все что робить цей метод - це выводит на консоль значения переменных name и age.

Но чтобы применить метод, его треба вызвать.

Виклик методу

Виклик методу здійснюється у формі:


имя_метода(аргументы);

Після імені методу вказуються дужки, у яких перераховуються аргументи - значення параметрів методу.

Обращаю внимание, что в классе Program определен метод main, и нам не треба буво его нияк вызывать. Дело в том, что цей метод автоматически вызывается средой виконання Java, и нам не треба его явным чином вызывать. Остальные методы нам треба явным чином вызвать. И поскольку при запуске программы запускає метод main(), то все остальные треба вручную явным чином вызывать в методе main (или в других методах, которые буви також вызваны ранїї). Наприклад, вызовем выше определенный метод print:


class Program{

    public static void main(String[] args) {  

  		// создаем об'єкт Person
        Person tom = new Person();

  		// устанавливаем значения полей класса
        tom.name = "Tom";
        tom.age = 41;

  		// вызываем метод print
        tom.print();
    }
}

class Person{ 
    String name;
    int age;

    void print(){
        System.out.printf("Name: %s	\tAge: %d\n", name, age);
    }
}

Для вызова метода print в методе main використовується выражение tom.print(). То есть як и в случае с полями, после имени змінної класса идет крапка и затем имя метода. Поскольку метод print не имеtт нияких параметров, то после его названия при вызове ставятся пустые скобки. В итоге при выполнении программы мы увидим на консоли информацию о полях об'єкта:


Name: Tom		Age: 41

Одна з переваг методів полягає в тому, що ми можемоо винести деякі спільні дії окремий метод і потім викликати багаторазово їх у різних місцях програми. Наприклад:


class Program{

    public static void main(String[] args) {  

        Person tom = new Person();

        tom.name = "Tom";
        tom.age = 41;
        tom.print();

        tom.name = "Tomas";
        tom.print();

        tom.name = "Tommy";
        tom.print();
    }
}

class Person{ 
    String name;
    int age;

    void print(){
        System.out.printf("Name: %s	\tAge: %d\n", name, age);
    }
}

Тут ми три рази встановлюємо ім'я людини та за допомогою методу print перевіряємо його встановлення. У результаті отримаємо наступний консольний висновок:


Name: Tom		Age: 41
Name: Tomas		Age: 41
Name: Tommy		Age: 41

Виклик методів у інших методах класу

Одні методи класу можуть викликати інші методи:


class Program{

    public static void main(String[] args) {  

        Person tom = new Person();  // создаем об'єкт Person

        tom.name = "Tom";
        tom.age = 41;
        tom.print();
    }
}

// класс Person
class Person{ 
    String name;
    int age;

    void printName(){
        System.out.printf("Name: %s\n", name);
    }
    void printAge(){
        System.out.printf("Age: %d\n", age);
    }
    void print(){
        printName();
        printAge();
    }
}

Тут методі print викликаються два інших методу класу - printName і printAge. Консольне виведення програми:


Name: Tom
Age: 41
