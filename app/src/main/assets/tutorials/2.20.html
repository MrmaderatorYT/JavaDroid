Конструкція switch

Конструкция switch/case оценивает некоторое выражение и сравнивает його значення с набором значений. И при совпадении значений выполняет определенный код. Конструкция switch/case в некоторой степени аналогична конструкции if/else, оскільки дозволяє обработать відразу декілька условий.

В классическом виде конструкция switch имїїт следующїї формальное определение: 


switch (выражение)
{
    case значение1:
        код,выполняемый якщо выражение имїїт значение1
        break;
    case значение2:
        код,выполняемый якщо выражение имїїт значение2
        break;
	//.............
	case значениеN:
        код, выполняемый якщо выражение имїїт значениеN
        break;
    default:
        код, выполняемый якщо выражение не имїїт ни одно из выше указанных значений
        break;
}

После ключевого слова switch в скобках идет сравниваемое выражение. Значение цього выражения последовательно сравнивается со значениями, помщенными после оператора сase. И якщо совпадение буде найдено, то буде выполняться определенный блок сase.

В конце каждого блока сase повинен ставиться один из операторов перехода: break, return або throw. Как правило, використовується оператор break. При его применении другие блоки case выполняться не будуть.

Наприклад:


class Program {
  
    public static void main(String[] args) {
          
        String name = "Tom";

        switch (name){

            case "Bob":
                System.out.println("Ваше имя - Bob");
                break;
            case "Tom":
                System.out.println("Ваше имя - Tom");
                break;
            case "Sam":
                System.out.println("Ваше имя - Sam");
                break;
        }
    }
}

В данном случае конструкция switch последовательно сравнивает значение змінної name с набором значений, которые указаны после операторов case. Поскольку здесь значение змінної name - строка "Tom", то буде выполняться блок


case "Tom":
	System.out.println("Ваше имя - Tom");
	break;

Відповідно, ми побачимо на консолі

Ваше имя - Tom

Если бы мы не использовали бы в данном случае операторы break, як в следующем случае:


String name = "Tom";

switch (name){

    case "Bob":
        System.out.println("Ваше имя - Bob");
    case "Tom":
         System.out.println("Ваше имя - Tom");
    case "Sam":
        System.out.println("Ваше имя - Sam");
}

То після виконання блоку


case "Tom":
    System.out.println("Ваше имя - Tom");

Почав би виконуватися блок

case "Sam": System.out.println("Ваше имя - Sam");

І ми отримали такий консольний висновок:


Ваше имя - Tom
Ваше имя - Sam

default

Если значение змінної name не совпадает ни с яким значением после операторов case, то ни один из блоков case не виконується. Однак якщо даже в цем случае нам все равно треба выполнить якие-нибудь действия, то мы можемо добавить в конструкцию switch необязательный блок default. Например:


string name = "Alex";

switch (name)
{
    case "Bob":
        System.out.println("Ваше имя - Bob");
        break;
    case "Tom":
        System.out.println("Ваше имя - Tom");
        break;
    case "Sam":
        System.out.println("Ваше имя - Sam");
        break;
    default:
        System.out.println("Неизвестное имя");
        break;
}

В даному випадку жодне із значень після операторів case не збігається зі значенням змінної name, тому буде виконуватись блок default:


default:
	System.out.println("Неизвестное имя");
	break;

Нова форма конструкції switch

В классической версии конструкции switch каждый блок case, якщо мы хочемо закончить на нем выполнение, завершуєся оператором break. И обычно так и есть, редко когда требуется выполнять другие блоки case. И поцему в результате развития Java в язык бува добавлена новая форма конструкции switch:


switch (выражение)
{
    case значение1 ->
        код,выполняемый якщо выражение имїїт значение1
    case значение2 ->
        код,выполняемый якщо выражение имїїт значение2
	//.............
	case значениеN ->
        код, выполняемый якщо выражение имїїт значениеN
    default ->
        код, выполняемый якщо выражение не имїїт ни одно из выше указанных значений
}

В данной форме блок default також необязателен. Наприклад, перепишем первый пример из текущей статьи:


class Program {
  
    public static void main(String[] args) {
          
        String name = "Tom";

        switch (name){

            case "Bob" -> System.out.println("Ваше имя - Bob");
            case "Tom" -> System.out.println("Ваше имя - Tom");
            case "Sam" -> System.out.println("Ваше имя - Sam");
        }
    }
}

Если в блок case треба поместить декілька инструкций або болїї сложный код, то он оформляется в блок кода:


case "Tom" -> {
  System.out.println("Ваше имя - Tom");
  System.out.println("Привет, Tom");
}

Повернення значення з switch

Ключевое слово switch також застосовується для создания выражения switch, которое дозволяє возвращать некоторое значение - то есть отримати результат виконання конструкции switch. Для возвращения значения в блоках case может применятся оператор return. Наприклад, определим следующую программу:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операция
        int a = 10, b = 6;  // числа, над которыми виконується операция

        int result = 0;     // результат операции

        switch (operation)
        {
            case 1:
                result = a + b;
                break;
            case 2: 
                result = a - b;
                break;
            case 3: 
                result = a * b;
                break;
            default: 
                result = 0;
        }

        System.out.println(result);  // 16
    }
}

Тут у нас є номер операції - змінна операція, два числа для виконання операції - a і b і змінна результат для зберігання результату операції. У конструкції switch в залежності від номера операції привласнюємо змінної result відповідне значення.

За допомогою нової форми конструкції switch ми можемоо трохи спростити код:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операция
        int a = 10, b = 6;  // числа, над которыми виконується операция

        int result = 0;     // результат операции

        switch (operation)
        {
            case 1 -> result = a + b;
            case 2 -> result = a - b;
            case 3 -> result = a * b;
            default -> result = 0;
        }

        System.out.println(result);  // 16
    }
}

Тем не менїї ключевым моментом в данном случае, є то, что во всех случаях нам треба установить переменную result - передать ей нужный результат операции над числами. И выражения switch дозволяють упростить эту задачу:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операция
        int a = 10, b = 6;  // числа, над которыми виконується операция

        // результат операции
        int result = switch (operation)
        {
            case 1 -> a + b;
            case 2 -> a - b;
            case 3 -> a * b;
            default -> 0;
        };  // в конце обязательна крапка с комою

        System.out.println(result);  // 16
    }
}

В данном случае мы напрямую присваиваем змінної result то значение, которое вычисляет выражение switch. Вычисляемое значение ставится после стрелки в блоке case

case 1 -> a + b;

Завдяки цьому код став простішим і зрозумілішим.

Но, йде відзначити, что блоки case повинні охватывать все потенциальные значения. Наприклад, в данном случае случае мы получаем значение в переменную result - переменную типу int. Тип int охватывает 322 значений. Естественно в данном случае для каждого из них визначати свой блок case нерационально, тем болїї нам нужны лише три значения - для каждой операции. И в цем случае для всех остальных значений застосовується выражение default, которое в данном случае просто повертає 0.

Если же нам треба выполнить декілька инструкций в каждом блоке case, то инструкции оформляются в отдельный блок кода:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операция
        int a = 10, b = 6;  // числа, над которыми виконується операция

        // результат операции
        int result = switch (operation)
        {
            case 1 -> {
                System.out.println("a + b");
                yield a + b;
            }
            case 2 -> {
                System.out.println("a - b");
                yield a - b;
            }
            case 3 -> {
                System.out.println("a * b");
                yield a * b;
            }
            default -> 0;
        }; 

        System.out.println(result);  // 16
    }
}

При цем для возвращения значения из блока кода застосовується оператор yield, после которого ставится возвращаемое значение:

yield a - b;

Варто відзначити, что також существует другая форма выражения switch, где после операторов case ставится двоеточие:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операция
        int a = 10, b = 6;  // числа, над которыми виконується операция

        // результат операции
        int result = switch (operation)
        {
            case 1:
                System.out.println("a + b");
                yield a + b;
            case 2:
                System.out.println("a - b");
                yield a - b;
            case 3:
                System.out.println("a * b");
                yield a * b;
            default: yield 0;
        };

        System.out.println(result);  // 16
    }
}

Но в цем случае в конце блока case треба обязательно використовувати оператор yield для возвращения значения
