Конструкція switch

Конструкция switch/case оценивает некоторое выражение и сравнивает його значення с набором значений. И при совпадении значений виконує визначенийный код. Конструкция switch/case в некоторой степени аналогична конструкции if/else, оскільки дозволяє обпрацювати відразу декілька условий.

В класическом виде конструкция switch имїїт следующїї формальное визначенийие: 


switch (выражение)
{
    case значення1:
        код,выполняемый якщо выражение имїїт значення1
        break;
    case значення2:
        код,выполняемый якщо выражение имїїт значення2
        break;
	//.............
	case значенняN:
        код, выполняемый якщо выражение имїїт значенняN
        break;
    default:
        код, выполняемый якщо выражение не имїїт ни одно из выше указанных значений
        break;
}

Після ключевого слова switch в скобках идет сравниваемое выражение. Значення цього выражения післядовательно сравнивается со значеннями, помщенными після оператора сase. И якщо совпадение буде найдено, то буде выполняться визначенийный блок сase.

В конце каждого блока сase повинен ставиться один из операторов перехода: break, return або throw. Как правило, використовується оператор break. При его применении другие блоки case выполняться не будуть.

Наприклад:


class Program {
  
    public static void main(String[] args) {
          
        String name = "Tom";

        switch (name){

            case "Bob":
                System.out.println("Ваше ім'я - Bob");
                break;
            case "Tom":
                System.out.println("Ваше ім'я - Tom");
                break;
            case "Sam":
                System.out.println("Ваше ім'я - Sam");
                break;
        }
    }
}

У даному випадку конструкция switch післядовательно сравнивает значення змінної name с набором значений, которые указаны після операторов case. Поскольку здесь значення змінної name - рядок "Tom", то буде выполняться блок


case "Tom":
	System.out.println("Ваше ім'я - Tom");
	break;

Відповідно, ми побачимо на консолі

Ваше ім'я - Tom

Если бы мы не использовали бы у даному випадку операторы break, як в следующем случае:


String name = "Tom";

switch (name){

    case "Bob":
        System.out.println("Ваше ім'я - Bob");
    case "Tom":
         System.out.println("Ваше ім'я - Tom");
    case "Sam":
        System.out.println("Ваше ім'я - Sam");
}

То після виконання блоку


case "Tom":
    System.out.println("Ваше ім'я - Tom");

Почав би виконуватися блок

case "Sam": System.out.println("Ваше ім'я - Sam");

І ми отримали такий консольний висновок:


Ваше ім'я - Tom
Ваше ім'я - Sam

default

Если значення змінної name не збігається ни с яким значенням після операторов case, то ни один из блоков case не виконується. Однак якщо даже в цем случае нам все равно треба выполнить якие-нибудь действия, то мы можемо добавить в конструкцию switch необязательный блок default. Наприклад:


string name = "Alex";

switch (name)
{
    case "Bob":
        System.out.println("Ваше ім'я - Bob");
        break;
    case "Tom":
        System.out.println("Ваше ім'я - Tom");
        break;
    case "Sam":
        System.out.println("Ваше ім'я - Sam");
        break;
    default:
        System.out.println("Неизвестное ім'я");
        break;
}

В даному випадку жодне із значень після операторів case не збігається зі значенням змінної name, тому буде виконуватись блок default:


default:
	System.out.println("Неизвестное ім'я");
	break;

Нова форма конструкції switch

В класической версии конструкции switch каждый блок case, якщо мы хочемо закончить на нем виконання, завершуєся оператором break. И обычно так и есть, редко когда требуется выполнять другие блоки case. И поцему в результаті развития Java в язык бува добавлена новая форма конструкции switch:


switch (выражение)
{
    case значення1 ->
        код,выполняемый якщо выражение имїїт значення1
    case значення2 ->
        код,выполняемый якщо выражение имїїт значення2
	//.............
	case значенняN ->
        код, выполняемый якщо выражение имїїт значенняN
    default ->
        код, выполняемый якщо выражение не имїїт ни одно из выше указанных значений
}

В данной форме блок default також необязателен. Наприклад, перепишем первый пример из текущей статьи:


class Program {
  
    public static void main(String[] args) {
          
        String name = "Tom";

        switch (name){

            case "Bob" -> System.out.println("Ваше ім'я - Bob");
            case "Tom" -> System.out.println("Ваше ім'я - Tom");
            case "Sam" -> System.out.println("Ваше ім'я - Sam");
        }
    }
}

Если в блок case треба поместить декілька инструкций або болїї сложный код, то он оформляется в блок коду:


case "Tom" -> {
  System.out.println("Ваше ім'я - Tom");
  System.out.println("Привет, Tom");
}

Повернення значення з switch

Ключевое слово switch також застосовується для створенняя выражения switch, которое дозволяє возвращать некоторое значення - то есть отримати результат виконання конструкции switch. Для возвращения значення в блоках case может применятся оператор return. Наприклад, визначимо следующую програму:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операція
        int a = 10, b = 6;  // числа, над которыми виконується операція

        int result = 0;     // результат операції

        switch (operation)
        {
            case 1:
                result = a + b;
                break;
            case 2: 
                result = a - b;
                break;
            case 3: 
                result = a * b;
                break;
            default: 
                result = 0;
        }

        System.out.println(result);  // 16
    }
}

Тут у нас є номер операції - змінна операція, два числа для виконання операції - a і b і змінна результат для зберігання результату операції. У конструкції switch в залежності від номера операції привласнюємо змінної result відповідне значення.

За допомогою нової форми конструкції switch ми можемоо трохи спростити код:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операція
        int a = 10, b = 6;  // числа, над которыми виконується операція

        int result = 0;     // результат операції

        switch (operation)
        {
            case 1 -> result = a + b;
            case 2 -> result = a - b;
            case 3 -> result = a * b;
            default -> result = 0;
        }

        System.out.println(result);  // 16
    }
}

Тем не менїї ключевым моментом у даному випадку, є то, что во всех случаях нам треба установить змінну result - передать ей нужный результат операції над числами. И выражения switch дозволяють упростить эту завданьу:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операція
        int a = 10, b = 6;  // числа, над которыми виконується операція

        // результат операції
        int result = switch (operation)
        {
            case 1 -> a + b;
            case 2 -> a - b;
            case 3 -> a * b;
            default -> 0;
        };  // в конце обязательна крапка с комою

        System.out.println(result);  // 16
    }
}

У даному випадку мы напрямую присваиваем змінної result то значення, которое вычисляет выражение switch. Вычисляемое значення ставится після стрелки в блоке case

case 1 -> a + b;

Завдяки цьому код став простішим і зрозумілішим.

Но, йде відзначити, что блоки case повинні охватывать все потенциальные значення. Наприклад, у даному випадку случае мы отримуємо значення в змінну result - змінну типу int. Тип int охватывает 322 значений. Естественно у даному випадку для каждого из них визначати свой блок case нерационально, тем болїї нам нужны лише три значення - для каждой операції. И в цем случае для всех остальных значений застосовується выражение default, которое у даному випадку просто повертає 0.

Если же нам треба выполнить декілька инструкций в каждом блоке case, то инструкции оформляются в отдельный блок коду:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операція
        int a = 10, b = 6;  // числа, над которыми виконується операція

        // результат операції
        int result = switch (operation)
        {
            case 1 -> {
                System.out.println("a + b");
                yield a + b;
            }
            case 2 -> {
                System.out.println("a - b");
                yield a - b;
            }
            case 3 -> {
                System.out.println("a * b");
                yield a * b;
            }
            default -> 0;
        }; 

        System.out.println(result);  // 16
    }
}

При цем для возвращения значення из блока коду застосовується оператор yield, після которого ставится повертаємоое значення:

yield a - b;

Варто відзначити, что також існує другая форма выражения switch, где після операторов case ставится двоеточие:


class Program {
  
    public static void main(String[] args) {
          
        int operation = 1;  // выполняемая операція
        int a = 10, b = 6;  // числа, над которыми виконується операція

        // результат операції
        int result = switch (operation)
        {
            case 1:
                System.out.println("a + b");
                yield a + b;
            case 2:
                System.out.println("a - b");
                yield a - b;
            case 3:
                System.out.println("a * b");
                yield a * b;
            default: yield 0;
        };

        System.out.println(result);  // 16
    }
}
