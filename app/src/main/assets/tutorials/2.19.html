Літерали

Останнє оновлення: 16.09.2025

Літерали представляютьь незмінюємоые значення (иногда их ще называют константами). Літерали можна передавать переменным в качестве значення. Літерали бывают логическими, целочисленными, вщественными, символьными и строчными. И отдельный литерал представляє ключевое слово null.

Логічні літерали

Есть две логических константи - true (истина) и false (ложь). Наприклад, напишем следующую програму на Java:


class Program{
    public static void main(String[] args) {  

        System.out.println(true);     // true
        System.out.println(false);    // false
    }
}

Здесь два методу System.out.println() післядовательно виводять на консоль значення логических литералов true и false. В итоге мы получим следующий консольный вивід:


true
false

Цілочисельні літерали

Цілочисленні літерали представляютььь позитивні та негативні цілі числа, наприклад, 1, 2, 3, 4, -7, -109. Цілочисленні літерали можуть бути виражені в десятковій, шістнадцятковій та двійковій формі.

З цілими числами у десятковій формі все має бути зрозумілим, оскільки вони використовуються нами у повсякденному житті:


class Program{
    public static void main(String[] args) {  

        System.out.println(-11);
        System.out.println(5);
        System.out.println(505);
    }
}

Числа у двійковій формі передуються символами 0b, після яких йде набір із нулів та одиниць:


System.out.println(0b11);        // 3
System.out.println(0b1011);      // 11
System.out.println(0b100001);    // 33

Для запису числа у шістнадцятковій формі застосовуються символи 0x, після яких йде набір символів від 0 до 9 і від A до F, які власне становлять число:


System.out.println(0x0A);    // 10
System.out.println(0xFF);    // 255
System.out.println(0xA1);    // 161

Причому символ "X" та шістнадцяткові цифри можуть бути як у нижньому, так і у верхньому регістрі:


System.out.println(0XFF);    // 255
System.out.println(0Xff);    // 255
System.out.println(0xFF);    // 255
System.out.println(0xff);    // 255

Кроме целых чисел в десятичном, двоичном и шестнадцатеричном формате також можна використовувати целые числа в вісімковій системі - такие числа предваряются нулем 0


class Program{
    public static void main(String[] args) {  

        System.out.println(03);        // 3
        System.out.println(013);      // 11
        System.out.println(041);    // 33
    }
}

Також цілі числа підтримують піділ розрядів числа за допомогою символу підкреслення, що підвищує наочність:


System.out.println(123_456); 	// 123456
System.out.println(234_567__789); 	// 234567789

Речові літерали

Речові літерали є дробовими числами. Цей тип літералів має дві форми. Перша форма - речові числа з фіксованою комою, при якій дробову частину відокремлюється від цілої частини крапкою. Наприклад:


3.14
100.001
-0.38

Приклад використання:


class Program{
    public static void main(String[] args) {  

        System.out.println(3.14); 
        System.out.println(100.001); 
        System.out.println(-0.38); 
    }
}

Також речові літерали можуть визначатися в експоненційній формі MEp, де M - мантиса, E - експонента, яка фактично означає "* 10 ^" (помножити на десять ступеня), а p - Порядок. Наприклад:


System.out.println(3.2e3);	// по сути равно 3.2 * 103 = 3200
System.out.println(1.2E-1);	// равно 1.2 * 10-1 = 0.12

Символьні літерали

Символьные литералы представляютьь одиночные символи. Символи заключаются в одинарные кавычки.

Символьні літерали бувають кількох видів. Насамперед це звичайні символи:


'2'
'A'
'T'

Також мы можемо передать их вывести на консоль за допомогою System.out.println:


System.out.println('2');
System.out.println('A');
System.out.println('T');

Специальную группу представляютьь керуючі послідовності Управляющая післядовательность представляє символ, перед которым ставится слеш. И данная післядовательность интерпретируется визначенийным чином. Наиболїї часто використовуємоые післядовательности:

  • '\n' - перевод рядки

  • '\t' - табуляция

  • '\' - слеш

І якщо компілятор зустріне в тексті послідовність \t, то він сприйматиме цю послідовність не як сліш і букву t, а як табуляцію - тобто довгий відступ.

Також символи можуть визначатися у вигляді шістнадцяткових кодів, також укладений в одинарні лапки.

Ще одной формой визначенийия символьных литералов представляє застосування кодов из таблиці символів Unicode. Для цього в одинарных кавычках вказуються символи '\u', після которых идет шестнадцатеричный код Unicode. Наприклад, код '\u0411' представляє кириллический символ 'Б', а код '\u0066' буде хранить символ 'f'.


System.out.println('\u0420');    // Р
System.out.println('\u0421');    // С

Рядкові літерали

Рядкові літерали представляютььь рядки. Рядки полягають у підвійні лапки:


class Program{
    public static void main(String[] args) {  

        System.out.println("hello ");
        System.out.println("hello word");
        System.out.println("test");
    }
}

Якщо всередині рядка необхідно вивести підвійну лапку, то така внутрішня лапка випереджається зворотним слешем:


System.out.println("Hello \"\"");

Також у рядках можна використовувати послідовності, що управляють як \n, \t та інші. Наприклад, послідовність '\n' здійснює переведення на новий рядок:


System.out.println("Hello\n");

В даному випадку послідовність \n буде сигналом, що необхідно зробити переведення на наступний рядок. І при виведенні на консоль частину рядка "" буде перенесено на новий рядок:


Hello

Java також піддерживает тестові блоки (text blocks) - многострочный текст, облеченный в тройные кавычки. Рассмотрим, в чем их практическая польза. Наприклад, выведем большой многострочный текст:


class Program{
    public static void main(String[] args) {  

        System.out.println("Підт мысль, которой весь я предан,\n"+
                "Итог всего, что ум скопил.\n"+
                "Лишь тот, кем бой за жизнь изведан,\n"+
                "Жизнь и свободу заслужил.");
    }
}

За допомогою операції + ми можемоо приєднати до одного тексту інший, причому продовження може розташовуватися на наступному рядку. Щоб під час виведення тексту відбувалося перенесення наступного рядка, застосовується послідовність \n.

Результат виконання цього коду:


Підт мысль, которой весь я предан,
Итог всего, что ум скопил.
Лишь тот, кем бой за жизнь изведан,
Жизнь и свободу заслужил.

Текстові блоки дозволяють упростить написание многострочного текста:


class Program{
    public static void main(String[] args) {  

        System.out.println("""
                Підт мысль, которой весь я предан,
                Итог всего, что ум скопил.
                Лишь тот, кем бой за жизнь изведан,
                Жизнь и свободу заслужил.""");
    }
}

Весь текстовий блок обертається в потрійні лапки, при цьому не потрібно використовувати з'єднання рядків або послідовність для їх перенесення. Результат виконання програми буде тим самим, що й у прикладі вище.

null

null представляє ссылку, которая не вказує ни на якой об'єкт. То есть по сути отсутствие значення.
