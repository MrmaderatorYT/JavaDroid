Методи skip и limit

Останнє оновлення: 29.04.2018

Метод skip(long n) використовується для пропуска n елементов. Цет метод повертає новый потік, в котором пропущены первые n елементов.

Метод limit(long n) застосовується для выборки первых n елементов потіков. Цет метод також повертає модифицированный потік, в котором не болїї n елементов.

Зачастую эта пара методів використовується вместе для створенняя эффекта постраничной навигации. Рассмотрим, як их застосовувати:


Stream<String> phoneStream = Stream.of("iPhone 6 S", "Lumia 950", "Samsung Galaxy S 6", "LG G 4", "Nexus 7");
        
phoneStream.skip(1)
    .limit(2)
    .forEach(s->System.out.println(s));

У даному випадку метод skip пропускает один первый елемент, а метод limit выбирает два следующих елемента. В итоге мы получим следующий консольный вивід:

Lumia 950
Samsung Galaxy S 6

Вполне может быть, что метод skip может принимать в качестве параметра число большїї, чем кількість елементів у потоці. В цем случае будуть пропущены всі елементи, а в результирующем потіке буде 0 елементов.

И якщо в метод limit передаєся число, большїї, чем кількість елементів, то просто выбираются всі елементи потіка.

Теперь розглянемо, як створити постраничную навигацию:


import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;
import java.util.stream.*;
import java.util.Scanner;

public class Program {

    public static void main(String[] args) {
        
        List<String> phones = new ArrayList<String>();
        phones.addAll(Arrays.asList(new String[]
                {"iPhone 6 S", "Lumia 950", "Huawei Nexus 6P",
                "Samsung Galaxy S 6", "LG G 4", "Xiaomi MI 5",
                "ASUS Zenfone 2", "Sony Xperia Z5", "Meizu Pro 5",
                "Lenovo S 850"}));
        
        int pageSize = 3; // кількість елементів на сторінку
        Scanner scanner = new Scanner(System.in);
        while(true){
           
            System.out.println("Введите номер страницы: ");
            int page = scanner.nextInt();
           
            if(page<1) break; // якщо число менше 1, виходимо з циклу
            
            phones.stream().skip((page-1) * pageSize)
                .limit(pageSize)
                .forEach(s->System.out.println(s));
       }
    }
}

У даному випадку у нас набор из 10 елементов. За допомогою змінної pageSize определяем кількість елементів на странице - 3. То есть у нас получится 4 страницы (на післядней буде лише один елемент).

В бесконечном цикле отримуємо номер страницы и выбираем лише те елементы, которые находятся на указанной странице.

Теперь введем якие-нибудь номера страниц, наприклад, 4 и 2:

Введите номер страницы: 
4
Lenovo S 850
Введите номер страницы: 
2
Samsung Galaxy S 6
LG G 4
Xiaomi MI 5
