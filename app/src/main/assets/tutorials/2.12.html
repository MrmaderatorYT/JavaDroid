Типи даних

Останнє оновлення: 16.09.2025

Однією з основних особливостей Java є те, що ця мова є строго типізованою. А це означає, що кожна змінна та константа представляє певний тип і цей тип суворо визначений. Тип даних визначає діапазон значень, які може зберігати змінна або константа.

Отже, розглянемо систему вбудованих базових типів даних, яка використовується для створення змінних Java. А вона представлена ​​такими типуми.

  • boolean: хранит значение true або false

    
boolean isActive = false;
boolean isAlive = true;

  • byte: хранит целое число от -128 до 127 и занимает 1 байт

    
byte a = 3;
byte b = 8;

  • short: хранит целое число от -32768 до 32767 и занимает 2 байта

    
short a = 3;
short b = 8;

  • int: хранит целое число от -2147483648 до 2147483647 и занимает 4 байта

    
int a = 4;
int b = 9;

  • long: хранит целое число от –9 223 372 036 854 775 808 до 9 223 372 036 854 775 807 и занимает 8 байт

    
long a = 5;
long b = 10;

  • double: хранит число с плавающей точкой от ±4.9*10-324 до ±1.7976931348623157*10308 и занимает 8 байт

    
double x = 8.5;
double y = 2.7;

    Як роздільник цілої та дробової частини в дробових літералах використовується крапка.

  • float: хранит число с плавающей точкой от -3.4*1038 до 3.4*1038 и занимает 4 байта

    
float x = 8.5F;
float y = 2.7F;

  • char: хранит одиночный символ в кодировке UTF-16 и занимает 2 байта, поцему диапазон хранимых значений от 0 до 65535

При цьому змінна може набувати лише тих значень, які відповідають її типу. Якщо змінна представляє цілий тип, то вона не може зберігати дробові числа.

Цілі числа

Все целочисленные литералы, наприклад, числа 10, 4, -5, воспринимаются як значения типу int, однак мы можемо присваивать целочисленные литералы другим целочисленным типум: byte, long, short. В цем случае Java автоматически осуществляет соответствующие преобразования:


byte a = 1;
short b = 2;
long c = 2121;

Однак якщо ми захочемо присвоїти змінній типу long двже велике число, яке виходить за межі допустимих значень для типу int, то ми зіткнемося з помилкою під час компіляції:

long num = 2147483649;

Здесь число 2147483649 є допустимым для типу long, но выходит за предельные значения для типу int. И оскільки все целочисленные значения по умолчанию расцениваются як значения типу int, то компилятор укажет нам на ошибку. Щоб решить проблему, треба добавить к числу суффикс l або L, який указывает, что число представляє тип long:

long num = 2147483649L;

Як правило, значення для цілих змінних задаються в десятковій системі числення, однак ми можемоо застосовувати й інші системи числення. Наприклад:


int num111 = 0x6F; // 16-теричная система, число 111
int num8 = 010; // 8-ричная система, число 8
int num13 = 0b1101; // 2-ичная система, число 13

Для задания шестнадцатеричного значения после символов 0x указывается число в шестнадцатеричном формате. Таким же чином восьмеричное значение указывается после символа 0, а двоичное значение - после символов 0b.

Числа з плаваючою точкою

При присвоении змінної типу float дробного литерала с плавающей точкой, наприклад, 3.1, 4.5 и т.д., Java автоматически рассматривает цей литерал як значение типу double. И чтобы указать, что данное значение повинно рассматриваться як float, нам треба використовувати суффикс f:


float fl = 30.6f;
double db = 30.6;

І хоча в даному випадку обидві змінні мають практично одне значення, але ці значення по-різному розглядатимуть і займатимуть різне місце в пам'яті.

Символи та рядки

В качестве значения переменная символьного типу отримує одиночный символ, заключенный в одинарные кавычки: char ch='e';. Крім того, змінної символьного типу також можна присвоить целочисленное значение от 0 до 65535. В цем случае переменная опять же буде хранить символ, а целочисленное значение буде указывать на номер символа в таблице символов Unicode (UTF-16). Например:


char ch=102; // символ 'f'
System.out.println(ch);

Символьные переменные не варто путать со строковыми, 'a' не идентично "a". Строковые переменные представляють об'єкт String, який в отличие от char або int не є примитивным типом в Java:


String hello = "Hello...";
System.out.println(hello);

Варто відзначити, что текстовые блоки в тройных кавычках також рассматриваются як дані типу String, которые удобно використовувати для установки многострочного текста:


String text = """
                Підт мысль, которой весь я предан,
                Итог всего, что ум скопил.
                Лишь тот, кем бой за жизнь изведан,
                Жизнь и свободу заслужил.
                """;
System.out.println(text);
