Оператор return. Результат методу

Останнє оновлення: 16.09.2025

Методи можуть возвращать некоторое значение. В примерах в предыдущих статьях применялись методы, которые имели тип void: методы с таким типом не повертають ниякого значения. Они просто выполняют некоторые действия. Для возвращения некоторого результата в методе застосовується оператор return:

return возвращаемое_значение;

После оператора return указывается возвращаемое значение, которое є результатом метода. Це может быть литеральное значение, значение змінної або якого-то сложного выражения.

Наприклад, определим метод, який повертає значення типу String:


class Person{ 

    String sayHello(){
        return "Hello";
    }
}

Метод sayHello имїїт тип String, следовательно, он повинен возвратить строку. Тому в теле метода використовується оператор return, после которого указана возвращаемая строка.

При цем методы, которые в качестве возвращаемого типу имеют любой тип, кроме void, обязательно повинні використовувати оператор return для возвращения значения. Наприклад, следующїї определение метода некорректно:


String sayHello(){

	System.out.println("Hello");
}

Також между возвращаемым типом метода и возвращаемым значением после оператора return повинно быть соответствие. Наприклад, в следующем случае возвращаемый тип - String, но метод повертає число (тип int), поцему такое определение метода некорректно:


String sayHello(){

	return 3;	// Ошибка! Метод повинен возвращать строку, а не число
}

Результати методів, які повертають значення, ми можемоо присвоїти змінним або використовувати іншим чином у програмі:


class Program{

    public static void main(String[] args) {  

        Person tom = new Person();
        // в метод System.out.println в качестве параметра передаєся результат метода tom.sayHello()
        System.out.println(tom.sayHello());  // Hello

        // присваиваем результат метода tom.sayHello() змінної
        String message = tom.sayHello();
        System.out.println(message);  // Hello
    }
}
class Person{ 

    String sayHello(){
        return "Hello";
    }
}

Метод sayHello() повертає значение типу String. Тому мы можемо присвоить це значение якой-нибудь змінної типу String:

String message = sayHello();

Або навіть передати значення параметру іншого методу:


System.out.println(tom.sayHello());

Після оператора return можна також вказувати складні вирази або виклики інших методів, які повертають певний результат. Наприклад, визначимо метод, який повертає суму чисел:


class Program{

    public static void main(String[] args) {  

        Calcualtor calc = new Calcualtor();
        int result = calc.sum(10, 12);  // 22
        System.out.println(result);  // 22

        System.out.println(calc.sum(10, 15));  // 25
    }
}
class Calcualtor{

    int sum(int a, int b){
		
        return a + b;
	}
}

Метод может використовувати декілька вызовов оператора return для возвращения разных значений в зависимости от некоторых условий:


class Program{

    public static void main(String[] args) {  

        Time time = new Time();
        System.out.println(time.asString(7));     // Good morning
        System.out.println(time.asString(13));    // Good after noon
        System.out.println(time.asString(18));    // Good evening
        System.out.println(time.asString(2));     // Good night
    }
}
class Time{
    // метод отримує текущий час
    String asString(int hour){
         
        // якщо значение некорретное
        if (hour >24 || hour < 0)
            return "Invalid data";
        // якщо от 21 часов вечера до 6 часов утра
        else if(hour > 21 || hour < 6)
            return "Good night";
        // якщо от 15 часов дня
        else if(hour >= 15)
            return "Good evening";
        // якщо от 11 часов дня
        else if(hour >= 11)
            return "Good after noon";
        // остальные случае - меньше 11часов дня
        else
            return "Good morning";
    }
}

Тут метод asString класу Time приймає поточну годину через параметр hour і повертає значення типу String, тобто рядок, тобто фактично перетворює годину на рядок. І в залежності від значення параметра hour рядок, що повертається, буде різнитися.

Вихід із методу

Оператор return застосовується для возвращаения значения из метода, но и для выхода из метода. sВ подобном качестве оператор return застосовується в методах, которые ничего не повертають, то есть имеют тип void:


class Program{

    public static void main(String[] args) {  

        Time time = new Time();
        time.print(7);     // Good morning
        time.print(13);    // Good after noon
        time.print(32);    // 
        time.print(56);    // 
        time.print(2);     // Good night
    }
}
class Time{
    // метод отримує текущий час
    void print(int hour){
         
        // якщо значение некорретное
        if (hour >24 || hour < 0)
            return;
        // якщо от 21 часов вечера до 6 часов утра
        else if(hour > 21 || hour < 6)
            System.out.println("Good night");
        // якщо от 15 часов дня
        else if(hour >= 15)
            System.out.println("Good evening");
        // якщо от 11 часов дня
        else if(hour >= 11)
            System.out.println("Good after noon");
        // остальные случае - меньше 11часов дня
        else
            System.out.println("Good morning");
    }
}

Якщо передане метод print значення більше 24 чи менше 0, то просто виходимо з методу. Значення, що повертається після return, вказувати в цьому випадку не потрібно.
