Умовні вирази

Останнє оновлення: 16.04.2018

Умовні вирази представляють собой некоторое условие и повертають значение типу boolean, то есть значение true (якщо условие истинно), або значение false (якщо условие ложно). К условным выражениям относятся операции сравнения и логические операции.

Операції порівняння

В операциях сравнения сравниваются два операнда, и повертаєся значение типу boolean - true, якщо выражение верно, и false, якщо выражение неверно.

  • ==

    сравнивает два операнда на равенство и повертає true (якщо операнды равны) и false (якщо операнды не равны)

    
int a = 10;
int b = 4;
boolean c = a == b;    		// false
boolean d = a == 10;       // true

  • !=

    сравнивает два операнда и повертає true, якщо операнды НЕ равны, и false, якщо операнды равны

    
int a = 10;
int b = 4;
boolean c = a != b;    		// true
boolean d = a != 10;       // false

  • < (меньше чем)

    Повертає true, якщо первый операнд меньше второго, иначе повертає false

    
int a = 10;
int b = 4;
boolean c = a < b; 	// false

  • > (больше чем)

    Повертає true, якщо первый операнд больше второго, иначе повертає false

    
int a = 10;
int b = 4;
boolean c = a > b; 	// true

  • >= (больше або равно)

    Повертає true, якщо первый операнд больше второго або равен второму, иначе повертає false

    
boolean c = 10 >= 10; 	// true
boolean b = 10 >= 4; 	// true
boolean d = 10 >= 20; 	// false

  • <= (меньше або равно)

    Повертає true, якщо первый операнд меньше второго або равен второму, иначе повертає false

    
boolean c = 10 <= 10; 	// true
boolean b = 10 <= 4; 	// false
boolean d = 10 <= 20; 	// true

Логічні операції

Також у Java є логічні операції, які також представляютьь умову та повертають true або false та зазвичай об'єднують кілька операцій порівняння. До логічних операцій відносять такі:

  • |

    c=a|b; (c равно true, якщо або a, або b (або и a, и b) равны true, иначе c буде равно false)

  • &

    c=a&b; (c равно true, якщо и a, и b равны true, иначе c буде равно false)

  • !

    c=!b; (c равно true, якщо b равно false, иначе c буде равно false)

  • ^

    c=a^b; (c равно true, якщо або a, або b (но не одновременно) равны true, иначе c буде равно false)

  • ||

    c=a||b; (c равно true, якщо або a, або b (або и a, и b) равны true, иначе c буде равно false)

  • &&

    c=a&&b; (c равно true, якщо и a, и b равны true, иначе c буде равно false)

Здесь у нас две пары операций | и || (а також & и &&) выполняют похожие действия, однак же они не равнозначны.

Выражение c=a|b; буде вычислять сначала оба значения - a и b и на их основе выводить результат.

В выражении же c=a||b; вначале буде вычисляться значение a, и якщо оно равно true, то вычисление значения b вже смысла не имїїт, оскільки у нас в любом случае вже c буде равно true. Значение b буде вычисляться лише в том случае, якщо a равно false

То же самое касается пары операций &/&&. В выражении c=a&b; будуть вычисляться оба значения - a и b.

В выражении же c=a&&b; сначала буде вычисляться значение a, и якщо оно равно false, то вычисление значения b вже не имїїт смысла, оскільки значение c в любом случае равно false. Значение b буде вычисляться лише в том случае, якщо a равно true

Таким чином, операции || и && болїї удобны в вычислениях, позволяя сократить час на вычисление значения выражения и тем самым повышая производительность. А операции | и & больше подходят для виконання поразрядных операций над числами.

Приклади:


boolean a1 = (5 > 6) || (4 < 6); // 5 > 6 - false, 4 < 6 - true, поцему повертаєся true
boolean a2 = (5 > 6) || (4 > 6); // 5 > 6 - false, 4 > 6 - false, поцему повертаєся false
boolean a3 = (5 > 6) && (4 < 6); // 5 > 6 - false, поцему повертаєся false (4 < 6 - true, но не вычисляется)
boolean a4 = (50 > 6) && (4 / 2 < 3); // 50 > 6 - true, 4/2 < 3 - true, поцему повертаєся true
boolean a5 = (5 < 6) ^ (4 > 6); // 5 < 6 - true, поцему повертаєся true (4 > 6 - false)
boolean a6 = (50 > 6) ^ (4 / 2 < 3); // 50 > 6 - true, 4/2 < 3 - true, поцему повертаєся false
