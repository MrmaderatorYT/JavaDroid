Основні операції з рядками

Останнє оновлення: 16.09.2025

З'єднання рядків

Для з'єднання рядків можна використовувати операцію сложения ("+"):


String str1 = "Java";
String str2 = "Hello";
String str3 = str1 + " " + str2;
        
System.out.println(str3); // Hello Java

При цем якщо в операції сложения рядків використовується нерядківовый об'єкт, наприклад, число, то цей об'єкт преобразуется к рядківе:

String str3 = "Год " + 2015;

Фактически же при сложении рядків с нерядківовыми об'єктами буде вызываться метод valueOf() класа String. Данный метод имїїт множина перегрузок и преобразует практически все типи данных к рядківе. Для преобразования об'єктов различных класов метод valueOf вызывает метод toString() цих класов.

Другой способ об'єднания рядків представляє метод concat():


String str1 = "Java";
String str2 = "Hello";
str2 = str2.concat(str1); // HelloJava

Метод concat() приймає рядок, с которой треба об'єднати вызывающую рядок, и повертає соединенную рядок.

Ще один метод об'єднания - метод join() дозволяє об'єднати рядки с учетом розділителя. Наприклад, выше две рядки сливались в одно слово "HelloJava", но в идеале мы бы хотели, чтобы две підрядки буви розділены пробелом. И для цього використовуємо метод join():


String str1 = "Java";
String str2 = "Hello";
String str3 = String.join(" ", str2, str1); // Hello Java

Метод join є статическим. Первым параметром идет розділитель, которым будуть розділяться підрядки в общей рядківе, а все післядующие параметри передають через запятую произвольный набор об'єднуваних підрядків - у даному випадку две рядки, хотя их может быть и больше

Проверка на пустоту

Метод isEmpty() дозволяє проверить, є ли рядок порожній. Пустая рядок не имїїт нияких символів, однак она визначенийа и не равна null:


String str1 = "Hello";
System.out.println(str1.isEmpty());  // false
String str2 = "  \t\n";
System.out.println(str2.isEmpty());  // false
String str3 = "";
System.out.println(str3.isEmpty());  // true

Из коду выше мы можемо увидеть, что даже якщо рядок містить пробельные символи, табуляцию, переводы рядки, то она все равно считаєся НЕ порожній. Метод isBlank() також дозволяє проверить, є ли рядок порожній, но теперь порожній считаєся рядок, которая містить також одни пробелы, табуляцию, переводы рядки:


String str1 = "Hello";
System.out.println(str1.isBlank());  // false
String str2 = "  \t\n";
System.out.println(str2.isBlank());  // true
String str3 = "";
System.out.println(str3.isBlank());  // true

Извлечение символів и підрядків

Для извлечения символів по индексу в класе String визначений метод char charAt(int index). Он приймає индекс, по которому треба отримати символів, и повертає извлеченный символ:


String str = "Java";
char c = str.charAt(2);
System.out.println(c); // v

Как и в масивах индексация начинается с нуля.

Если треба извлечь відразу группу символів або підрядок, то можна використовувати метод getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin). Он приймає следующие параметри:

  • srcBegin: индекс в рядківе, с которого начинается извлечение символів

  • srcEnd: индекс в рядківе, до которого идет извлечение символів

  • dst: масив символів, в який будуть извлекаться символи

  • dstBegin: индекс в масиве dst, с которого треба добавлять извлеченные из рядки символи


String str = "Hello world!";
int start = 6;
int end = 11;
char[] dst=new char[end - start];
str.getChars(start, end, dst, 0);
System.out.println(dst); // world

Сравнение рядків

Для сравнения рядків використовуються методи equals() (с учетом регистра) и equalsIgnoreCase() (без учета регистра). Оба методу в качестве параметра приймають рядок, с которой треба сравнить:


String str1 = "Hello";
String str2 = "hello";
        
System.out.println(str1.equals(str2)); // false
System.out.println(str1.equalsIgnoreCase(str2)); // true

В отличие от сравнения числовых и других данных примитивных типів для рядків не застосовується знак равенства ==. Вместо него треба використовувати метод equals().

Ще один специальный метод regionMatches() сравнивает отдельные підрядки в рамках двух рядків. Он имїїт следующие формы:


boolean regionMatches(int toffset, String other, int oofset, int len)
boolean regionMatches(boolean ignoreCase, int toffset, String other, int oofset, int len)

Метод приймає следующие параметри:

  • ignoreCase: треба ли игнорировать регистр символів при сравнении. Если значення true, регистр игнорируется

  • toffset: начальный индекс в вызывающей рядківе, с которого начнется сравнение

  • other: рядок, с которой сравнивается вызывающая

  • oofset: начальный индекс в сравниваемой рядківе, с которого начнется сравнение

  • len: кількість сравниваемых символів в обеих рядокх

Використовуємо метод:


String str1 = "Hello world";
String str2 = "I work";
boolean result = str1.regionMatches(6, str2, 2, 3);
System.out.println(result); // true

У даному випадку метод сравнивает 3 символа с 6-го индекса первой рядки ("wor") и 3 символа со 2-го индекса второй рядки ("wor"). Так як эти підрядки одинаковы, то повертаєся true.

И ще одна пара методів int compareTo(String str) и int compareToIgnoreCase(String str) також дозволяють сравнить две рядки, но при цем они також дозволяють узнать больше ли одна рядок, чем другая або нет. Если повертаємоое значення больше 0, то первая рядок больше второй, якщо меньше нуля, то, наоборот, вторая больше первой. Если рядки равны, то повертаєся 0.

Для визначенийия больше або меньше одна рядок, чем другая, використовується лексикографический порядок. То есть, наприклад, рядок "A" меньше, чем рядок "B", оскільки символ 'A' в алфавите варто перед символом 'B'. Если первые символи рядків равны, то в расчет берутся следующие символи. Наприклад:


String str1 = "hello";
String str2 = "world";
String str3 = "hell";
        
System.out.println(str1.compareTo(str2)); // -15 - str1 меньше чем strt2
System.out.println(str1.compareTo(str3)); // 1 - str1 больше чем str3

Поиск в рядківе

Метод indexOf() находит индекс первого вхождения підрядки в рядок, а метод lastIndexOf() - индекс післяднего вхождения. Если підрядок не буде найдена, то оба методу повертають -1:


String str = "Hello world";
int index1 = str.indexOf('l'); // 2
int index2 = str.indexOf("wo"); //6
int index3 = str.lastIndexOf('l'); //9

Метод startsWith() дозволяють визначити начинается ли рядок с визначенийной підрядки, а метод endsWith() дозволяє визначити заканчивается рядок на визначенийную підрядок:


String str = "myfile.exe";
boolean start = str.startsWith("my"); //true
boolean end = str.endsWith("exe"); //true

Замена в рядківе

Метод replace() дозволяє заменить в рядківе одну післядовательность символів на другую:


String str = "Hello world";
String replStr1 = str.replace('l', 'd'); // Heddo wordd
String replStr2 = str.replace("Hello", "Bye"); // Bye world

Видалення начальных и конечных пробелов

Метод strip() дозволяє удалить начальные и конечные пробелы:


String str = "   Hello   ";
System.out.println(str.length());  // 22
str = str.strip(); 	// видаляємо пробелы
System.out.println(str.length());  // 17

Варто відзначити, что ранїї - в болїї ранних версиях Java для цей цели служил метод trim(), однак он не дуже корректно работает с некоторыми символами Unicode, поцему вместо него рекомендуется strip()

Дополнительно метод stripLeading() дозволяє удалить лише начальные, а метод stripTrailing() - лише конечные пробелы:


String str = "   Hello   ";
System.out.println(str.length());  // 22
str = str.stripLeading();   // видаляємо начальные пробелы
System.out.println(str.length());  // 19
str = str.stripTrailing();   // видаляємо конечные пробелы
System.out.println(str.length());  // 17

Отримання підрядки

Метод substring() повертає підрядок, начиная с визначенийного индекса до конца або до визначенийного индекса:


String str = "Hello world";
String substr1 = str.substring(6); // world
String substr2 = str.substring(3,5); //lo

Зміна регистра

Метод toLowerCase() переводит все символи рядки в нижний регистр, а метод toUpperCase() - в верхний:


String str = "Hello World"; 
System.out.println(str.toLowerCase()); // hello world
System.out.println(str.toUpperCase()); // HELLO WORLD

Split

Метод split() дозволяє разбить рядок на підрядки по визначенийному розділителю. Разделитель - якой-нибудь символ або набор символів передаєся в качестве параметра в метод. Наприклад, разобьем текст на отдельные слова:


String text = "FIFA will never regret it";
String[] words = text.split(" ");
for(String word : words){
	System.out.println(word);
}

У даному випадку рядок буде розділяться по пробелу. Консольный вивід:

will
never
regret
it
