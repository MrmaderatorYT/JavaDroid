Основні операції з рядками

Останнє оновлення: 16.09.2025

Соединение строк

Для соединения строк можна використовувати операцию сложения ("+"):


String str1 = "Java";
String str2 = "Hello";
String str3 = str1 + " " + str2;
        
System.out.println(str3); // Hello Java

При цем якщо в операции сложения строк використовується нестроковый об'єкт, наприклад, число, то цей об'єкт преобразуется к строке:

String str3 = "Год " + 2015;

Фактически же при сложении строк с нестроковыми об'єктами буде вызываться метод valueOf() класса String. Данный метод имїїт множество перегрузок и преобразует практически все типы данных к строке. Для преобразования об'єктов различных классов метод valueOf вызывает метод toString() цих классов.

Другой способ об'єднания строк представляє метод concat():


String str1 = "Java";
String str2 = "Hello";
str2 = str2.concat(str1); // HelloJava

Метод concat() приймає строку, с которой треба об'єднати вызывающую строку, и повертає соединенную строку.

Ще один метод об'єднания - метод join() дозволяє об'єднати строки с учетом разделителя. Наприклад, выше две строки сливались в одно слово "HelloJava", но в идеале мы бы хотели, чтобы две подстроки буви разделены пробелом. И для цього используем метод join():


String str1 = "Java";
String str2 = "Hello";
String str3 = String.join(" ", str2, str1); // Hello Java

Метод join є статическим. Первым параметром идет разделитель, которым будуть разделяться подстроки в общей строке, а все последующие параметры передают через запятую произвольный набор об'єднуваних подстрок - в данном случае две строки, хотя их может быть и больше

Проверка на пустоту

Метод isEmpty() дозволяє проверить, є ли строка пустой. Пустая строка не имїїт нияких символов, однак она определена и не равна null:


String str1 = "Hello";
System.out.println(str1.isEmpty());  // false
String str2 = "  \t\n";
System.out.println(str2.isEmpty());  // false
String str3 = "";
System.out.println(str3.isEmpty());  // true

Из кода выше мы можемо увидеть, что даже якщо строка містить пробельные символы, табуляцию, переводы строки, то она все равно считаєся НЕ пустой. Метод isBlank() також дозволяє проверить, є ли строка пустой, но теперь пустой считаєся строка, которая містить також одни пробелы, табуляцию, переводы строки:


String str1 = "Hello";
System.out.println(str1.isBlank());  // false
String str2 = "  \t\n";
System.out.println(str2.isBlank());  // true
String str3 = "";
System.out.println(str3.isBlank());  // true

Извлечение символов и подстрок

Для извлечения символов по индексу в классе String определен метод char charAt(int index). Он приймає индекс, по которому треба отримати символов, и повертає извлеченный символ:


String str = "Java";
char c = str.charAt(2);
System.out.println(c); // v

Как и в массивах индексация начинается с нуля.

Если треба извлечь відразу группу символов або подстроку, то можна використовувати метод getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin). Он приймає следующие параметры:

  • srcBegin: индекс в строке, с которого начинается извлечение символов

  • srcEnd: индекс в строке, до которого идет извлечение символов

  • dst: массив символов, в який будуть извлекаться символы

  • dstBegin: индекс в массиве dst, с которого треба добавлять извлеченные из строки символы


String str = "Hello world!";
int start = 6;
int end = 11;
char[] dst=new char[end - start];
str.getChars(start, end, dst, 0);
System.out.println(dst); // world

Сравнение строк

Для сравнения строк використовуються методы equals() (с учетом регистра) и equalsIgnoreCase() (без учета регистра). Оба метода в качестве параметра принимают строку, с которой треба сравнить:


String str1 = "Hello";
String str2 = "hello";
        
System.out.println(str1.equals(str2)); // false
System.out.println(str1.equalsIgnoreCase(str2)); // true

В отличие от сравнения числовых и других данных примитивных типов для строк не застосовується знак равенства ==. Вместо него треба використовувати метод equals().

Ще один специальный метод regionMatches() сравнивает отдельные подстроки в рамках двух строк. Он имїїт следующие формы:


boolean regionMatches(int toffset, String other, int oofset, int len)
boolean regionMatches(boolean ignoreCase, int toffset, String other, int oofset, int len)

Метод приймає следующие параметры:

  • ignoreCase: треба ли игнорировать регистр символов при сравнении. Если значение true, регистр игнорируется

  • toffset: начальный индекс в вызывающей строке, с которого начнется сравнение

  • other: строка, с которой сравнивается вызывающая

  • oofset: начальный индекс в сравниваемой строке, с которого начнется сравнение

  • len: количество сравниваемых символов в обеих строках

Используем метод:


String str1 = "Hello world";
String str2 = "I work";
boolean result = str1.regionMatches(6, str2, 2, 3);
System.out.println(result); // true

В данном случае метод сравнивает 3 символа с 6-го индекса первой строки ("wor") и 3 символа со 2-го индекса второй строки ("wor"). Так як эти подстроки одинаковы, то повертаєся true.

И ще одна пара методів int compareTo(String str) и int compareToIgnoreCase(String str) також дозволяють сравнить две строки, но при цем они також дозволяють узнать больше ли одна строка, чем другая або нет. Если возвращаемое значение больше 0, то первая строка больше второй, якщо меньше нуля, то, наоборот, вторая больше первой. Если строки равны, то повертаєся 0.

Для определения больше або меньше одна строка, чем другая, використовується лексикографический порядок. То есть, наприклад, строка "A" меньше, чем строка "B", оскільки символ 'A' в алфавите варто перед символом 'B'. Если первые символы строк равны, то в расчет берутся следующие символы. Например:


String str1 = "hello";
String str2 = "world";
String str3 = "hell";
        
System.out.println(str1.compareTo(str2)); // -15 - str1 меньше чем strt2
System.out.println(str1.compareTo(str3)); // 1 - str1 больше чем str3

Поиск в строке

Метод indexOf() находит индекс первого вхождения подстроки в строку, а метод lastIndexOf() - индекс последнего вхождения. Если подстрока не буде найдена, то оба метода повертають -1:


String str = "Hello world";
int index1 = str.indexOf('l'); // 2
int index2 = str.indexOf("wo"); //6
int index3 = str.lastIndexOf('l'); //9

Метод startsWith() дозволяють определить начинается ли строка с определенной подстроки, а метод endsWith() дозволяє определить заканчивается строка на определенную подстроку:


String str = "myfile.exe";
boolean start = str.startsWith("my"); //true
boolean end = str.endsWith("exe"); //true

Замена в строке

Метод replace() дозволяє заменить в строке одну последовательность символов на другую:


String str = "Hello world";
String replStr1 = str.replace('l', 'd'); // Heddo wordd
String replStr2 = str.replace("Hello", "Bye"); // Bye world

Видалення начальных и конечных пробелов

Метод strip() дозволяє удалить начальные и конечные пробелы:


String str = "   Hello   ";
System.out.println(str.length());  // 22
str = str.strip(); 	// удаляем пробелы
System.out.println(str.length());  // 17

Варто відзначити, что ранїї - в болїї ранних версиях Java для цей цели служил метод trim(), однак он не очень корректно работает с некоторыми символами Unicode, поцему вместо него рекомендуется strip()

Дополнительно метод stripLeading() дозволяє удалить лише начальные, а метод stripTrailing() - лише конечные пробелы:


String str = "   Hello   ";
System.out.println(str.length());  // 22
str = str.stripLeading();   // удаляем начальные пробелы
System.out.println(str.length());  // 19
str = str.stripTrailing();   // удаляем конечные пробелы
System.out.println(str.length());  // 17

Получение подстроки

Метод substring() повертає подстроку, начиная с определенного индекса до конца або до определенного индекса:


String str = "Hello world";
String substr1 = str.substring(6); // world
String substr2 = str.substring(3,5); //lo

Изменение регистра

Метод toLowerCase() переводит все символы строки в нижний регистр, а метод toUpperCase() - в верхний:


String str = "Hello World"; 
System.out.println(str.toLowerCase()); // hello world
System.out.println(str.toUpperCase()); // HELLO WORLD

Split

Метод split() дозволяє разбить строку на подстроки по определенному разделителю. Разделитель - якой-нибудь символ або набор символов передаєся в качестве параметра в метод. Наприклад, разобьем текст на отдельные слова:


String text = "FIFA will never regret it";
String[] words = text.split(" ");
for(String word : words){
	System.out.println(word);
}

В данном случае строка буде разделяться по пробелу. Консольный вывод:


FIFA
will
never
regret
it
