Створення и підключение бібліотеки

Останнє оновлення: 21.09.2025

Если мы хочемо неякий функционал повторно використовувати в различных проєктах або распространять для использования другими разработчиками в их проєктах, то мы можемо оформить цей функционал в виде бібліотеки. В Java популярным форматом распространения бібліотеки є JAR-файл. Рассмотрим, як мы можемо створити такую бібліотеку и підключить її в свой проєкт.

Компиляция класов

Створимо простейшую бібліотеку с одним класом, який буде выполнять математические операції. Сначала визначимо на диске папку для проєкта. Наприклад, в моем случае це папка workspace/java/mathlib. В папке проєкта створимо каталог math, а в нем - файл Operation.java. То есть у нас буде клас Operation, який буде принадлежать пакету "math" (поцему всередині проєкта мы помещаем файл Operation.java в папку "math").

В файле Operation.java визначимо следующий код:

package math;

public class Operation {
    public static int add(int a, int b) {
        return a + b;
    }

    public static int subtract(int a, int b) {
        return a - b;
    }
}

Сначала потрібно скомпилировать цей файл в клас - файл с расширением .class. Для цього перейдем в консоли в корневой каталог проєкта (в моем случае workspace/java) и виконаємо следующую команду:


javac -d . math/Operation.java

Параметр -d . вказує, что скомпилированные файли с расширением .class повинні быть помщены в текущий каталог, сохраняя структуру пакетів.

Створення JAR-файлу

Теперь, когда у нас есть скомпилированные класы, можна упаковать их в JAR-файл. Для цього застосовується утилита jar:


jar -cvf mathlib.jar math/

Параметри утилиты jar:

  • -c: створює новый JAR-файл

  • -v: виводить підробную информацию о процесе (verbose)

  • -f: встановлює ім'я создаваемого файлу (mathlib.jar)

  • math/: каталог, який потрібно добавить в архив (містить скомпилированные класы)

В результаті виконання цей команди в каталогу появится файл mathlib.jar, який містить скомпилированные класы (клас Operaton). Полный вивід команди:


eugene@Eugene:/workspace/java/mathlib$ jar -cvf mathlib.jar math/
added manifest
adding: math/(in = 0) (out= 0)(stored 0%)
adding: math/Operation.class(in = 306) (out= 217)(deflated 29%)
adding: math/Operation.java(in = 187) (out= 101)(deflated 45%)
eugene@Eugene:/workspace/java/mathlib$

Підключение JAR-файлу к програме на Java

Теперь, когда у нас есть бібліотека mathlib.jar, створимо отдельный проєкт и використовуємо її. Для проєкта визначимо на диске якую-нибудь новую папку с файлом Program.java. Пусть в цем файле буде следующий код:


import math.Operation;   // підключаем клас из бібліотеки

class Program{

    public static void main(String[] args) {
        
        int sum = Operation.add(5, 3);
        System.out.println("Sum: " + sum); // Sum: 8
    }
}

Здесь за допомогою выражения импорта підключаем клас "math.Operation" (клас Operation располагается в пакеті "math") и затем використовуємо его в програме, вызывая один из его методів.

Також скопируем файл бібліотеки mathlib.jar в папку проєкта, чтобы не прописывать полный путь к библиотеке. То есть в папке проєкта у нас буде два файлу:

  • Program.java

  • mathlib.jar

Щоб скомпилировать файл Program.java, компилятору потрібно знать, где найти клас Operation. Для цього при компиляции можна використовувати опцию -cp:


javac -cp mathlib.jar Program.java

Параметр -cp mathlib.jar вказує, что компилятор повинен искать класы не лише в текущем каталогу, но и в файле mathlib.jar (jar-файл выступает в роли каталогу).

Після успешной компиляции можна запустить програму, також указав путь к JAR-файлу:


// для Linux
java -cp mathlib.jar:. Program

// для Windows
java -cp mathlib.jar;. Program

Для виконання програми здесь застосовується параметр -cp mathlib.jar:. (для Linux) або -cp mathlib.jar;. (для Windows), який вказує, что путь к класам повинен включать "mathlib.jar" и текущий каталог ("."). Це важливо, оскільки JVM повинна найти и Program.class (який располагается в текущем каталогу), и Operation.class (який располагается в архиве mathlib.jar).

И після запуску мы увидим результат работы програми:


eugene@Eugene:/workspace/java/hello$ java -cp mathlib.jar:. Program
Sum: 8
eugene@Eugene:/workspace/java/hello$

Використання в IDE (IntelliJ IDEA, Eclipse)

Большинство разработчиков використовують интегрированные среды разработки (IDE), где процес підключения JAR-файлів значительно проще.

Встановлення в IntelliJ IDEA:

  1. Откройте пункт меню File -> Project Structure...

  2. Перейдите в розділ Modules и выберите ваш модуль.

  3. Нажмите на вкладку Dependencies.

  4. Нажмите + и выберите JARs or directories....

  5. Найдите файл jar-бібліотеки и нажмите "OK".

Встановлення в Eclipse:

  1. Кликните правой кнопкой мыши по вашему проєкту в Package Explorer.

  2. Выберите Build Path -> Configure Build Path....

  3. Перейдите на вкладку Libraries.

  4. Нажмите Add External JARs....

  5. Найдите файл jar-бібліотеки и нажмите Open.

Після цих шагов IDE автоматически добавит JAR-файл в путь компиляции и запуску.
