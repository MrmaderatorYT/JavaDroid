Класы DataOutputStream и DataInputStream

Останнє оновлення: 26.04.2018

Класы DataOutputStream и DataInputStream дозволяють записывать и считывать дані примитивных типів.

Запис данных и DataOutputStream

Клас DataOutputStream представляє потік вивіда и призначений для записи данных примитивных типів, таких, як int, double и т.д. Для записи каждого из примитивных типів призначений свой метод:

  • writeBoolean(boolean v) : записує в потік булевое однобайтовое значення

  • writeByte(int v): записує в потік 1 байт, которые представлен в виде целочисленного значення

  • writeChar(int v): записує 2-байтовое значення char

  • writeDouble(double v): записує в потік 8-байтовое значення double

  • writeFloat(float v): записує в потік 4-байтовое значення float

  • writeInt(int v): записує в потік целочисленное значення int

  • writeLong(long v): записує в потік значення long

  • writeShort(int v): записує в потік значення short

  • writeUTF(String str): записує в потік рядок в кодировке UTF-8

Считывание данных и DataInputStream

Клас DataInputStream действует противоположным чином - он считывает из потіка дані примитивных типів. Соответственно для каждого примитивного типу визначений свой метод для считывания:

  • boolean readBoolean(): считывает из потіка булевое однобайтовое значення

  • byte readByte(): считывает из потіка 1 байт

  • char readChar(): считывает из потіка значення char

  • double readDouble(): считывает из потіка 8-байтовое значення double

  • float readFloat(): считывает из потіка 4-байтовое значення float

  • int readInt(): считывает из потіка целочисленное значення int

  • long readLong(): считывает из потіка значення long

  • short readShort(): считывает значення short

  • String readUTF(): считывает из потіка рядок в кодировке UTF-8

  • int skipBytes(int n): пропускает при чтении из потіка n байтов

Рассмотрим застосування класов на примере:


import java.io.*;

public class Program {
 
    public static void main(String[] args) {
         
        Person tom = new Person("Tom", 34, 1.68, false);
		// запис в файл
        try(DataOutputStream dos = new DataOutputStream(new FileOutputStream("data.bin")))
        {
           // записуємо значення
            dos.writeUTF(tom.name);
            dos.writeInt(tom.age);
            dos.writeDouble(tom.height);
            dos.writeBoolean(tom.married);
            System.out.println("File has been written");
        }
        catch(IOException ex){
             
            System.out.println(ex.getMessage());
        }  
         
        // обратное считывание из файлу
        try(DataInputStream dos = new DataInputStream(new FileInputStream("data.bin")))
        {
           // записуємо значення
            String name = dos.readUTF();
            int age = dos.readInt();
            double height = dos.readDouble();
            boolean married = dos.readBoolean();
            System.out.printf("Name: %s  Age: %d  Height: %f  Married: %b", 
                    name, age, height, married);
        }
        catch(IOException ex){
             
            System.out.println(ex.getMessage());
        }  
    } 
}
 
class Person
{
    public String name;
    public int age;
    public double height;
    public boolean married;
     
    public Person(String n, int a, double h, boolean m)
    {
        this.name=n;
        this.height=h;
        this.age=a;
        this.married=m;
    }
}

Здесь мы післядовательно записуємо в файл дані об'єкта Person.

Об'єкт DataOutputStream в конструкторе приймає потік вивіда: DataOutputStream (OutputStream out). У даному випадку в качестве потіка вивіда використовується об'єкт FileOutputStream, поцему вивід буде происходить в файл. И за допомогою выше рассмотренных методів типу writeUTF() производится запис значений в бинарный файл.

Затем происходит читання ранїї записанных данных. Об'єкт DataInputStream в конструкторе приймає потік для читання: DataInputStream(InputStream in). Здесь таким потіком выступает об'єкт FileInputStream
