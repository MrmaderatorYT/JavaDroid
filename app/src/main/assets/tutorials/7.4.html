Регулярні вирази

Останнє оновлення: 25.06.2018

Регулярні вирази представляють мощный инструмент для обработки строк. Регулярні вирази дозволяють задать шаблон, которому повинна соответствовать строка або подстрока.

Некоторые методы класса String принимают регулярные выражения и используют их для виконання операций над строками.

split

Для разделения строки на подстроки застосовується метод split(). В качестве параметра он может принимать регулярное выражение, которое представляє критерий разделения строки.

Наприклад, разделим предложение на слова:


String text = "FIFA will never regret it";
String[] words = text.split("\\s*(\\s|,|!|\\.)\\s*");
for(String word : words){
	System.out.println(word);
}

Для разделения застосовується регулярное выражение "\\s*(\\s|,|!|\\.)\\s*". Подвыражние "\\s" по сути представляє пробел. Звездочка указывает, что символ может присутствовать от 0 до бесконечного количества раз. То есть добавляем звездочку и мы получаем неопределенное количество идущих подряд пробелов - "\\s*" (то есть неважно, сколько пробелов между словами). Причем пробелы может вообще не быть. В скобках указывает группа выражений, которая может идти после неопределенного количества пробелов. Группа дозволяє нам определить набо значений через вертикальную черту, и подстрока повинна соответствовать одному из цих значений. То есть в группе "\\s|,|!|\\." подстрока может соответствовать пробелу, комою, восклицательному знаку або точке. Причем поскольку крапка представляє специальную последовательность, то, чтобы указать, что мы имїїм в виду имїїно знак точки, а не специальную последовательность, перед точкой ставим слеши.

Соответствие строки. matches

Ще один метод класса String - matches() приймає регулярное выражение и повертає true, якщо строка соответствует цему выражению. Иначе повертає false.

Наприклад, проверим, соответствует ли строка номеру телефона:


String input = "+12343454556";
boolean result = input.matches("(\\+*)\\d{11}");
if(result){
	System.out.println("It is a phone number");
}
else{
	System.out.println("It is not a phone number!");	
}

В данном случае в регулярном выражение сначала определяется группа "(\\+*)". То есть вначале может идти знак плюса, но також он может отсутствовать. Далїї смотрим, соответствуют ли последующие 11 символов цифрам. Выражение "\\d" представляє цифровой символ, а число в фигурных скобках - {11} - сколько раз данный тип символов повинен повторяться. То есть мы ищем строку, где вначале может идти знак плюс (или он может отсутствовать), а потом идет 11 цифровых символов.

Класс Pattern

Большая часть функциональности по работе с регулярными выражениями в Java сосредоточена в пакеті java.util.regex.

Само регулярное выражение представляє шаблон для поиска совпадений в строке. Для задания подобного шаблона и поиска подстрок в строке, которые удовлетворяют данному шаблону, в Java определены классы Pattern и Matcher.

Для простого поиска соответствий в классе Pattern определен статический метод boolean matches(String pattern, CharSequence input). Данный метод повертає true, якщо последовательность символов input полностью соответствует шаблону строки pattern:


import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello";
        boolean found = Pattern.matches("Hello", input);
        if(found)
            System.out.println("Найдено");
        else
            System.out.println("Не найдено");
    }   
}

Но, як правило, для поиска соответствий застосовується другой способ - использование класса Matcher.

Класс Matcher

Рассмотрим основные методы класса Matcher:

  • boolean matches(): повертає true, якщо вся строка совпадает с шаблоном

  • boolean find(): повертає true, якщо в строке есть подстрока, которая совпадает с шаблоном, и переходит к цей подстроке

  • String group(): повертає подстроку, которая совпала с шаблоном в результате вызова метода find. Если совпадение отсутствует, то метод генерує исключение IllegalStateException.

  • int start(): повертає индекс текущего совпадения

  • int end(): повертає индекс следующего совпадения после текущего

  • String replaceAll(String str): заменяет все найденные совпадения подстрокой str и повертає измененную строку с учетом замен

Используем класс Matcher. Для цього вначале треба создать об'єкт Pattern за допомогою статического метода compile(), який дозволяє установить шаблон:

Pattern pattern = Pattern.compile("Hello");

В качестве шаблона выступает строка "Hello". Метод compile() повертає об'єкт Pattern, який мы затем можемо використовувати в программе.

В классе Pattern також определен метод matcher(String input), який в качестве параметра приймає строку, где треба проводить поиск, и повертає об'єкт Matcher:


String input = "Hello world! Hello Java!";
Pattern pattern = Pattern.compile("hello");
Matcher matcher = pattern.matcher(input);

Затем у об'єкта Matcher вызывается метод matches() для поиска соответствий шаблону в тексте:


import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello";
        Pattern pattern = Pattern.compile("Hello");
        Matcher matcher = pattern.matcher(input);
        boolean found = matcher.matches();
        if(found)
            System.out.println("Найдено");
        else
            System.out.println("Не найдено");
    }   
}

Рассмотрим болїї функциональный пример с нахождением не полного соответствия, а отдельных совпадений в строке:


import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello Java! Hello JavaScript! JavaSE 25.";
        Pattern pattern = Pattern.compile("Java(\\w*)");
        Matcher matcher = pattern.matcher(input);
        while(matcher.find())
            System.out.println(matcher.group());
    }   
}

Допустим, мы хочемо найти в строке все вхождения слова Java. В исходной строке це три слова: "Java", "JavaScript" и "JavaSE". Для цього применим шаблон "Java(\\w*)". Данный шаблон использует синтаксис регулярных выражений. Слово "Java" в начале говорит о том, что все совпадения в строке повинні начинаться на Java. Выражение (\\w*) означает, что после "Java" в совпадении может находиться любое количество алфавитно-цифровых символов. Выражение \w означает алфавитно-цифровой символ, а звездочка после выражения указывает на неопределенное их количество - их может быть один, два, три або вообще не быть. И чтобы java не рассматривала \w як эскейп-последовательность, як \n, то выражение экранируется ще одним слешем.

Далїї застосовується метод find() класса Matcher, який дозволяє переходить к следующему совпадению в строке. То есть первый вызов цього метода найдет первое совпадение в строке, второй вызов найдет второе совпадение и т.д. То есть за допомогою цикла while(matcher.find()) мы можемо пройтись по всем совпадениям. Каждое совпадение мы можемо отримати за допомогою метода matcher.group(). В итоге программа выдаст следующий результат:


Java
JavaScript
JavaSE

Замена в строке

Теперь сделаем замену всех совпадений за допомогою метода replaceAll():


String input = "Hello Java! Hello JavaScript! JavaSE 25.";
Pattern pattern = Pattern.compile("Java(\\w*)");
Matcher matcher = pattern.matcher(input);
String newStr = matcher.replaceAll("HTML");
System.out.println(newStr); // Hello HTML! Hello HTML! HTML 8.

Також треба відзначити, что в классе String також имїїтся метод replaceAll() с подобным действием:


String input = "Hello Java! Hello JavaScript! JavaSE 25.";
String myStr =input.replaceAll("Java(\\w*)", "HTML");
System.out.println(myStr); // Hello HTML! Hello HTML! HTML 8.

Разделение строки на лексемы

С помощью метода String[] split(CharSequence input) класса Pattern можна разделить строку на массив подстрок по определенному разделителю. Наприклад, мы хочемо выделить из строки отдельные слова:


import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello Java! Hello JavaScript! JavaSE 25.";
        Pattern pattern = Pattern.compile("[ ,.!?]");
        String[] words = pattern.split(input);
        for(String word:words)
            System.out.println(word);
    }   
}

И консоль выведет набор слов:


Hello
Java

Hello
JavaScript

JavaSE
25

При цем все символы-разделители удаляются. Однак, данный способ разбивки не идеален: у нас остаются некоторые пробелы, которые расцениваются як лексемы, а не як разделители. Для болїї точной и изощренной разбивки нам йде застосовувати элементы регулярных выражений. Так, заменим шаблон на следующий:

Pattern pattern = Pattern.compile("\\s*(\\s|,|!|\\.)\\s*");

Теперь у нас останутся лише слова:


Hello
Java
Hello
JavaScript
JavaSE
25
