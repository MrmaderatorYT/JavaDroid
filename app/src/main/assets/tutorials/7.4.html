Регулярні вирази

Останнє оновлення: 25.06.2018

Регулярні вирази представляютьь мощный инструмент для обработки рядків. Регулярні вирази дозволяють задать шаблон, которому повинна соответствовать рядок або підрядок.

Некоторые методи класа String приймають регулярные выражения и використовують их для виконання операцій над рядокми.

split

Для розділения рядки на підрядки застосовується метод split(). В качестве параметра он может принимать регулярное выражение, которое представляє критерий розділения рядки.

Наприклад, розділим предложение на слова:


String text = "FIFA will never regret it";
String[] words = text.split("\\s*(\\s|,|!|\\.)\\s*");
for(String word : words){
	System.out.println(word);
}

Для розділения застосовується регулярное выражение "\\s*(\\s|,|!|\\.)\\s*". Підвыражние "\\s" по сути представляє пробел. Звездочка вказує, что символ может присутствовать от 0 до бесконечного количества раз. То есть додаємо звездочку и мы отримуємо невизначенийное кількість идущих підряд пробелов - "\\s*" (то есть неважливо, сколько пробелов між словами). Причем пробелы может вообще не быть. В скобках вказує группа выражений, которая может йти після невизначенийного количества пробелов. Группа дозволяє нам визначити набо значений через вертикальную черту, и підрядок повинна соответствовать одному из цих значений. То есть в группе "\\s|,|!|\\." підрядок может соответствовать пробелу, комою, восклицательному знаку або точке. Причем поскольку крапка представляє специальную післядовательность, то, чтобы указать, что мы имїїм в виду имїїно знак точки, а не специальную післядовательность, перед точкой ставим слеши.

Соответствие рядки. matches

Ще один метод класа String - matches() приймає регулярное выражение и повертає true, якщо рядок відповідає цему выражению. Иначе повертає false.

Наприклад, проверим, відповідає ли рядок номеру телефона:


String input = "+12343454556";
boolean result = input.matches("(\\+*)\\d{11}");
if(result){
	System.out.println("It is a phone number");
}
else{
	System.out.println("It is not a phone number!");	
}

У даному випадку в регулярном выражение сначала визначаєся группа "(\\+*)". То есть вначале может йти знак плюса, но також он может отсутствовать. Далїї смотрим, відповідають ли післядующие 11 символів цифрам. Выражение "\\d" представляє цифровой символ, а число в фигурных скобках - {11} - сколько раз данный тип символів повинен повторяться. То есть мы ищем рядок, где вначале может йти знак плюс (или он может отсутствовать), а потом идет 11 цифровых символів.

Клас Pattern

Большая часть функциональности по работе с регулярными выражениями в Java сосредоточена в пакеті java.util.regex.

Само регулярное выражение представляє шаблон для поиска совпадений в рядківе. Для задания підобного шаблона и поиска підрядків в рядківе, которые удовлетворяют данному шаблону, в Java визначенийы класы Pattern и Matcher.

Для простого поиска соответствий в класе Pattern визначений статический метод boolean matches(String pattern, CharSequence input). Данный метод повертає true, якщо післядовательность символів input полностью відповідає шаблону рядки pattern:


import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello";
        boolean found = Pattern.matches("Hello", input);
        if(found)
            System.out.println("Найдено");
        else
            System.out.println("Не найдено");
    }   
}

Но, як правило, для поиска соответствий застосовується другой способ - використання класа Matcher.

Клас Matcher

Рассмотрим основные методи класа Matcher:

  • boolean matches(): повертає true, якщо вся рядок збігається с шаблоном

  • boolean find(): повертає true, якщо в рядківе есть підрядок, которая збігається с шаблоном, и переходит к цей підрядківе

  • String group(): повертає підрядок, которая совпала с шаблоном в результаті вызова методу find. Если совпадение отсутствует, то метод генерує виняток IllegalStateException.

  • int start(): повертає индекс текущего совпадения

  • int end(): повертає индекс следующего совпадения після текущего

  • String replaceAll(String str): заменяет все найденные совпадения підрядківой str и повертає измененную рядок с учетом замен

Використовуємо клас Matcher. Для цього вначале треба створити об'єкт Pattern за допомогою статического методу compile(), який дозволяє установить шаблон:

Pattern pattern = Pattern.compile("Hello");

В качестве шаблона выступает рядок "Hello". Метод compile() повертає об'єкт Pattern, який мы затем можемо використовувати в програме.

В класе Pattern також визначений метод matcher(String input), який в качестве параметра приймає рядок, где треба проводить поиск, и повертає об'єкт Matcher:


String input = "Hello world! Hello Java!";
Pattern pattern = Pattern.compile("hello");
Matcher matcher = pattern.matcher(input);

Затем у об'єкта Matcher вызывается метод matches() для поиска соответствий шаблону в тексте:


import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello";
        Pattern pattern = Pattern.compile("Hello");
        Matcher matcher = pattern.matcher(input);
        boolean found = matcher.matches();
        if(found)
            System.out.println("Найдено");
        else
            System.out.println("Не найдено");
    }   
}

Рассмотрим болїї функциональный пример с нахождением не полного соответствия, а отдельных совпадений в рядківе:


import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello Java! Hello JavaScript! JavaSE 25.";
        Pattern pattern = Pattern.compile("Java(\\w*)");
        Matcher matcher = pattern.matcher(input);
        while(matcher.find())
            System.out.println(matcher.group());
    }   
}

Допустим, мы хочемо найти в рядківе все вхождения слова Java. В исходной рядківе це три слова: "Java", "JavaScript" и "JavaSE". Для цього застосуємо шаблон "Java(\\w*)". Данный шаблон використовує синтаксис регулярных выражений. Слово "Java" в начале говорит о том, что все совпадения в рядківе повинні начинаться на Java. Выражение (\\w*) означает, что після "Java" в совпадении может находиться любое кількість алфавитно-цифровых символів. Выражение \w означает алфавитно-цифровой символ, а звездочка після выражения вказує на невизначенийное их кількість - их может быть один, два, три або вообще не быть. И чтобы java не рассматривала \w як эскейп-післядовательность, як \n, то выражение экранируется ще одним слешем.

Далїї застосовується метод find() класа Matcher, який дозволяє переходить к следующему совпадению в рядківе. То есть первый вызов цього методу найдет первое совпадение в рядківе, второй вызов найдет второе совпадение и т.д. То есть за допомогою цикла while(matcher.find()) мы можемо пройтись по всем совпадениям. Каждое совпадение мы можемо отримати за допомогою методу matcher.group(). В итоге програма выдаст следующий результат:


Java
JavaScript
JavaSE

Замена в рядківе

Теперь сделаем замену всех совпадений за допомогою методу replaceAll():


String input = "Hello Java! Hello JavaScript! JavaSE 25.";
Pattern pattern = Pattern.compile("Java(\\w*)");
Matcher matcher = pattern.matcher(input);
String newStr = matcher.replaceAll("HTML");
System.out.println(newStr); // Hello HTML! Hello HTML! HTML 8.

Також треба відзначити, что в класе String також имїїтся метод replaceAll() с підобным действием:


String input = "Hello Java! Hello JavaScript! JavaSE 25.";
String myStr =input.replaceAll("Java(\\w*)", "HTML");
System.out.println(myStr); // Hello HTML! Hello HTML! HTML 8.

Разделение рядки на лексемы

За допомогою методу String[] split(CharSequence input) класа Pattern можна розділить рядок на масив підрядків по визначенийному розділителю. Наприклад, мы хочемо выделить из рядки отдельные слова:


import java.util.regex.Pattern;

public class StringsApp {

    public static void main(String[] args) {
        
        String input = "Hello Java! Hello JavaScript! JavaSE 25.";
        Pattern pattern = Pattern.compile("[ ,.!?]");
        String[] words = pattern.split(input);
        for(String word:words)
            System.out.println(word);
    }   
}

И консоль выведет набор слов:


Hello
Java

Hello
JavaScript

JavaSE
25

При цем все символи-розділители видаляються. Однак, данный способ разбивки не идеален: у нас остаются некоторые пробелы, которые расцениваются як лексемы, а не як розділители. Для болїї точной и изощренной разбивки нам йде застосовувати елементы регулярных выражений. Так, заменим шаблон на следующий:

Pattern pattern = Pattern.compile("\\s*(\\s|,|!|\\.)\\s*");

Теперь у нас останутся лише слова:


Hello
Java
Hello
JavaScript
JavaSE
25
