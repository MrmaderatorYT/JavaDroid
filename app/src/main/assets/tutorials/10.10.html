Паралельні операції над масивами

Останнє оновлення: 30.04.2018

В JDK 8 к класу Arrays буво добавлено ряд методів, которые дозволяють в параллельном режиме совершать обработку елементов масива. И хотя дані методи формально не входят в Stream API, но реалізують схожую функциональность, что и параллельные потіки:

  • parallelPrefix(): вычисляет некоторое значення для елементов масива (наприклад, сумму елементов)

  • parallelSetAll(): встановлює елементы масива за допомогою лямбда-выражения

  • parallelSort(): сортує масив

Використовуємо метод parallelSetAll() для установки елементов масива:


import java.util.Arrays;
public class Program {

    public static void main(String[] args) {
		
        int[] numbers = initializeArray(6);
        for(int i: numbers)
            System.out.println(i);
        
    } 
    public static int[] initializeArray(int size) {
        int[] values = new int[size];
        Arrays.parallelSetAll(values, i -> i*10);
        return values;
    }
}

В метод Arrays.parallelSetAll передаєся два параметра: змінюємоый масив и функція, которая встановлює елементы масива. Эта функція перебирает всі елементи и в качестве параметра отримує индекс текущего перебираемого елемента. Выражение i -> i*10 означает, что по каждому индексу в масиве буде хранится число, равное i * 10. В итоге мы получим следующий вивід:

0
10
20
30
40
50

Рассмотрим болїї сложный пример. Пусть у нас есть следующий клас Phone:


class Phone{
    
    private String name;
    private int price;
    
    public Phone(String name, int price){
        this.name=name;
        this.price = price;
    }
    
    public String getName() {
        return name;
    }
    public void setName(String val) {
        this.name=val;
    }
    public int getPrice() {
        return price;
    }
    public void setPrice(int val) {
        this.price=val;
    }
}

Теперь произведем манипуляции с масивом об'єктов Phone:


Phone[] phones = new Phone[]{new Phone("iPhone 8", 54000), 
    new Phone("Pixel 2", 45000),
    new Phone("Samsung Galaxy S9", 40000),
    new Phone("Nokia 9", 32000)};
        
Arrays.parallelSetAll(phones, i -> {
    phones[i].setPrice(phones[i].getPrice()-10000); 
    return phones[i];
});
        
for(Phone p: phones)
    System.out.printf("%s - %d \n", p.getName(), p.getPrice());

Теперь лямбда-выражение в методі Arrays.parallelSetAll представляє блок коду. И оскільки лямбда-выражение повинно возвращать об'єкт, то нам треба явным чином використовувати оператор return. В цем лямбда-выражении опять же функція отримує индексы перебираемых елементов, и по цим индексам мы можемо обратиться к елементам масива и их змінити. Конкретно у даному випадку происходит уменьшение цены смартфонов на 10000 единиц. В итоге мы получим следующий консольный вивід:

iPhone 8 - 44000 
Pixel 2 - 35000 
Samsung Galaxy S9 - 30000 
Nokia 9 - 22000

Сортування

Отсортуємо масив чисел в параллельном режиме:


int[] nums = {30, -4, 5, 29, 7, -8};
Arrays.parallelSort(nums);
for(int i: nums)
    System.out.println(i);

Метод Arrays.parallelSort() в качестве параметра приймає масив и сортує его за зростанням:

-8
-4
5
7
29
30

Если же нам треба як-то по-іншому отсортировать об'єкты, наприклад, по модулю числа, або у нас болїї сложные об'єкты, то мы можемо створити свой компаратор и передать его в качестве второго параметра в Arrays.parallelSort(). Наприклад, возьмем выше визначенийный клас Phone и створимо для него компаратор:


import java.util.Arrays;
import java.util.Comparator;
public class Program {
 
    public static void main(String[] args) {
         
        Phone[] phones = new Phone[]{new Phone("iPhone 8", 54000), 
		new Phone("Pixel 2", 45000),
		new Phone("Samsung Galaxy S9", 40000),
		new Phone("Nokia 9", 32000)};
        
        Arrays.parallelSort(phones,new PhoneComparator());
        
         for(Phone p: phones)
            System.out.println(p.getName());
    }
}
class PhoneComparator implements Comparator<Phone>{
 
    public int compare(Phone a, Phone b){
     
        return a.getName().toUpperCase().compareTo(b.getName().toUpperCase());
    }
}

Метод parallelPrefix

Метод parallelPrefix() походит для тех случаев, когда треба отримати елемент масива або об'єкт того же типу, что и елементы масива, який обладает некоторыми признаками. Наприклад, в масиве чисел це может быть максимальное, минимальное значення и т.д. Наприклад, найдем произведение чисел:


int[] numbers = {1, 2, 3, 4, 5, 6};
Arrays.parallelPrefix(numbers, (x, y) -> x * y);

for(int i: numbers)
    System.out.println(i);

Мы получим следующий результат:

1
2
6
24
120
720

То есть, як мы видим из консольного вивіда, лямбда-выражение из Arrays.parallelPrefix, которое представляє бинарную функцію, отримує два елемента и виконує над ними операцію. Результат операції сохраняется и передаєся в следующий вызов бинарной функції.
