Змінні та константи

Останнє оновлення: 16.09.2025

Для зберігання даних у програмі призначені змінні. Змінна представляє іменовану область пам'яті, яка зберігає значення певного типу. Кожна змінна має тип, ім'я та значення. Тип визначає, яку інформацію може зберігати змінна або діапазон допустимих значень.

Змінні оголошуються так:

тип_данных имя_змінної;

Наприклад, визначимо змінну, яка називатиметься x і матиме тип int:

int x;

У цьому виразі ми оголошуємо змінну x типу int. Тобто x зберігатиме деяке число не більше 4 байт.

Як ім'я змінної може виступати будь-яка довільна назва, яка задовольняє наступним вимогам:

  • ім'я може містити будь-які алфавітно-цифрові символи, а також знак підкреслення, при цьому перший символ у імені не повинен бути цифрою

  • в імені не повинно бути знаків пунктуації та прогалин

  • ім'я не може бути ключовим чи зарезервованим словом мови Java. Ключові слова — це слова, які використовуються для виконання певних завдань, є частиною синтаксису мови і слугують будівельними блоками для побудови програм. В Java не так багато ключових і зарезервованих слів: abstract, assert, Boolean, break, byte, Case, catch, char, class, const, continue, default, do, double, else, enum, extends, final, finally, float, for, goto, if, implements, import, instanceof, interface, int, long, native, new, null, package, private, protected, Public, return, short, static, strictfp, super, Switch, synchronized, this, throw, throws, transient, try, void, volatile, while

    Цей список може змінюватись з випуском нових версій мови.

Крім того, при оголошенні та подальшому використанні треба враховувати, що Java - регістрозалежна мова, тому наступні оголошення int num; и int NUM; представлятимуть дві різні змінні.

Оголосивши змінну, ми можемоо надати їй значення:


int x;		// оголошення змінної
x = 10;		// присвоєння значення
System.out.println(x); 	// 10

Також можна присвоїти значення змінній при її оголошенні. Цей процес називається ініціалізацією:


int x = 10;		// оголошення та ініціалізація змінної
System.out.println(x); 	// 10

Якщо ми не надамо змінної значення до її використання, то ми можемоо отримати помилку, наприклад, у наступному випадку:


int x;
System.out.println(x);

Через кому можна оголосити відразу кілька змінних одного типу:


int x, y;
x = 10;
y = 25;
System.out.println(x); 	// 10
System.out.println(y); 	// 25

Також можна їх одразу ініціалізувати:


int x = 8, y = 15;
System.out.println(x); 	// 8
System.out.println(y); 	// 15

Відмінною рисою змінних і те, що ми можемоо у процесі роботи програми змінювати їх значення:


int x = 10;
System.out.println(x); 	// 10
x = 25;
System.out.println(x); 	// 25

Ключове слово var

Починаючи з Java 10 у мову було додано ключове слово var, яке також дозволяє визначати змінну:


var x = 10;
System.out.println(x); 	// 10

Слово var ставится вместо типу данных, а сам тип змінної виводиться з того значення, яке їй присвоюється. Наприклад, змінної x присвоюється число 10, значить, змінна представлятиме тип int.

Але якщо змінна оголошується за допомогою var, то ми обов'язково маємо ініціалізувати її, тобто надати їй початкового значення, інакше ми отримаємо помилку, як, наприклад, у такому разі:


var x;		// ! Помилка, змінна не ініціалізована
x = 10;

Константи

Окрім змінних, в Java для зберігання даних можна використовувати константи. На відміну від змінних, константам можна присвоїти значення лише один раз. Константа оголошується так само, як і змінна, тільки спочатку йде ключове слово final:


final int LIMIT = 5;
System.out.println(LIMIT); 	// 5
// LIMIT=57; // так ми вже не можемо написать, оскільки LIMIT - константа

Як правило, константи мають імена у верхньому регістрі.

Константи дозволяють встановити такі змінні, які не повинні більше змінюватися. Наприклад, якщо ми маємо змінну для зберігання числа pi, ми можемоо оголосити її константою, оскільки її значення постійно.
