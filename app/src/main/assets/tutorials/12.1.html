Математичні обчислення та клас Math

Останнє оновлення: 16.09.2025

Для виконання различных математических операций в Java в пакеті java.lang определен класс Math. Рассмотрим его основные методы:

  • abs(double value): повертає абсолютное значение для аргумента value

    
import java.lang.Math;

public class Program{
     
	public static void main(String[] args) {
        double result = Math.abs(-13.5); // 13.5
        System.out.println(result); // 13.5
    }
}

  • acos(double value): повертає арккосинус value. Параметр value повинен иметь значение от -1 до 1

    
double result = Math.acos(1); // 0.0

  • asin(double value): повертає арксинус value. Параметр value повинен иметь значение от -1 до 1

  • atan(double value): повертає арктангенс value

  • cbrt(double value): повертає кубический корень числа value

    
double result = Math.cbrt(27); // 3

  • ceil(double value): повертає наименьшїї целое число с плавающей точкой, которое не меньше value

    
double result = Math.ceil(2.34); // 3

  • clamp(double num, double a, double b): заставляет число num вписываться в задані границы - между a и b - якщо num меньше a, то повертаєся a, якщо num больше b, то повертаєся b

    : 
    
import java.lang.Math;

public class Program{
     
	public static void main(String[] args) {
        
        System.out.println(Math.clamp(-1, 0, 5)); // -1 не входит в границы дипазаона [0, 5], поцему повертаєся 0
        System.out.println(Math.clamp(6, 0, 5)); // 6 не входит в границы дипазаона [0, 5], поцему повертаєся 5
        System.out.println(Math.clamp(4, 0, 5)); // 4 входит в границы дипазаона [0, 5], поцему повертаєся 4
    }
}

  • cos(double d): повертає косинус угла d

  • cosh(double d): повертає гиперболический косинус угла d

  • exp(double d): повертає основание натурального логарифма, возведенное в степень d

  • floor(double d): повертає наибольшїї целое число, которое не больше d

    
double result = Math.floor(2.56); // 2

  • floorDiv(int a, int b): повертає целочисленный результат деления a на b

    
System.out.println(Math.floorDiv(1, 2)); // 0
System.out.println(Math.floorDiv(7, 2)); // 3
System.out.println(Math.floorDiv(9, 2)); // 4

  • log(double a): повертає натуральный логарифм числа a

  • log1p(double d): повертає натуральный логарифм числа (d + 1)

  • log10(double d): повертає десятичный логарифм числа d

  • max(double a, double b): повертає максимальное число из a и b

  • min(double a, double b): повертає минимальное число из a и b

  • pow(double a, double b): повертає число a, возведенное в степень b

  • random(): повертає случайное число от 0.0 до 1.0

  • rint(double value): повертає число double, которое представляє ближайшїї к числу value целое число

    
System.out.println(Math.rint(2)); // 2.0
System.out.println(Math.rint(2.3)); // 2.0
System.out.println(Math.rint(2.5)); // 2.0
System.out.println(Math.rint(2.5001)); // 3.0
System.out.println(Math.rint(2.8)); // 3.0

  • round(double d): повертає число d, округленное до ближайшего целого числа

    
System.out.println(Math.round(2.3)); // 2
System.out.println(Math.round(2.5)); // 3
System.out.println(Math.round(2.5001)); // 3
System.out.println(Math.round(2.8)); // 3

  • scalb(double value, int factor): повертає произведение числа value на 2 в степени factor

    
System.out.println(Math.scalb(5, 3)); // 5*2*2*2 = 40
System.out.println(Math.scalb(3, 4)); // 3*2*2*2*2 = 48

  • signum(double value): повертає число 1, якщо число value положительное, и -1, якщо значение value отрицательное. Если value равно 0, то повертає 0

    
System.out.println(Math.signum(2.3)); // 1
System.out.println(Math.signum(-2.3)); // -1

  • sin(double value): повертає синус угла value

  • sinh(double value): повертає гиперболический синус угла value

  • sqrt(double value): повертає квадратный корень числа value

    
double result1 = Math.sqrt(16); // 4

  • tan(double value): повертає тангенс угла value

  • tanh(double value): повертає гиперболический тангенс угла value

  • toDegrees(double value) переводит радианы в градусы и toRadians(double value) - градусы в радианы

    
System.out.println(Math.toDegrees(3.14159)); // 180
System.out.println(Math.toRadians(90)); // 1,57079....

Також класс Math определяет три константы: Math.E, Math.TAU и Math.PI. Наприклад, вычислим площадь круга:


import java.lang.Math;
import java.util.Scanner;

public class Program {
  
    public static void main(String[] args) {
          
        Scanner in = new Scanner(System.in);
        
        System.out.print("Введите радиус круга: ");
        int radius = in.nextInt();
        long area = Math.round(Math.PI * Math.pow(radius, 2));
        System.out.printf("Площадь круга с радиусом %d равна %d \n", radius, area);
    }
}
