Буферизація символьних потіків. BufferedReader та BufferedWriter

Останнє оновлення: 26.04.2018

Запис текста через буфер и BufferedWriter

Клас BufferedWriter записує текст в потік, предварительно буферизируя записуємоые символи, тем самым снижая кількість обращений к физическому носителю для записи данных.

Клас BufferedWriter имїїт следующие конструктори:


BufferedWriter(Writer out) 
BufferedWriter(Writer out, int sz)

В качестве параметра он приймає потік вивіда, в який треба осуществить запис. Второй параметр вказує на размер буфера.

Наприклад, осуществим запис в файл:


import java.io.*;

public class Program {
 
    public static void main(String[] args) {
         
		try(BufferedWriter bw = new BufferedWriter(new FileWriter("notes4.txt")))
        {
			String text = "Hello  World!\nHey! Teachers! Leave the kids alone.";
			bw.write(text);
        }
        catch(IOException ex){
             
            System.out.println(ex.getMessage());
        } 
    } 
}

Читання текста и BufferedReader

Клас BufferedReader считывает текст из символьного потіка ввода, буферизируя прочитанные символи. Використання буфера призвано увеличить производительность читання данных из потіка.

Клас BufferedReader имїїт следующие конструктори:


BufferedReader(Reader in) 
BufferedReader(Reader in, int sz)

Второй конструктор, кроме потіка ввода, из которого производится читання, також визначає размер буфера, в який будуть считываться символи.

Так як BufferedReader найдеся от класа Reader, то он может використовувати все те методи для читання из потіка, которые визначенийы в Reader. И також BufferedReader визначає свой собственный метод readLine(), який дозволяє считывать из потіка построчно.

Рассмотрим застосування BufferedReader:


import java.io.*;

public class Program {

    public static void main(String[] args) {
       
        try(BufferedReader br = new BufferedReader (new FileReader("notes4.txt")))
        {
           // читання посимвольно
            int c;
            while((c=br.read())!=-1){
                
                System.out.print((char)c);
            }
        }
        catch(IOException ex){
            
            System.out.println(ex.getMessage());
        } 
    } 
}

Також можна считать текст построчно:


try(BufferedReader br = new BufferedReader(new FileReader("notes4.txt")))
{
    //читання построчно
    String s;
    while((s=br.readLine())!=null){
                
        System.out.println(s);
    }
}
 catch(IOException ex){
            
    System.out.println(ex.getMessage());
}

Считывание с консоли в файл

Соединим оба класа BufferedReader и BufferedWriter для считывания с консоли в файл. Для цього визначимо следующий код програми:


import java.io.*;

public class Program {

    public static void main(String[] args) {
        
        try(BufferedReader br = new BufferedReader (new InputStreamReader(System.in)); 
                BufferedWriter bw = new BufferedWriter(new FileWriter("notes5.txt")))
        {
           // читання построчно
            String text;
            while(!(text=br.readLine()).equals("ESC")){
                 
                bw.write(text + "\n");
                bw.flush();
            }
        }
        catch(IOException ex){
             
            System.out.println(ex.getMessage());
        } 
    }   
}

Здесь об'єкт BufferedReader встановлюєся для читання с консоли за допомогою об'єкта new InputStreamReader(System.in). В цикле while считывается введенный текст. И поки пользователь не введет рядок "ESC", об'єкт BufferedWriter буде записывать текст файл.
