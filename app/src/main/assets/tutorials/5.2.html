Клас ArrayList та інтерфейс List

Останнє оновлення: 11.10.2025

Для створення простих списків застосовується інтерфейс List, який расширяет функцональность інтерфейса Collection. Деякі найбільш часто використовувані методи інтерфейсу List:

  • void add(int index, E obj): додає до списку за індексом index об'єкт obj

  • void addFirst(E obj): додає об'єкт на початок списку

  • void addLast(E obj): додає об'єкт в кінець списку

  • boolean addAll(int index, Collection<? extends E> col): додає до списку за індексом index всі елементи колекції col. Якщо в результаті додавання список був змінений, то повертається true, інакше повертається false

  • E get(int index): повертає об'єкт зі списку за індексом index

  • E getFirst(): повертає перший елемент списку

  • E getLast(): повертає останній елемент списку

  • int indexOf(Object obj): повертає індекс першого входження об'єкта obj у список. Якщо об'єкт не знайдено, то повертається -1

  • int lastIndexOf(Object obj): повертає індекс останнього входження об'єкта obj у список. Якщо об'єкт не знайдено, то повертається -1

  • ListIterator<E> listIterator (): повертає об'єкт ListIterator для обходу елементів списку

  • static <E> List<E> of(элементы): створює з набору елементів об'єкт List

  • E remove(int index): видаляє об'єкт зі списку за індексом index, повертаючи при цьому видалений об'єкт

  • E removeFirst(): видаляє перший елемент зі списку

  • E removeLast(): видаляє останній елемент зі списку

  • E set(int index, E obj): присвоює значення об'єкта obj елементу, який знаходиться за індексом index

  • void sort(Comparator<? super E> comp): сортує список за допомогою компаратора comp

  • List<E> subList(int start, int end): отримує набір елементів, які знаходяться у списку між індексами start та end

  • static <E> List<E>of(E... elements): повертає незмінний список (тип java.util.ImmutableCollections$ListN), який містить елементи elements.

За замовчуванням в Java є вбудована реалізація цього інтерфейсу - класс ArrayList. Класс ArrayList представляє обобщенную коллекцию, яка успадковує свою функціональність від класу AbstractList і застосовує інтерфейс List. Простіше кажучи, ArrayList представляє простий список, аналогічний масиву, за винятком того, що кількість елементів у ньому не фіксована.

ArrayList має наступні конструктори:

  • ArrayList(): створює порожній список

  • ArrayList(Collection <? extends E> col): створює список, до якого додаються всі елементи колекції col.

  • ArrayList (int capacity): створює список, який має початкову ємність capacity

Ємність в ArrayList представляє розмір масиву, який буде використовуватися для зберігання об'єктів. При добавлении элементов фактически происходит перераспределение памяти - створення нового масиву та копіювання в нього елементів зі старого масиву. Изначальное задание емкости ArrayList дозволяє снизить подобные перераспределения памяти, тим самим підвищуючи продуктивність.

Розглянемо основні операції ArrayList. Створення списку:


import java.util.ArrayList;  // импортируем класс

public class Program{
      
    public static void main(String[] args) {

      ArrayList<String> people = new ArrayList<String>();
    }
}

Оскільки компілятор може сам вивести тип, то ми можемоо скоротити визначення:


ArrayList<String> people = new ArrayList<>();
// або так
var people = new ArrayList<String>();

Размер списка

Для отримання розміру списку застосовується метод size(), який повертає значення типу int:


var people = new ArrayList<String>();
System.out.println(people.size()); // 0

Добавление

Для додавання застосовується метод add(), якому передається додаваний елемент:


var people = new ArrayList<String>();

people.add("Tom");
people.add("Bob");
people.add("Sam");

System.out.println(people.size()); // 3

Крім того, за допомогою перевантаження методу add() ми можемоо додати елемент на визначену позицію у списку:


var people = new ArrayList<String>();

people.add("Tom");
people.add("Bob");
people.add("Sam");
people.add(1, "Alice"); // добавляем элемент по индексу 1

System.out.println(people.size()); // 4

В выражении people.add(1, "Alice") мы добавляем строку "Alice" в список по индексу 1.

Варто відзначити, что при создании ArrayList выделяется начальная область памяти, яка за замовчуванням вміщує 10 елементів списку. Наприклад, в коде выше при создании списка для него выделяется область памяти, достатня для вміщення 10 рядків String. Якщо ми будемо додавати більшу кількість рядків, то будутьь відбуватися повторні виділення пам'яті для вміщення нових об'єктів. Оскільки виділення пам'яті на низькому рівні - досить затратний процес, то, якщо ми заздалегідь знаємо скільки у нас буде об'єктів у списку або знаємо принаймні максимальну кількість об'єктів, то рекомендуется це количество передавать в конструктор при создании об'єкта ArrayList. Наприклад, створимо список, у якого початкова ємність дорівнює 16 об'єктам:


var people = new ArrayList<String>(16);
System.out.println(people.size()); // 0

Причому важливо, що ємність списку та його розмір - це не одне й те саме. Так, у даному випадку при створенні списку виділяється область для зберігання 16 об'єктів, але поки вона порожня і містить 0 об'єктів. Тем не менїї, когда мы будем добавлять 11-й элемент, не відбуватиметься додаткового виділення пам'яті, тому що необхідна пам'ять вже виділена. Що в результаті збільшить продуктивність.

Або це можна сделать за допомогою метода ensureCapacity:

people.ensureCapacity(25);

Додавання елементів з колекції

Додавати до списку по одному елементу може бути не двже зручно, особливо коли треба додати безліч елементів. Але Java надає ряд способів для додавання набору елементів. Прежде всего інтерфейс List определяет статический метод of(), якому можна передати довільну кількість об'єктів:


import java.util.List;

public class Program{
       
    public static void main(String[] args) {
           
        // создаем список из 3 строк
        List<String> people = List.<String>of("Tom", "Bob", "Sam");

        System.out.println(people.size()); // 3
    }
}

У даному випадку змінна people представляє інтерфейс List<String>. У реальності ця змінна представляє тип java.util.ImmutableCollections$ListN - внутрішній клас Java, який представляє незмінний список фіксованої довжини. Подібний список може бути корисним у деяких ситуаціях, но в данном случае мы говорим об ArrayList - змінному списку довільного розміру. І для його створення ми можемоо використовувати конструктор:


import java.util.List;
import java.util.ArrayList;

public class Program{
       
    public static void main(String[] args) {
           
        ArrayList<String> people = new ArrayList<String>(List.<String>of("Tom", "Bob", "Sam"));
        System.out.println(people.size()); // 3
    }
}

Из других доступных способов можна відзначити статический метод Collections.addAll():

static <T> boolean addAll(Collection<? super T> c, T... elements)

Перший параметр - колекція, до якої треба додати дані, а второй - добавляемые об'єкты. Застосування:


import java.util.Collections;
import java.util.ArrayList;

public class Program{
       
    public static void main(String[] args) {
           
        var people = new ArrayList<String>();
        Collections.addAll(people, "Tom", "Bob", "Sam");
        System.out.println(people.size()); // 3
    }
}

У даному випадку нам не треба створювати для додаваних об'єктів окрему колекцію.

Отримання елементів списку

Для получения элементов списка застосовується метод get(), якому передається індекс елемента:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var people = new ArrayList<String>(16);
        System.out.println(people.size()); // 0

        people.add("Tom");
        people.add("Bob");
        people.add("Sam");
        people.add(1, "Alice"); // добавляем элемент по индексу 1

        System.out.println(people.get(0));      // Tom
        System.out.println(people.get(1));      // Alice
        System.out.println(people.get(2));      // Bob
        System.out.println(people.get(3));      // Sam
    }
}

Перебір списку

Для перебора списка можна використовувати традиционный цикл for:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var people = new ArrayList<String>(16);
        System.out.println(people.size()); // 0

        people.add("Tom");
        people.add("Bob");
        people.add("Sam");
        people.add(1, "Alice"); // добавляем элемент по индексу 1

        for(int i =0; i < people.size(); i++){

            System.out.println(people.get(i)); 
        }
    }
}

Или цикл for в стиле for-each:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var people = new ArrayList<String>(16);
        System.out.println(people.size()); // 0

        people.add("Tom");
        people.add("Bob");
        people.add("Sam");
        people.add(1, "Alice"); // добавляем элемент по индексу 1

        for(String person : people){

            System.out.println(person); 
        }
    }
}

Зміна елемента

Для изменения элемента застосовується метод set(int index, E obj), де перший параметр представляє індекс елемента для зміни, а другий - розміщуваний за цим індексом об'єкт:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var people = new ArrayList<String>(16);
        System.out.println(people.size()); // 0

        people.add("Tom");
        people.add("Bob");
        people.add("Sam");
        people.set(1, "Alex"); // изменяем элемент по индексу 1

        System.out.println(people.get(1));      // Alex
    }
}

Індекс елемента

Нерідко буває, що нам треба дізнатися індекс певного елемента, то застосовуються методы indexOf(Object obj) (для першого індекса у списку) и lastIndexOf(Object obj) для останнього індекса:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var people = new ArrayList<String>();

        people.add("Tom");
        people.add("Bob");
        people.add("Tom");
        people.add("Sam");

        System.out.println(people.indexOf("Tom"));      // 0
        System.out.println(people.lastIndexOf("Tom"));      // 2
    }
}

Видалення

Для удаления элементов застосовується метод remove(Object obj), якому передається видаляємий об'єкт:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var people = new ArrayList<String>();

        people.add("Tom");
        people.add("Bob");
        people.add("Sam");

        people.remove("Bob"); // удаляем элемент "Bob"

        for(String person : people){

            System.out.println(person);      // Tom
                                            // Sam

        }
    }
}

Консольный вывод:


Tom
Sam

Но метод remove() имїїт и другую версию, якій передається індекс видаляємого елемента - remove(int index). І тут може виникнути питання, що буде, якщо у нас список чисел, і нам треба видалити яке-сь число. Например:


import java.util.ArrayList;

public class Program{
      
    public static void main(String[] args) {

        var numbers = new ArrayList<Integer>();

        numbers.add(11);
        numbers.add(12);
        numbers.add(13);

        numbers.remove(12); // пытаемся удалить элемент 12 

        for(Integer n : numbers){

            System.out.println(n);

        }
    }
}

Дана програма завершиться з помилкою, поскольку при передаче числа в метод remove() це число буде восприниматься як индекс, а не як елемент. И при вызове numbers.remove(12) программа буде пытаться удалить элемент по индексу 12, якого в принципі не існує. І в цьому випадку треба буде спочатку знайти індекс видаляємого значення, а потім видалити його:


numbers.remove(numbers.indexOf(12)); // удаляем элемент 12

Ще один метод - removeIf() дозволяє удалить все элементы, які відповідають деякому предикату - умові. Например:


import java.util.Collections;
import java.util.ArrayList;

public class Program{
       
    public static void main(String[] args) {
           
        var numbers = new ArrayList<Integer>();
        Collections.addAll(numbers, 1, 2, 3, 4, 5, 6, 7, 8);
        numbers.removeIf( num -> num % 2 == 0);
        for(var n : numbers) System.out.print(n + ", "); // 1, 3, 5, 7, 
        System.out.println(); 
    }
}

В метод removeIf() передаєся условие в виде функції, яка отримує кожен елемент списку и повертає true, якщо цей элемент соответствует некоторому условию. В данном случае удаляем все четные числа, тобто якщо при діленні числа на 2 остача дорівнює 0:

numbers.removeIf( num -> num % 2 == 0);
