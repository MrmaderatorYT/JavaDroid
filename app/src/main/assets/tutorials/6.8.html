Читання та запис текстових файлів

Останнє оновлення: 26.04.2018

Хотя за допомогою ранїї рассмотренных классов можна записывать текст в файлы, однак они призначенийы прежде всего для работы с бинарными потоками данных, и их возможнастей для полноценной работы с текстовыми файлами недостатньо. И для цей цели служат совсем другие классы, которые є наследниками абстрактных классов Reader и Writer.

Запись файлов. Класс FileWriter

Класс FileWriter є производным от класса Writer. Он використовується для записи текстовых файлов.

Щоб создать об'єкт FileWriter, можна використовувати один из следующих конструкторов:


FileWriter(File file)
FileWriter(File file, boolean append)
FileWriter(FileDescriptor fd)
FileWriter(String fileName)
FileWriter(String fileName, boolean append)

Так, в конструктор передаєся або путь к файлу в виде строки, або об'єкт File, який ссылается на конкретный текстовый файл. Параметр append указывает, повинні ли дані дозаписываться в конец файла (якщо параметр равен true), або файл повинен перезаписываться.

Запишем в файл якой-нибудь текст:


import java.io.*;

public class Program {

    public static void main(String[] args) {
       
        try(FileWriter writer = new FileWriter("notes3.txt", false))
        {
           // запись всей строки
            String text = "Hello Gold!";
            writer.write(text);
            // запись по символам
            writer.append('\n');
            writer.append('E');
			
			writer.flush();
        }
        catch(IOException ex){
            
            System.out.println(ex.getMessage());
        } 
    } 
}

В конструкторе использовался параметр append со значением false - то есть файл буде перезаписываться. Затем за допомогою методів, определенных в базовом классе Writer производится запись данных.

Чтение файлов. Класс FileReader

Класс FileReader найдеся от абстрактного класса Reader и надає функциональность для чтения текстовых файлов.

Для создания об'єкта FileReader мы можемо використовувати один из его конструкторов:


FileReader(String fileName) 
FileReader(File file)
FileReader(FileDescriptor fd)

А используя методы, определенные в базом классе Reader, произвести чтение файла:


import java.io.*;

public class Program {

    public static void main(String[] args) {
       
        try(FileReader reader = new FileReader("notes3.txt"))
        {
           // читаем посимвольно
            int c;
            while((c=reader.read())!=-1){
                
                System.out.print((char)c);
            } 
        }
        catch(IOException ex){
            
            System.out.println(ex.getMessage());
        }   
    } 
}

Також мы можемо считывать в промежуточный буфер из массива символов:


import java.io.*;
import java.util.Arrays;

public class Program {
 
    public static void main(String[] args) {
         
        try(FileReader reader = new FileReader("notes3.txt"))
        {
			char[] buf = new char[256];
            int c;
            while((c = reader.read(buf))>0){
                
				if(c < 256){
					buf = Arrays.copyOf(buf, c);
				}
                System.out.print(buf);
            } 
        }
        catch(IOException ex){
            
            System.out.println(ex.getMessage());
        }		
    }	  
}

В данном случае считываем последовательно символы из файла в массив из 256 символов, поки не дойдем до конца файла в цем случае метод read возвратит число -1.

Поскольку считанная порция файла может быть меньше 256 символов (наприклад, в файле всего 73 символа), и якщо количество считанных данных меньше размера буфера (256), то выполняем копирование массива за допомогою метода Arrays.copy. То есть фактически обрезаем массив buf, оставляя в нем лише те символы, которые считаны из файла.
