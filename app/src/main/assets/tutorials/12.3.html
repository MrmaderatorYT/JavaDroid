Робота із датами. LocalDate

Останнє оновлення: 17.09.2025

Клас LocalDate из пакету java.time призначений для работы с датами. Функционал цього класа дозволяє создавать даты и изменять их, добавляя и отнимая необхідное кількість дней/месяцев/лет.

Основные методи LocalDate:

  • static LocalDate now(): повертає об'єкт, який представляє текущую дату

  • static LocalDate of(int year, int month, int day): повертає об'єкт, який представляє дату с визначенийными годом, месяцем и днем

  • int getYear(): повертає год даты

  • int getMonthValue(): повертає месяц

  • int getDayOfMonth(): повертає день месяца (значення от 1 до 31)

  • int getDayOfYear(): повертає номер дня года (значення от 1 до 365)

  • DayOfWeek getDayOfWeek(): повертає день недели в виде значення перечисления DayOfWeek

  • LocalDate plusDays(int n): додає к дате некоторое кількість дней

  • LocalDate plusWeeks(int n): додає к дате некоторое кількість недель

  • LocalDate plusMonths(int n): додає к дате некоторое кількість месяцев

  • LocalDate plusYears(int n): додає к дате некоторое кількість лет

  • LocalDate minusDays(int n): отнимает от даты некоторое кількість дней

  • LocalDate minusMonths(int n): отнимает от даты некоторое кількість месяцев

  • LocalDate minusWeeks(int n): отнимает от даты некоторое кількість недель

  • LocalDate minusYears(int n): отнимает от даты некоторое кількість лет

Отримання информации о текущей дате:


import java.time.DayOfWeek;
import java.time.LocalDate;

public class Program {
  
    public static void main(String[] args) {
        
        LocalDate date = LocalDate.now(); // отримуємо текущую дату
        int year = date.getYear();
        int month = date.getMonthValue();
        int dayOfMonth = date.getDayOfMonth();
        DayOfWeek dayOfWeek = date.getDayOfWeek();
        System.out.println(date);
        System.out.println(dayOfWeek);
        System.out.printf("%d.%d.%d \n", dayOfMonth, month, year);
    }
}

Створення и отримання информации по случайной дате:


LocalDate date = LocalDate.of(1914, 12, 31);
System.out.println(date);

Операції с датами - додавання и видалення:


LocalDate date = LocalDate.of(1914, 7, 28);
        
date = date.plusYears(4);
date = date.plusMonths(3);
date = date.plusDays(14);
System.out.println(date);   // 1918-11-11
        
date = date.minusMonths(10);
date = date.minusDays(3);
System.out.println(date);   // 1918-01-08
