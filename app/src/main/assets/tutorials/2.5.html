Умовні конструкції

Останнє оновлення: 16.09.2025

Одним из фундаментальных элементов многих языков программирования є умовні конструкції. Данные конструкции дозволяють направить работу программы по одному из путей в зависимости от определенных условий.

В мові Java для цей цели имїїтся условная конструкция if..else и тернарный оператор ?:

Конструкція if/else

Выражение if..else проверяет истинность некоторого условия и в зависимости от результатов проверки выполняет определенный код. Ее простейшая форма соварто из блока if:


if(условие){

    выполняемые инструкции
}

После ключевого слова if идет условие. Условие повинно представлять значение типу boolean. Це может быть непосредственно значение типу boolean або результат условного выражения или другого выражения, которое повертає значение типу boolean. И якщо це условие истинно (равно true), то срабатывает код, який помщен далїї после условия внутри фигурных скобок.

Наприклад

: 

class Program {
  
    public static void main(String[] args) {
          
        int num1 = 6;
        int num2 = 4;
        if(num1 > num2){
            System.out.printf("число %d больше числа %d\n", num1, num2);
        }
    }
}

Так як, в данном случае первое число больше второго, то выражение num1 > num2 истинно и повертає значение true. Следовательно, управление переходит в блок кода после фигурных скобок и начинает выполнять содержащиеся там инструкции, а конкретно метод System.out.printf, який выводит строку на консоль. Если бы первое число виявилося бы меньше второго або равно ему, то инструкции в блоке if не выполнялись бы.

Якщо блок if містить одну інструкцію, ми можемоо його скоротити, прибравши фігурні дужки:


class Program {
  
    public static void main(String[] args) {
          
        int num1 = 6;
        int num2 = 4;
        if(num1 > num2) 
            System.out.printf("число %d больше числа %d\n", num1, num2);
    }
}

Блок else

Но что, якщо мы захочемо, чтобы при несоблюдении условия також выполнялись якие-або действия? В цем случае мы можемо добавить блок else:


class Program {
  
    public static void main(String[] args) {
          
        int num1 = 6;
        int num2 = 40;
        if(num1 > num2){
            System.out.println("число num1 больше числа num2");
        }
        else{
            System.out.println("число num1 меньше або равно числу num2");
        }
    }
}

Поскольку здесь число num1 НЕ больше числа num2, то условие num1 > num2 не виконується и повертає false, поцему буде выполняться блок инструкций после else.

Опять же якщо блок else містить толко одну инструкцию, то оять же мы можемо его сократить, убрав фигурные скобки:


int num1 = 6;
int num2 = 40;
if(num1 > num2)
    System.out.println("число num1 больше числа num2");
else
    System.out.println("число num1 меньше або равно числу num2");

else if

Но при сравнении чисел мы можемо насчитать три состояния: первое число больше второго, первое число меньше второго и числа равны. С помощью блоков else if мы можемо обробляти дополнительные условия:


class Program {
  
    public static void main(String[] args) {
          
        int num1 = 6;
        int num2 = 40;
        if(num1 > num2){
            System.out.println("число num1 больше числа num2");
        }
        else if(num1 < num2){
            System.out.println("число num1 меньше числа num2");
        }
        else{
            System.out.println("числа num1 и num2 равны");
        }
    }
}

За потреби можна додати кілька виразів else if::


class Program {
  
    public static void main(String[] args) {
          
        String name = "Eugene";

        if (name == "Tom")
            System.out.println("Вас зовут Tomas");
        else if (name == "Bob")
            System.out.println("Вас зовут Robert");
        else if (name == "Mike")
            System.out.println("Вас зовут Michael");
        else
            System.out.println("Неизвестное имя");
    }
}

Тернарна операція

Тернарна операція має наступний синтаксис:

[первый операнд - условие] ? [второй операнд] : [третий операнд]

Таким чином, в цей операции участвуют відразу три операнда. В зависимости от условия тернарная операция повертає второй або третий операнд: якщо условие равно true, то повертаєся второй операнд; якщо условие равно false, то третий. Например:


int x = 3;
int y = 2;
int z = x < y ? (x+y) : (x-y);

Здесь результатом тернарной операции є переменная z. Сначала проверяется условие x<y. И якщо оно соблюдается, то z буде равно второму операнду - (x+y), иначе z буде равно третьему операнду.
