Фільтрування, перебір елементів та відображення

Останнє оновлення: 02.05.2018

Перебор елементов. Метод forEach

Для перебора елементов потіка застосовується метод forEach(), який представляє терминальную операцію. В качестве параметра он приймає об'єкт Consumer<? super String>, який представляє действие, выполняемое для каждого елемента набора. Наприклад:


Stream<String> citiesStream = Stream.of("Париж", "Лондон", "Мадрид","Берлин", "Брюссель");
citiesStream.forEach(s->System.out.println(s));

Фактически це буде аналогично перебору всех елементов в цикле for и выполнению с ними действия, а именно вивід на консоль. В итоге консоль выведет:

Париж
Лондон
Мадрид
Берлин
Брюссель

Кстати мы можемо сократить у даному випадку застосування методу forEach наступним чином:


Stream<String> citiesStream = Stream.of("Париж", "Лондон", "Мадрид","Берлин", "Брюссель");
citiesStream.forEach(System.out::println);

Фактически здесь переадается ссылка на статический метод, який виводить рядок на консоль.

Фільтрація. Метод filter

Для фильтрации елементов у потоці застосовується метод filter(), який представляє промежуточную операцію. Он приймає в качестве параметра некоторое условие в виде об'єкта Predicate<T> и повертає новый потік из елементов, которые удовлетворяют цему условию:


Stream<String> citiesStream = Stream.of("Париж", "Лондон", "Мадрид","Берлин", "Брюссель");
citiesStream.filter(s->s.length()==6).forEach(s->System.out.println(s));

Здесь условие s.length()==6 повертає true для тех елементов, длина которых равна 6 символам. То есть в итоге програма выведет:

Лондон
Мадрид
Берлин

Рассмотрим ще один пример фильтрации с болїї сложными данными. Допустим, у нас есть следующий клас Phone:


class Phone{
    
    private String name;
    private int price;
    
    public Phone(String name, int price){
        this.name=name;
        this.price=price;
    }
    
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
    
    public int getPrice() {
        return price;
    }
    
    public void setPrice(int price) {
        this.price = price;
    }
}

Отфільтруємо набор телефонов по цене:


Stream<Phone> phoneStream = Stream.of(new Phone("iPhone 6 S", 54000), new Phone("Lumia 950", 45000),
                new Phone("Samsung Galaxy S 6", 40000));
        
phoneStream.filter(p->p.getPrice()<50000).forEach(p->System.out.println(p.getName()));

Отображение. Метод map

Отображение або маппинг дозволяє задать функцію преобразования одного об'єкта в другой, то есть отримати из елемента одного типу елемент другого типу. Для отображения використовується метод map, який имїїт следующїї визначенийие:


<R> Stream<R> map(Function<? super T, ? extends R> mapper)

Передавана в метод map функція задає перетворення від об'єктов типу T к типу R. І в результаті повертається новий потік з перетвореними об'єктами.

Візьмемо вищевизначений клас телефонів і виконаємо перетворення від типу Phone до типу String:


Stream<Phone> phoneStream = Stream.of(new Phone("iPhone 6 S", 54000), new Phone("Lumia 950", 45000),
                new Phone("Samsung Galaxy S 6", 40000));
        
phoneStream
    .map(p-> p.getName()) // розміщуємо в потік лише назви телефонів
    .forEach(s->System.out.println(s));

Операція map(p-> p.getName()) помещает в новый потік лише названия телефонов. В итоге на консоли будуть лише названия:

iPhone 6 S
Lumia 950
Samsung Galaxy S 6

Ще проведем преобразования:


phoneStream
    .map(p-> "название: " + p.getName() + " цена: " + p.getPrice())
    .forEach(s->System.out.println(s));

Здесь також результирующий потік містить лише рядки, лише теперь названия соединяются с ценами.

Для преобразования об'єктов в типи Integer, Long, Double визначенийы специальные методи mapToInt(), mapToLong() и mapToDouble() соответственно.

Плоское отображение. Метод flatMap

Плоское отображение виконується тогда, когда из одного елемента потрібно отримати декілька. Данную операцію виконує метод flatMap:


<R> Stream<R> flatMap(Function<? super T, ? extends Stream<? extends R>> mapper)

Наприклад, в примере выше мы вивідим название телефона и его цену. Но что, якщо мы хочемо установить для каждого телефона цену со скидкой и цену без скидки. То есть из одного об'єкта Phone нам треба отримати два об'єкта с информацией, наприклад, в виде рядки. Для цього застосуємо flatMap:


Stream<Phone> phoneStream = Stream.of(new Phone("iPhone 6 S", 54000), new Phone("Lumia 950", 45000),
                new Phone("Samsung Galaxy S 6", 40000));
         
phoneStream
    .flatMap(p->Stream.of(
            String.format("название: %s  цена без скидки: %d", p.getName(), p.getPrice()),
            String.format("название: %s  цена со скидкой: %d", p.getName(), p.getPrice() - (int)(p.getPrice()*0.1))
	))
    .forEach(s->System.out.println(s));

Результат работы програми:

название: iPhone 6 S цена без скидки: 54000
название: iPhone 6 S цена со скидкой: 48600
название: Lumia 950 цена без скидки: 45000
название: Lumia 950 цена со скидкой: 40500
название: Samsung Galaxy S 6 цена без скидки: 40000
название: Samsung Galaxy S 6 цена со скидкой: 36000
